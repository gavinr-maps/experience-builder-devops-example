Classify Pixels Using Deep Learning

The Classify Pixels Using Deep Learning tool uses a deep learning model on an input imagery layer to produce a classified raster, and each valid pixel has an assigned class label.

The imagery layer that will be used to classify pixels.

Specifies how raster items in an imagery layer will be processed after being mosaicked: together or individually.

The deep learning package that contains the model that will be used to classify the pixels.

The model arguments that are present in the deep learning model together with their default values.

The name of the output imagery layer that will be classified.

The name of the layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
