A ferramenta Criar Buffers cria polígonos que abrangem uma determinada distância a partir de elementos de entrada.

Os elementos de ponto, linha ou polígono para os quais serão criados buffers.

Especifica se a distância de buffer será definida por uma constante (Valor) ou por um campo numérico (Campo).

A distância que será utilizada para criar o buffer. Se forem adicionados vários valores, cada valor será utilizado para criar um buffer em cada elemento de entrada.

O campo que irá conter a distância de buffer para cada elemento.

As unidades que serão utilizadas para especificar a distância de buffer.

Especifica como os buffers de diferentes elementos de entrada serão tratados quando se sobrepõem.

Especifica se os buffers de múltiplas distâncias serão buffers concêntricos que se sobrepõem (discos sobrepostos) ou anéis que não se sobrepõem (anéis distintos).

Para elementos de linha, especifica se o buffer será criado em ambos os lados da linha (Em Volta), apenas no lado esquerdo (Esquerda) ou apenas no lado direito (Direita).

Para elementos de polígono, especifica se os elementos de polígono de entrada serão incluídos no buffer (Incluir) ou excluídos do buffer (Excluir).

Especifica se a forma do buffer no fim de uma linha será redonda (Arredondado) ou plana (Plano).

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
