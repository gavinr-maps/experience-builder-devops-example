A ferramenta Encontrar Valores Atípicos irá determinar se existem localizações com valores significativamente diferentes dos seus vizinhos, com base no padrão espacial dos dados.

A camada de ponto ou polígono a partir da qual os valores atípicos serão identificados.

Especifica se os valores atípicos serão determinados com base na localização dos elementos (Contagem de pontos) ou nos valores do campo de elementos (Campo).

O campo que será analisado para determinar os valores atípicos.

O tipo de elementos delimitadores dentro dos quais os elementos de entrada serão agregados. Pode ser uma camada de polígonos (Camada de polígonos) ou uma grelha de tesselação de quadrados (células Fishnet) ou hexágonos (células Hexagonais).

A camada que contém elementos de polígonos na qual os elementos de ponto de entrada serão agregados.

A camada de polígonos que irá definir o limite da análise. Os elementos dentro do polígono serão incluídos na análise de valores atípicos.

Especifica como dividir os valores do campo Análise ou as contagens de pontos agregados. A opção Campo divide os valores do campo Análise pelos valores de um campo especificado no parâmetro da Camada de entrada. A opção Enriquecer dados irá enriquecer os elementos ou as formas de agregação com dados da população e dividir os valores do campo Análise ou as contagens de pontos agregados pela população. Esta opção consome créditos adicionais.

A banda de distância que será utilizada para definir uma vizinhança. Os elementos de vizinhança dentro da banda de distância de um elemento de destino serão incluídos na vizinhança do elemento de destino durante a análise.
