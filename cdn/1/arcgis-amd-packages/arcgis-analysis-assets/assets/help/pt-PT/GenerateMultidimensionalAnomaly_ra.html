A ferramenta Gerar Anomalia Multidimensional calcula a anomalia para cada secção numa camada de imagens multidimensionais existente para gerar uma nova camada de imagens multidimensionais.

A camada de imagens multidimensionais que será analisada.

O método que será utilizado para calcular as anomalias na camada de imagens.

O intervalo temporal que será utilizado para calcular a média.

A camada de imagens de referência que contém os meios calculados para cada píxel.

Especifica se os valores em NoData ou em falta serão ignorados na análise.

O nome da camada que será criada e adicionada ao mapa.

O tipo de saída que será criado. A saída pode ser uma camada de imagens em mosaico ou uma camada de imagens dinâmicas.
