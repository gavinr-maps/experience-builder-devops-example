A ferramenta Extrair Dados seleciona elementos dentro de uma extensão e cria um item no seu conteúdo.

As camadas que serão extraídas. As camadas de entrada suportadas incluem camadas que possui e camadas com exportação ativada.

A extensão dos elementos que serão extraídos.

Especifica se os elementos que intersectam a extensão serão recortados. Se a opção está desselecionada, serão extraídos todos os elementos que intersectam a extensão. Quando a opção está selecionada, os elementos que intersectam a extensão serão recortados e apenas as partes desses elementos dentro da extensão serão extraídas.

O formato de ficheiro dos dados que serão extraídos.

O nome do item de saída onde o ficheiro extraído será armazenado.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
