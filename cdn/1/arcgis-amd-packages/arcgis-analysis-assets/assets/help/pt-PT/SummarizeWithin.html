A ferramenta Resumir no Interior calcula estatísticas para elementos dentro de caixas ou polígonos especificados.

A camada que será resumida.

O tipo de elementos delimitadores dentro dos quais os elementos de entrada serão resumidos. Pode ser uma camada de polígonos ou uma grelha de tesselação de caixas quadradas ou hexagonais que são geradas quando a ferramenta é executada.

A camada de polígonos dentro da qual os elementos de entrada serão resumidos.

O tamanho das caixas geradas. Para caixas quadradas, o número e unidades especificadas determinam a altura e o comprimento do quadrado. Para caixas hexagonais, o número e unidades especificados determinam a distância entre lados paralelos.

As unidades que serão utilizadas para especificar o tamanho da caixa.

Calcule o tamanho total dos elementos de resumo em cada polígono delimitado. Para linhas, é calculado o comprimento total. Para polígonos, é calculada a área total.

As unidades que serão utilizadas para calcular o tamanho total dos elementos de resumo.

Os campos de número ou data da camada de entrada que serão resumidos e as estatísticas que serão calculadas para cada campo.

O campo da camada de entrada que será utilizado para separar os dados em grupos distintos para cálculos estatísticos.

Calcule os grupos minoritários (menos dominantes) e maioritários (mais dominantes) dentro de cada área de resumo.

Calcule a percentagem dos elementos de resumo que pertencem a cada grupo dentro de cada área de resumo.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em O Meu Conteúdo onde os resultados serão guardados.
