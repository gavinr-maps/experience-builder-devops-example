A ferramenta Ligar Elementos adiciona atributos de uma camada ou tabela para outra com base em relações espaciais e de atributos. Em alternativa, as estatísticas podem ser calculadas quando existem múltiplos registos de correspondência.

A camada ou tabela que terá atributos da camada de ligação anexada à respetiva tabela de atributos.

A camada ou tabela com os atributos que serão anexados à camada de destino.

Especifique se deve ligar camadas com base numa relação espacial. Isto exige que ambas as camadas possuam geometria.

Especifique a relação espacial que determina que elementos são ligados. Os elementos na camada de ligação que têm a relação especificada com a camada de destino serão ligados. As relações disponíveis dependem do tipo de geometria das entradas.

A distância a partir dos elementos da camada de destino que será utilizada para determinar se os elementos devem ser ligados.

As unidades que serão utilizadas para especificar a distância.

Especifique se deve ligar camadas com base em valores de campo correspondentes. Esta é a única opção se uma ou mais das entradas for uma tabela.

Especifique a relação de atributo que determina que elementos são ligados. Os elementos são correspondidos quando os valores de campo na camada de ligação são iguais a valores de campo da camada de destino.

Especifique como serão tratadas as ligações se vários registos da camada de ligação cumprirem os critérios de ligação. Escolha uma das seguintes opções:

  • Ligar um a um: ligue um único registo na camada de ligação a cada elemento da camada de destino.
  • Ligar um a muitos: ligue todos os elementos correspondentes na camada de ligação à camada de destino. A camada resultante conterá múltiplos registos do elemento de destino.

Quando múltiplos elementos da camada de ligação corresponderem a um elemento de destino, especifique se deve ligar um único registo da camada de ligação ou calcular estatísticas de resumo de todos os registos correspondentes a partir da camada de ligação.

Quando existem múltiplos elementos de ligação que correspondem a um elemento de destino, serão calculadas estatísticas de resumo para todos os registos correspondentes. Especifique os campos de número ou data da camada de ligação e as estatísticas que serão calculadas para cada campo.

Quando existem múltiplos elementos de ligação que correspondem a um elemento de destino, o primeiro registo da tabela de atributos da camada de ligação será incluído na saída. Em alternativa, escolha um campo de número ou data e a sequência de ordenação que irá determinar a ordenação da tabela de atributos.

Determina se elementos de destino sem correspondência serão incluídos na saída. Uma ligação interior devolve apenas registos de destino correspondentes e uma ligação à esquerda devolve todos os registos de destino, mesmo que o elemento de destino não tenha uma correspondência na camada de ligação.

O nome da camada ou tabela que será criada e adicionada ao mapa.
