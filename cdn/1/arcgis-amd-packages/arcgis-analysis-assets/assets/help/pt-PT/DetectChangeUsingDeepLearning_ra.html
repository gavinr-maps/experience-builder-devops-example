A ferramenta Detetar Alterações Utilizando Aprendizagem Profunda utiliza um modelo de aprendizagem profunda treinado para detetar alterações entre dois rasters.

A camada de imagens de entrada antes da alteração que será utilizada como base de comparação.

A camada de imagens de entrada após a alteração que será utilizada para comparar com a original.

A camada de imagens classificadas de saída que mostra a alteração.

Os argumentos de modelo que estão presentes no modelo de aprendizagem profunda juntamente com os seus valores predefinidos.

O pacote de aprendizagem profunda que contém o modelo que será utilizado para detetar a alteração entre camadas de imagens.

O tipo de saída que será criado. A saída pode ser uma camada de imagens em mosaico ou uma camada de imagens dinâmicas.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
