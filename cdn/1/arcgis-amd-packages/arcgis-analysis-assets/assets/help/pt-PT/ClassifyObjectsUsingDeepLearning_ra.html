A ferramenta Classificar Objetos Utilizando Aprendizagem Profunda utiliza um modelo de aprendizagem profunda para atribuir uma categoria ou etiqueta de classe a um objeto ou elemento.

A camada de imagens ou de elementos que será utilizada para classificar objetos.

A camada de elementos que identifica a localização de cada objeto a ser classificado ou etiquetado.

Especifica como os itens de raster numa camada de imagens serão processados após terem sido colocados em mosaico: juntos ou individualmente.

O pacote de aprendizagem profunda que contém o modelo que será utilizado para classificar os objetos.

Os argumentos de modelo que estão presentes no modelo de aprendizagem profunda juntamente com os seus valores predefinidos.

O nome do campo que irá conter a etiqueta de classificação na camada de elemento de saída.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
