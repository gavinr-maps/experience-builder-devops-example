A ferramenta Geocodificar Locais a partir de Tabela converte endereços ou locais de uma tabela de entrada em coordenadas.

A tabela de entrada com os endereços ou locais a geocodificar. A entrada tem de ser uma tabela, um ficheiro .csv ou um ficheiro Excel.

O serviço de localizador a utilizar para a geocodificação. Caso o serviço de localizador que pretende utilizar não se encontre no menu pendente, contacte o seu administrador para que o serviço de localizador seja configurado para geocodificação em batch.

Especifica se os dados de entrada estão contidos num único campo ou em vários campos.

Os campos que serão utilizados pelo localizador para determinar as localizações dos dados. Assume-se que determinados campos estão nos dados, como o endereço, a cidade, o estado e o código postal.

O país em que os dados estão localizados. Se todos os endereços forem de um país, indique esse país para garantir os resultados mais exatos. Caso contrário, deixe este parâmetro por preencher.

Especifica os campos que serão incluídos no conjunto de dados de saída. As opções são as seguintes:

  • Tudo: todos os campos de saída do localizador disponíveis serão incluídos nos resultados do geocódigo.
  • Mínimo: um subconjunto de campos do localizador, bem como os campos da tabela de entrada, serão incluídos nos resultados.
  • Apenas localização: o campo Shape do localizador, bem como os campos da tabela de entrada, serão incluídos nos resultados.

O tipo de saída que será criado. A saída pode ser um serviço de elementos, um ficheiro .csv ou um ficheiro .xls.

O nome do item de saída onde os dados localizados serão armazenados.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
