A ferramenta Classificar Píxeis Utilizando Aprendizagem Profunda utiliza um modelo de aprendizagem profunda numa imagem de entrada para produzir um raster classificado.

A camada de imagens que será utilizada para classificar os píxeis.

Especifica como os itens de raster numa camada de imagens serão processados após terem sido colocados em mosaico: juntos ou individualmente.

O pacote de aprendizagem profunda que contém o modelo que será utilizado para classificar os píxeis.

Os argumentos de modelo que estão presentes no modelo de aprendizagem profunda juntamente com os seus valores predefinidos.

O nome da camada de imagens de saída que será classificada.

O nome da camada que será criada e adicionada ao mapa.

O tipo de saída que será criado. A saída pode ser uma camada de imagens em mosaico ou uma camada de imagens dinâmicas.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
