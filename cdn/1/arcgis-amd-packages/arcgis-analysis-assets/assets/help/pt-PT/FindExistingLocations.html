A ferramenta Procurar por Atributos e Localização cria uma nova camada das elementos ou partes de elementos que cumprem os critérios espaciais ou de atributos especificados.

A camada de pontos, linhas ou polígonos para a qual serão encontrados elementos, com base numa combinação de atributos e consultas espaciais.

Os critérios espaciais ou de atributos que serão utilizados para encontrar elementos. Clique no botão Criar nova consulta para definir um ou mais critérios espaciais ou de atributos. Esses elementos, ou partes de elementos, que cumprem todos os critérios serão escritos na camada de resultados.

Quando apenas uma parte de um elemento cumpre os critérios, especifica se o resultado inclui a totalidade do elemento ou apenas a parte do elemento gráfico que cumpre todos os critérios. Quando selecionada, apenas as partes do elemento que cumprem todos os critérios serão devolvidas. Quando está desselecionada, todo o elemento será devolvido desde que pelo menos uma parte do elemento cumpra todos os critérios.
