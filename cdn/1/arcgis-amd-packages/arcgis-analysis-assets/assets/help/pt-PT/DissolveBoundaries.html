A camada que contém elementos de polígono que serão dissolvidos.

Especifica o campo utilizado para dissolver polígonos com base em valores correspondentes.

Especifica se a saída pode conter elementos de polígono com múltiplas partes, ou seja, múltiplos polígonos definidos como um único elemento (ativada) ou pode conter apenas elementos individuais (desativada).

Determina se as estatísticas de resumo são calculadas para os polígonos dissolvidos.
