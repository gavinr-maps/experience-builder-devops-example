A ferramenta Converter Elemento para Raster converte dados de elementos num conjunto de dados raster.

Os dados de elementos que serão convertidos. A camada de entrada pode conter elementos de ponto, linha ou polígono.

O campo que será utilizado para atribuir valores ao raster de saída.

O tamanho da célula e unidade para os rasters de saída.

O nome da camada de raster que será criada e adicionada ao mapa.

O tipo de saída que será criado. A saída pode ser uma camada de imagens em mosaico ou uma camada de imagens dinâmicas.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
