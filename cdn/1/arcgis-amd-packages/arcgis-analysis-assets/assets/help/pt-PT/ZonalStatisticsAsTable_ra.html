A ferramenta Estatísticas de Zona como Tabela resume os valores de um raster dentro das zonas de outro conjunto de dados e comunica os resultados como uma tabela.

A entrada que define os limites das zonas, ou áreas, que serão resumidas.

O campo que contém os valores que definem cada zona.

A camada de raster que contém os valores das células que serão utilizadas para calcular as estatísticas.

Especifica o tipo de estatística que será calculada.

Especifica os valores percentuais que serão calculados.

Especifica o método que será utilizado quando o número de valores do raster de entrada a ser calculado for um número par.

Especifica como serão processados os valores de células do Raster de valor de entrada para dados circulares.

Se não for selecionada, serão calculadas estatísticas aritméticas (lineares) ordinárias. Se for selecionada, serão calculadas estatísticas circulares.

Especifica o valor que será utilizado para arredondar um valor linear para o intervalo de uma determinada estatística circular.

Especifica se as células NoData no Raster de valor de entrada serão ignoradas nos resultados das zonas em que se inserem.

Se for selecionada, as células NoData no raster de valor serão ignoradas no cálculo estatístico. Se não for selecionada, as células NoData não serão ignoradas e a zona inteira receberá o valor NoData no raster de saída.

Especifica como os rasters de entrada serão processados se forem multidimensionais.

Se não for selecionada, as estatísticas serão calculadas para todas as dimensões do conjunto de dados multidimensional de entrada. Se for selecionada, as estatísticas serão calculadas a partir da secção atual do conjunto de dados multidimensional de entrada.

O nome da camada de tabela de estatísticas de zona que será criada e adicionada ao mapa.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
