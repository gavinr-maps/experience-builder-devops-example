A ferramenta Agregar Pontos calcula uma contagem de pontos e outras estatísticas dentro de caixas ou polígonos especificados.

A camada de ponto que será agregada em áreas de resumo.

A camada de polígonos na qual os pontos serão agregados.

O tipo de elementos delimitadores dentro dos quais os pontos de entrada serão agregados. Pode ser uma camada de polígonos ou uma grelha de tesselação de caixas quadradas ou hexagonais que são geradas quando a ferramenta é executada.

O tamanho das caixas geradas. Para caixas quadradas, o número e unidades especificadas determinam a altura e o comprimento do quadrado. Para caixas hexagonais, o número e unidades especificados determinam a distância entre lados paralelos.

As unidades que serão utilizadas para especificar o tamanho da caixa.

Determina se os elementos que não têm pontos aparecem na camada de resultados. Se não for selecionada, os elementos sem pontos não aparecem no resultado. Se for selecionada, os elementos sem pontos aparecem no resultado.

Os campos de número ou data da camada de entrada que serão resumidos e as estatísticas que serão calculadas para cada campo.

O campo da camada de entrada que será utilizado para separar os dados em grupos distintos para cálculos estatísticos.

Calcule os grupos minoritários (menos dominantes) e maioritários (mais dominantes) dentro de cada área de resumo.

Calcular a percentagem de pontos pertencentes a cada grupo dentro de cada área de resumo.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
