A ferramenta Resumir nas Proximidades calcula estatísticas para elementos a uma distância especificada de outros elementos. A distância pode ser calculada com uma distância em linha reta ou um modo de viagem.

A camada a resumir a uma distância da camada nas proximidades.

A camada de ponto, linha ou polígono utilizada como localizações de início em torno da qual serão criadas áreas de delimitação para resumir os elementos nas proximidades.

O método com que serão criadas as áreas de delimitação. O método pode ser uma distância em linha reta ou um modo de viagem.

A distância ou o tempo que será utilizado para criar áreas de delimitação. Se forem adicionados vários valores, cada valor será utilizado para calcular as áreas de delimitação para todos os elementos de entrada.

As unidades que serão utilizadas para especificar a distância ou o tempo.

Especifique se a camada de resultados deve ser as áreas de delimitação (selecionada) ou a mesma geometria que a camada de entrada nas proximidades (não selecionada).

Calcule o tamanho total dos elementos de resumo em cada área delimitada. Para linhas, é calculado o comprimento total. Para polígonos, é calculada a área total.

As unidades que serão utilizadas para calcular o tamanho total dos elementos de resumo.

Os campos de número ou data da camada de entrada que serão resumidos e as estatísticas que serão calculadas para cada campo.

O campo da camada de entrada que será utilizado para separar os dados em grupos distintos para cálculos estatísticos.

Calcule os grupos minoritários (menos dominantes) e maioritários (mais dominantes) dentro de cada área de resumo.

Calcule a percentagem dos elementos de resumo que pertencem a cada grupo dentro de cada limite.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em O Meu Conteúdo onde os resultados serão guardados.
