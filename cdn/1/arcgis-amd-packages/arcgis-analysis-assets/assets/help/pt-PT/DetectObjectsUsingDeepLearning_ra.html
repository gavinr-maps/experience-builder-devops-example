A ferramenta Detetar Objetos Utilizando Aprendizagem Profunda utiliza um modelo de aprendizagem profunda numa camada de imagens de entrada para produzir uma camada de elementos que contém os objetos detetados.

A camada de imagens ou de elementos que será utilizada para detetar objetos.

Especifica como os itens de raster numa camada de imagens serão processados após terem sido colocados em mosaico: juntos ou individualmente.

O pacote de aprendizagem profunda que contém o modelo que será utilizado para detetar os objetos.

Os argumentos de modelo que estão presentes no modelo de aprendizagem profunda juntamente com os seus valores predefinidos.

Especifica se a supressão não máxima será executada para remover objetos duplicados.

O nome do campo que será adicionado à camada de elementos de saída para indicar o valor de confiança comunicado.

O nome do campo que será adicionado à camada de elementos de saída para indicar a etiqueta classificada para o objeto.

O rácio máximo permitido para cada elemento encontrado que se sobreponha a outro elemento. O valor predefinido é 0.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em O Meu Conteúdo onde o resultado será guardado.
