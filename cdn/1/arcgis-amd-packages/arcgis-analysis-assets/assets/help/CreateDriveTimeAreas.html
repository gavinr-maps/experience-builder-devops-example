The Generate Travel Areas tool creates areas that can be reached within a travel time or distance.

The point features around which areas will be calculated.

One or more point features that act as temporary restrictions (barriers) when traveling on the underlying streets. Travel is permitted on the street but not through the barrier.

One or more line features that act as temporary restrictions (barriers) when traveling on the underlying streets. The line barriers prohibit travel anywhere the barriers intersect the streets.

One or more polygon features that act as temporary restrictions (barriers) when traveling on the underlying streets. The polygon barriers prohibit travel anywhere the polygons intersect the streets.

The travel mode that will be used to create the travel areas. The travel mode can be based on distance or time.

The travel distance or time limit for each input point. If multiple values are added, each value will be used to calculate travel areas for all input points.

The units used to specify travel distance or time. The Cutoff value is interpreted in these units.

Determines whether travel areas are created for travel away from the input points or toward the input points.

Specifies the date and time that the traveler departs from or arrives at the input locations.

Determines how shared areas from different inputs are handled when they reach one another.

Specifies whether each output area will be one continuous polygon (unchecked) or include holes on streets that are not reachable but are fully surrounded by reachable streets (checked).

The name of the layer that will be created and added to the map.

Specifies whether the streets that are used to create the travel areas will be included as an output layer.

The name of the folder in My Content where the results will be saved.
