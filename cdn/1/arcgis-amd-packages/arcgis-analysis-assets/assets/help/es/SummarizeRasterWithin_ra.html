La herramienta Estadísticas zonales resume los valores de un ráster dentro de las zonas de otro dataset.

La entrada que define los límites de las zonas, o áreas, que se resumirán.

El campo que contiene los valores que definen cada zona.

La capa ráster que contiene los valores de celda que se utilizarán para calcular estadísticas.

Especifica el tipo de estadísticas que se calcularán.

Especifica el valor de percentil que se calculará.

Especifica el método que se utilizará si el número de valores del ráster de entrada calculado es par.

Especifica cómo se procesarán los valores de la celda Ráster de valor de entrada para los datos circulares.

Si no se activa, se calculan estadísticas aritméticas (lineales) ordinarias. Si se activa, se calculan estadísticas circulares.

Especifica el valor que se utilizará para redondear un valor lineal al rango de una estadística circular dada.

Especifica si las celdas NoData de Ráster de valor de entrada se ignorarán en los resultados de las zonas en las que se encuentran.

Si está activada, las celdas NoData del ráster de valores se ignorarán en el cálculo estadístico. Si está desactivada, las celdas NoData no se ignorarán y toda la zona recibirá el valor NoData en el ráster de salida.

Especifica cómo se procesan los rásteres de entrada si son multidimensionales.

Si se desactiva, las estadísticas se calcularán a partir de todas las dimensiones del dataset multidimensional de entrada. Si se activa, se calcularán estadísticas de la parte actual del dataset multidimensional de entrada.

El nombre de la capa ráster de estadísticas zonales que se creará y se agregará al mapa.

El tipo de salida que se creará. La salida puede ser una capa de imágenes en teselas o una capa de imágenes dinámicas.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
