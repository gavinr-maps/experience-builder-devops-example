La herramienta Clasificar píxeles con aprendizaje profundo utiliza un modelo de aprendizaje profundo entrenado en una imagen de entrada para generar un ráster clasificado.

La capa de imágenes que se utilizará para clasificar píxeles.

Especifica cómo se procesarán los elementos de ráster de una capa de imágenes después de crearlos en mosaico: juntos o individualmente.

El paquete de aprendizaje profundo que contiene el modelo que se utilizará para clasificar los píxeles.

Los argumentos de modelo que están presentes en el modelo de aprendizaje profundo junto con sus valores predeterminados.

El nombre de la capa de imágenes de salida que se clasificará.

El nombre de la capa que se creará y se agregará al mapa.

El tipo de salida que se creará. La salida puede ser una capa de imágenes en teselas o una capa de imágenes dinámicas.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
