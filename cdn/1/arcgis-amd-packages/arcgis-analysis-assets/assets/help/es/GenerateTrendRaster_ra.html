La herramienta Generar ráster de tendencia estima la tendencia de cada píxel a lo largo de una dimensión para una o varias variables de una capa de imágenes multidimensional.

La capa de imágenes multidimensional que se utilizará para generar la tendencia.

La dimensión a lo largo de la cual se extraerá una tendencia para la variable o variables seleccionadas en el análisis.

La variable o variables para las que se calcularán tendencias.

La unidad de tiempo que se utilizará para la duración de un periodo estacional al utilizar el tipo de línea Seasonal-Kendall.

El tipo de análisis de tendencia que se realizará en valores de píxel a lo largo de una dimensión.

La frecuencia o número de orden polinómico que se utilizarán en el ajuste de tendencia.

Especifica si los valores NoData o los que faltan se ignorarán en el análisis.

La longitud de la variación periódica que se modelará para el ciclo armónico.

La unidad de tiempo que se utilizará para la longitud del ciclo armónico.

Especifica si se calculará el error cuadrático medio (RMSE) de la línea de ajuste de tendencia.

Especifica si se calculará la estadística de idoneidad de ajuste de R cuadrado para la línea de ajuste de tendencia.

Especifica si se calculará la estadística de valor p para el coeficiente de pendiente de la línea de ajuste de tendencia.

El nombre de la capa que se creará y se agregará al mapa.

El tipo de salida que se creará. La salida puede ser una capa de imágenes en teselas o una capa de imágenes dinámicas.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
