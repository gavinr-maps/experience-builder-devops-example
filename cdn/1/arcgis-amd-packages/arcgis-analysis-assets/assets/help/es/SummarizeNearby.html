La herramienta Resumir cerca de calcula estadísticas para entidades dentro de una distancia especificada respecto de otras entidades. La distancia se puede calcular utilizando una distancia en línea recta o un modo de viaje.

La capa que se va a resumir dentro de una distancia de la capa cercana.

La capa de puntos, líneas o polígonos utilizada como ubicaciones de partida alrededor de las cuales se crearán áreas de delimitación para resumir las entidades cercanas.

El método con el que se crearán las áreas de delimitación. El método puede ser una distancia en línea recta o un modo de viaje.

La distancia o el tiempo que se utilizará para crear áreas de delimitación. Si se agregan varios valores, cada valor se utilizará para calcular áreas de delimitación para todas las entidades de entrada.

Las unidades que se utilizarán para especificar la distancia o el tiempo.

Especifique si la capa de resultados debe ser las áreas de delimitación (activada) o la misma geometría que la capa cercana de entrada (desactivada).

Calcule el tamaño total de las entidades resumidas en cada área de delimitación. Para las líneas, se calcula la longitud total. Para los polígonos, se calcula el área total.

Las unidades que se utilizarán para calcular el tamaño total de las entidades resumidas.

Campos de número o fecha de la capa de entrada que se va a resumir y estadísticas que se calcularán para cada campo.

Campo de la capa de entrada que se utilizará para separar los datos en grupos distintos para cálculos estadísticos.

Calcula los grupos minoritarios (menos dominantes) y mayoritarios (más dominantes) dentro de cada área de resumen.

Calcule el porcentaje de las entidades resumidas que pertenecen a cada grupo dentro de cada límite.

El nombre de la capa que se creará y se agregará al mapa.

El nombre de la carpeta de Mi contenido en la que se guardarán los resultados.
