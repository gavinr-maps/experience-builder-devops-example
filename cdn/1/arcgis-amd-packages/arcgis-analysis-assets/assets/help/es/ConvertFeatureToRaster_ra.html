La herramienta Convertir entidad a ráster convierte datos de entidad a un dataset ráster.

Los datos de entidad que se van a convertir. La capa de entrada puede contener entidades de puntos, líneas o de polígonos.

El campo que se utilizará para asignar los valores al ráster de salida.

El tamaño de celda y las unidades para los rásteres de salida.

El nombre de la capa ráster que se creará y se agregará al mapa.

El tipo de salida que se creará. La salida puede ser una capa de imágenes en teselas o una capa de imágenes dinámicas.
