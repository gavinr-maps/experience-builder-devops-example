La herramienta Generar áreas de viaje crea áreas a las que se pueda llegar en un tiempo o distancia de viaje.

Las entidades de puntos en torno a las cuales se calcularán las áreas.

Una o varias entidades de puntos que actúan como restricciones temporales (barreras) al viajar por las calles subyacentes. Se permite el desplazamiento por la calle, pero no por la barrera.

Una o varias entidades de línea que actúan como restricciones temporales (barreras) al viajar por las calles subyacentes. Las barreras de línea prohíben viajar a cualquier ubicación en la que las barreras se intersequen con las calles.

Una o varias entidades poligonales que actúan como restricciones temporales (barreras) al viajar por las calles subyacentes. Las barreras de polígono prohíben viajar a cualquier punto en que los polígonos intersequen las calles.

Modo de viaje que se utilizará para crear las áreas de viaje. El modo de viaje puede estar basado en la distancia o el tiempo.

La distancia de viaje o el límite de tiempo para cada punto de entrada. Si se agregan varios valores, cada valor se utilizará para calcular áreas de viaje para todos los puntos de entrada.

Las unidades utilizadas para especificar la distancia o el tiempo de viaje. El valor de Valor límite se interpreta en estas unidades.

Determina si se crean áreas de viaje para alejarse de los puntos de entrada o acercarse a ellos.

Especifica la fecha y la hora a la que el viajero sale de las ubicaciones de entrada o llega a ellas.

Determina cómo se manejan las áreas compartidas de puntos de entrada diferentes cuando estos confluyen.

Especifica si cada área de salida será un polígono continuo (desactivado) o si incluye agujeros en calles que no son accesibles pero están completamente rodeados por calles alcanzables (activadas).

El nombre de la capa que se creará y se agregará al mapa.

Especifica si las calles que se utilizarán para crear las áreas de viaje se incluirán como capa de salida.

El nombre de la carpeta de Mi contenido en la que se guardarán los resultados.
