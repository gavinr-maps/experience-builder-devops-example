La herramienta Buscar ubicaciones similares evalúa la similitud entre las ubicaciones candidatas y una ubicación de referencia en función de una serie de criterios y clasifica las ubicaciones candidatas de más a menos en cuanto a similitud.

Capa de puntos, líneas o polígonos que contiene las ubicaciones de referencia que coincidirán. Si la capa incluye varias ubicaciones, se hará un promedio de todas las ubicaciones para crear una sola ubicación de referencia.

Capa de puntos, líneas o polígonos que contiene las ubicaciones candidatas de búsqueda. Todas las ubicaciones candidatas de búsqueda se compararán con la ubicación de referencia según el valor de Selección de criterios especificado y se clasificarán de más a menos en cuanto a similitud.

Campos numéricos que se utilizarán para evaluar la similitud entre la ubicación de referencia y cada ubicación candidata de búsqueda. Cada valor de Selección de criterios tendrá un valor de Campo de capa de referencia y un valor de Campo de capa candidata. Los valores de Campo de capa de referencia y Campo de capa candidata son campos coincidentes en los valores de Capa de ubicaciones de referencia y Capa de candidatos de búsqueda.

Campo Capa de ubicaciones de referencia que se corresponde con el campo Capa de candidatos de búsqueda especificado por el valor de Campo de capa candidata. Los valores de estos campos coincidentes se compararán para evaluar la similitud entre la ubicación de referencia y cada ubicación candidata de búsqueda.

Campo Capa de candidatos de búsqueda que se corresponde con el campo Capa de ubicaciones de referencia especificado por el valor de Campo de capa de referencia. Los valores de estos campos coincidentes se compararán para evaluar la similitud entre la ubicación de referencia y cada ubicación candidata de búsqueda.
