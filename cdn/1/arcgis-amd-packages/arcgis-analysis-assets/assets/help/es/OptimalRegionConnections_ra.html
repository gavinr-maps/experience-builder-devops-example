La herramienta Conexiones de región óptimas calcula la red de conectividad óptima entre dos o más regiones de entrada.

La capa de entidades o ráster que identifica las regiones que se conectarán mediante la red óptima.

La capa de entidades o ráster que identifica los obstáculos que se deben evitar a la hora de generar las rutas.

Para una capa ráster, las celdas que tengan algún valor (incluido cero) se tratarán como barreras.

La capa ráster que define la impedancia o el coste de moverse por cada celda.

El valor de cada ubicación de celda representa la distancia de coste por unidad para moverse a través de la celda. No puede ser negativo ni cero.

Especifica si la distancia se calculará utilizando un método planar (Tierra plana) o geodésico (elipsoide).

Especifica si las rutas seguirán y se conectarán dentro de las regiones de entrada.

El nombre de la capa de red óptima de rutas que conecta cada una de las regiones de entrada que se crearán y agregarán al mapa.

El nombre de la capa de líneas de conexión vecina que se creará y se agregará al mapa. La capa identifica todas las rutas de cada región hacia sus vecinos de coste o más próximos.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
