La herramienta Generar teselaciones crea una cuadrícula de polígonos basada en la extensión, la forma y el tamaño especificados.

La forma de las teselaciones que se crearán.

Especifica si el tamaño de teselación se definirá por su área (Área) o mediante un valor de distancia (Distancia).

El tamaño (área o distancia) de las teselaciones generadas. Para los cuadrados, hexágonos, triángulos y rombos, la distancia determina la altura de las teselaciones. Para los hexágonos transversales, la distancia determina el ancho de las teselaciones.

Resolución H3 de las teselaciones. Rangos de resolución de 0 a 15. Con cada tamaño de resolución creciente, el área de los polígonos disminuirá una séptima parte del tamaño a la resolución inferior.

Especifica las unidades que se utilizarán para el tamaño de teselación.

La extensión que cubrirán las teselaciones.

Para las capas de entidades, especifica si las teselaciones cubrirán toda la extensión o solo cubrirán las áreas donde las teselaciones se intersecan con las entidades.

El nombre de la capa que se creará y se agregará al mapa.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
