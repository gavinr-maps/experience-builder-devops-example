La herramienta Cuenca hidrográfica determina el área de la cuenca contribuyente sobre un conjunto de celdas de un ráster.

La dirección del flujo desde cada celda.

Se debe utilizar el método D8 para determinar la dirección del flujo.

Las ubicaciones sobre las que se determinará el área de la cuenca contribuyente.

El campo que se utilizará para asignar los valores a las ubicaciones de punto de fluidez.

El nombre de la capa ráster de cuenca hidrográfica que se creará y se agregará al mapa. La capa contiene distintas cuencas hidrográficas representadas por diferentes valores de celda desde el Campo de punto de fluidez.

El tipo de salida que se creará. La salida puede ser una capa de imágenes en teselas o una capa de imágenes dinámicas.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
