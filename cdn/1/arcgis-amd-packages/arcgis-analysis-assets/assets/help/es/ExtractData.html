La herramienta Extraer datos selecciona entidades en una extensión y crea un elemento en su contenido.

Las capas se extraerán. Entre las capas de entrada admitidas se incluyen las capas que le pertenecen y las capas en las que se ha habilitado la exportación.

Extensión de las entidades que se extraerán.

Especifica si se recortarán las entidades que se crucen con la extensión. Cuando se desactiva, las entidades que se cruzan con la extensión se extraen. Si se activa, las entidades que se cruzan con la extensión se recortan y solo se extraen partes de esas entidades que están dentro de la extensión.

Nombre del elemento de salida donde se guardará el archivo extraído.
