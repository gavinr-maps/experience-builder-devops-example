La herramienta Detectar cambios con aprendizaje profundo utiliza un modelo de aprendizaje profundo para detectar cambios entre dos capas ráster.

Capa de imágenes de entrada anterior al cambio que se utilizará como base durante la comparación.

Capa de imágenes de entrada posterior al cambio que se utilizará en la comparación con la original.

Capa de imágenes clasificadas de salida que muestra el cambio.

Los argumentos de modelo que están presentes en el modelo de aprendizaje profundo junto con sus valores predeterminados.

Paquete de aprendizaje profundo que contiene el modelo que se utilizará para detectar cambios entre capas de imágenes.

El tipo de salida que se creará. La salida puede ser una capa de imágenes en teselas o una capa de imágenes dinámicas.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
