La herramienta Resumir dentro de calcula estadísticas para entidades dentro de bins o polígonos especificados.

La capa que se resumirá.

El tipo de entidades de delimitación en las que se resumirán las entidades de entrada. Puede ser una capa de polígonos o una cuadrícula de teselación de bins cuadrados o hexagonales que se generan cuando se ejecuta la herramienta.

La capa de polígonos dentro de la cual se resumirán las entidades de entrada.

El tamaño de los bins generados. Para los bins cuadrados, el número y las unidades especificados determinan la altura y la longitud del cuadrado. Para los bins hexagonales, el número y las unidades especificados determinan la distancia entre los lados paralelos.

Las unidades que se utilizarán para especificar el tamaño de bin.

Calcule el tamaño total de las entidades resumidas en cada polígono de delimitación. Para las líneas, se calcula la longitud total. Para los polígonos, se calcula el área total.

Las unidades que se utilizarán para calcular el tamaño total de las entidades resumidas.

Campos de número o fecha de la capa de entrada que se va a resumir y estadísticas que se calcularán para cada campo.

Campo de la capa de entrada que se utilizará para separar los datos en grupos distintos para cálculos estadísticos.

Calcula los grupos minoritarios (menos dominantes) y mayoritarios (más dominantes) dentro de cada área de resumen.

Calcule el porcentaje de las entidades resumidas que pertenecen a cada grupo dentro de cada área de resumen.

El nombre de la capa que se creará y se agregará al mapa.

El nombre de la carpeta de Mi contenido en la que se guardarán los resultados.
