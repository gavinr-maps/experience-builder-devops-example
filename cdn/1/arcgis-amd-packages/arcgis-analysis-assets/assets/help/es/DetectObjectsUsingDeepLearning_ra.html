La herramienta Detectar objetos con aprendizaje profundo utiliza un modelo de aprendizaje profundo en una capa de imágenes de entrada para producir una capa de entidades que contiene los objetos detectados.

Las imágenes o la capa de entidades que se utilizarán para detectar objetos.

Especifica cómo se procesarán los elementos de ráster de una capa de imágenes después de crearlos en mosaico: juntos o individualmente.

El paquete de aprendizaje profundo que contiene el modelo que se utilizará para detectar los objetos.

Los argumentos de modelo que están presentes en el modelo de aprendizaje profundo junto con sus valores predeterminados.

Especifica si se realizará la supresión no máxima para quitar los objetos duplicados.

El nombre del campo que se agregará a la capa de entidades de salida para indicar el valor de confianza notificado.

El nombre del campo que se agregará a la capa de entidades de salida para indicar la etiqueta clasificada del objeto.

El ratio máximo permitido para cada entidad encontrada que se superpone a otra entidad. El valor predeterminado es 0.

El nombre de la capa que se creará y se agregará al mapa.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
