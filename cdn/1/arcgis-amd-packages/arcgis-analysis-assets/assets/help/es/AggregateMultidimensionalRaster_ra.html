La herramienta Agregar ráster multidimensional genera una capa de imágenes multidimensional combinando variables de una capa de imágenes multidimensional existente a lo largo de una dimensión.

La capa de imágenes multidimensional que se agregará.

La dimensión de la capa de imágenes multidimensional a lo largo de la cual se agregará la variable.

Indica si la capa tiene valores de dimensión.

La variable de la capa de imágenes multidimensional que se agregará a lo largo de la dimensión definida.

El método matemático que se usará para agregar las divisiones de la capa de imágenes multidimensional.

El valor de percentil que se utilizará para los cálculos una vez especificado el método de agregación.

El método de interpolación de percentil que se utilizará si el número de valores de la capa de imágenes de entrada es par.

La función ráster personalizada que se usará para calcular los valores de píxel de las capas de imágenes agregadas.

El intervalo de dimensión que se utilizará para agregar los datos multidimensionales.

Cada palabra clave especifica la unidad de tiempo para la que se agregará cada estadística. La lista de palabras clave incluye: Cada hora, Cada día, Cada semana, Cada mes, Cada trimestre y Cada año.

El tamaño del intervalo que se usará al agregar a lo largo de la dimensión.

Las unidades de medida para el valor del intervalo.

La tabla de valores que se usará al agregar a lo largo de la dimensión especificada. Los valores mínimo y máximo especifican el rango que se incluirá.

El valor mínimo del rango de intervalos.

El valor máximo del rango de intervalos.

Indica si los valores NoData se ignorarán en la agregación.

El nombre de la capa que se creará y se agregará al mapa.

El tipo de salida que se creará. Las salidas pueden ser capas de imágenes en teselas o capas de imágenes dinámicas.

El nombre de la carpeta de Mi contenido en la que se guardará el resultado.
