Verktyget Flödesackumulering genererar ett raster med ackumulerade flöden till varje cell från ett flödesriktningsraster.

Det raster som definierar riktningen för flödet ut ur varje cell.

Det kan skapas med verktyget Flödesriktning.

Rasterlagret som definierar andelen flöde som bidrar till flödesackumulering i varje cell.

Anger typen av flödesriktningsraster för indata.

Anger datatypen för utdata för flödesackumuleringsrastret.

Rasterlagret för flödesackumulering som skapas och läggs till i kartan.

Typ av utdata som skapas. Utdata kan vara bildlager i tileform eller dynamiska bildlager.

Namnet på mappen i Mitt innehåll där resultaten ska sparas.
