Verktyget Aggregera flerdimensionellt raster genererar ett flerdimensionellt bildlager genom att kombinera befintliga flerdimensionella bildlagervariabler längs en dimension.

Det flerdimensionella bildlager som kommer att aggregeras.

Dimensionen för det flerdimensionella bildlager längs vilken variabeln kommer att aggregeras.

Anger om lagret har dimensionsvärden.

Variabeln för det flerdimensionella bildlagret som aggregeras längs den definierade dimensionen.

Den matematiska metod som används för att aggregera sektorerna i det flerdimensionella bildlagret.

Percentilvärdet som används för beräkningar när aggregeringsmetoden anges.

Metoden för interpolering av percentil som ska användas när antalet värden från indatabildlagret är jämnt.

Den anpassade rasterfunktion som används för att beräkna pixelvärden för aggregerade bildlager.

Dimensionsintervallet som används för att aggregera flerdimensionella data.

Varje nyckelord anger den tidsenhet som varje statistik ska aggregeras för. Listan över nyckelord inkluderar timvis, dagligen, veckovis, månadsvis, kvartalsvis eller årsvis.

Storleken på det intervall som kommer att användas vid aggregering längs dimensionen.

Mätenheterna för intervallvärdet.

Värdetabellen som kommer att användas vid aggregering längs angiven dimension. Minimi- och maximivärden anger intervallet som ska inkluderas.

Minsta värde för intervallet.

Maximalt värde för intervallet.

Anger huruvida NoData-värden ignoreras i aggregeringen.

Namnet på lagret som skapas och läggs till i kartan.

Typ av utdata som skapas. Utdata kan vara bildlager i tileform eller dynamiska bildlager.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
