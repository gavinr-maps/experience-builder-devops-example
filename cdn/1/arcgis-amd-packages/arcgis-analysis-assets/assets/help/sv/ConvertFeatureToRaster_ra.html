Verktyget Konvertera geoobjekt till raster konverterar geoobjektdata till ett rasterdataset.

De geoobjektdata som ska konverteras. Indatalagret kan innehålla punkt-, linje- eller polygongeoobjekt.

Det fält som ska användas för att tilldela värden till utdatarastret.

Cellstorleken och enheten för utdatarastren.

Namnet på rasterlagret som skapas och läggs till i kartan.

Typ av utdata som skapas. Utdata kan vara ett bildlager i tileform eller ett dynamiskt bildlager.
