Verktyget Geodetiskt siktfält identifierar platserna på en rasteryta som är synliga för indataobservatörsplatserna.

Det höjdraster som siktfältet ska beräknas på.

Observatörsplatserna som siktfältet ska beräknas för.

Optimerar beräkningen av siktfältet baserat på antingen snabbhet eller noggrannhet.

Anger det maximala siktavståndet för beräkningen av siktfältet.

Avståndet kan definieras efter ett värde och dess enheter, eller från ett fält i Observatörspunkt- eller linjegeoobjekt som indata.

Anger det minimala siktavståndet för beräkningen av siktfältet.

Avståndet kan definieras efter ett värde och dess enheter, eller från ett fält i Observatörspunkt- eller linjegeoobjekt som indata.

Anger typ för maximala och minimala siktavstånd.

Om alternativet inte är markerat hanteras visningsavstånden som tvådimensionella euklidiska avstånd. Om alternativet är markerat hanteras visningsavstånden som tredimensionella avstånd.

Anger ythöjden för observatörsplatserna.

Höjden kan definieras efter ett värde och dess enheter, eller från ett fält i Observatörspunkt- eller linjegeoobjekt som indata.

Anger höjden för observatörna över marken.

Höjden kan definieras efter ett värde och dess enheter eller från ett fält i Observatörspunkt- eller linjegeoobjekt som indata.

Anger höjden för målen på marken.

Höjden kan definieras efter ett värde och dess enheter eller från ett fält i Observatörspunkt- eller linjegeoobjekt som indata.

Namnet på rasterlagret för siktfält som skapas och läggs till i kartan. Det här lagret innehåller de synliga områden där observatörerna kan se, och cellvärdena representerar antalet observatörer som kan visa cellerna.

Namnet på rasterlagret över marknivå (AGL) som skapas och läggs till i kartan.

Typ av utdata som skapas. Utdata kan vara ett bildlager i tileform eller ett dynamiskt bildlager.
