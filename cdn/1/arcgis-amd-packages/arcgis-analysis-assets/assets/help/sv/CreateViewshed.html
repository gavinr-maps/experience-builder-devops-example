Verktyget Skapa siktområde använder Esris tjänst för höjdanalys för att identifiera områden som är synliga från angivna observatörsplatser.

Punktgeoobjekten som används som observatörsplatser.

Anger höjden för observatörna över marken.

Mätenheterna som används för parametern Observatörshöjd.

Anger höjden för målen på marken.

Mätenheterna som används för parametern Målhöjd.

Anger det maximala siktavståndet för beräkningen av siktfältet.

Mätenheterna som används för parametern Maximalt siktavstånd.

Namnet på polygonlagret för siktfält som skapas och läggs till i kartan. Det här lagret innehåller de synliga områden där observatörerna kan se, och antalet observatörer registreras i frekvensfälten.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
