Verktyget Klassificera pixlar med djupinlärning använder en tränad djupinlärningsmodell på en indatabild för att skapa ett klassificerat raster.

Bildlagret som används för att klassificera pixlar.

Anger hur rasterobjekt i ett bildlager ska bearbetas när de mosaikbehandlats: tillsammans eller var för sig.

Det djupinlärningspaket som innehåller modellen som ska användas för att klassificera pixlarna.

Modellargumenten som finns i djupinlärningsmodellen tillsammans med sina standardvärden.

Namnet på utdatabildlagret som ska klassificeras.

Namnet på lagret som skapas och läggs till i kartan.

Typ av utdata som skapas. Utdata kan vara ett bildlager i tileform eller ett dynamiskt bildlager.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
