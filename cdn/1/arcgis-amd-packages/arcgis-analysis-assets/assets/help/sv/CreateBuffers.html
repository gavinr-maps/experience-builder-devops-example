Verktyget Skapa buffertar skapar polygoner som täcker ett angivet avstånd från indatageoobjekt.

Det punkt-, linje- eller polygongeoobjekt som ska buffras.

Anger om buffertavståndet definieras av en konstant (Värde) eller ett numeriskt fält (Fält).

Avståndet som används för att skapa bufferten. Om flera värden läggs till används varje värde för att skapa en buffert vid varje indatageoobjekt.

Fältet som kommer att innehålla buffertavståndet för varje geoobjekt.

Enheterna som används för att ange buffertavståndet.

Anger hur buffertar från olika indatageoobjekt ska hanteras när de överlappar.

Anger om buffertar med flera avstånd ska vara koncentriska buffertar som överlappar (Överlappande diskar ) eller ringar som inte överlappar (Distinkta ringar).

För linjegeoobjekt anger detta om bufferten skapas på båda sidor av linjen (Runt), bara på vänster sida (Vänster) eller bara på höger sida (Höger).

För polygongeoobjekt anger detta om indatapolygongeoobjekten inkluderas i bufferten (Inkludera) eller exkluderas från bufferten (Exkludera).

Anger om formen på bufferten i slutet av ett linjegeoobjekt är rund (Rund) eller platt (Platt).

Namnet på lagret som skapas och läggs till i kartan.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
