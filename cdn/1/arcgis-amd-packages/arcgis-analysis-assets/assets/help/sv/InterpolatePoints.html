Verktyget Interpoleringspunkter förutspår värden på nya platser baserat på punktmätningar. Verktyget skapar geoobjektutdata.

Välj punktlagret för platserna som ska interpoleras.

Välj det fält som innehåller de uppmätta värdena som ska interpoleras.

Välj om beräkningshastighet eller noggrannhet och precision ska prioriteras. Mer noggranna och precisa förutsägelser tar längre tid att beräkna.

Välj om ett lager med standardfel för interpoleringsförutsägelserna ska inkluderas i utdata.

Välj hur klassbrytningsvärdena ska beräknas.

Ange antalet klasser (intervall med förutsagda värden) som ska ingå i resultatlagret. Varje klass definierar gränsen för en polygon och representerar konturerna för de förutsagda värdena. Klassbrytningsvärdena fastställs av klassificeringstypen.

Ange de klassbrytningsvärden som ska användas.

Välj ett polygonlager som ska användas för att definiera ett undersökningsområde. Resultaten kommer att klippas så att de bara visas inom undersökningsområdet.

Välj ett punktlager som representerar platserna som ska användas för att förutsäga värden. Om det anges skapas ett punktlager som innehåller förutsagda värden på de angivna platserna.

Namnet på lagret som skapas och läggs till i kartan.

Namnet på mappen i Mitt innehåll där resultaten ska sparas.
