Verktyget Lös upp gränser hittar polygoner som överlappar eller delar en gemensam gräns och sammanfogar dem så att en enda polygon skapas.

Lagret som innehåller polygongeoobjekt som ska lösas upp.

Avgör om verktyget bara ska lösa upp polygonerna som överlappar eller delar en gemensam gräns och har samma fältvärde (aktiverat) eller alla polygoner som överlappar eller delar en gemensam gräns (inaktiverat).

Anger fältet som används för att lösa upp polygoner baserat på matchande värden.

Anger om utdata kan innehålla flerdelade geoobjekt, vilket innebär flera polygoner som definieras som ett enda geoobjekt (aktiverat) eller bara kan innehålla enskilda geoobjekt (inaktiverat).

Avgör om sammanfattningsstatistik beräknas för de upplösta polygonerna.

Nummer- eller datumfält från indatalagret som ska summeras, och statistiken som ska beräknas för varje fält.

Namnet på lagret som skapas och läggs till i kartan.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
