Miljöinställningar för analys i Map Viewer.

Anger koordinatsystem för analysen och resultatlagret.

Anger det område som ska användas för analys.

Justerar utbredningen för utdata så att den matchar celljusteringen för det angivna snapprasterlagret.

Anger cellstorlek eller upplösning som ska användas för analys och för att skapa utdatarasterlagret.

Anger ett masklager i vilket endast de celler som ligger inom maskområdet kommer att användas för analys.

Anger hur pixelvärden ska interpoleras när indata- och utdatalager inte överensstämmer.

Anger vilken typ av processor som ska användas. Alternativen är CPU, GPU och Automatisk. Om Automatisk har angivits definierar verktyget processortypen.

Anger antalet rasterbearbetningstjänstinstanser som används under körningen av verktyget. Värden som stöds inkluderar tomma strängar (verktyget fastställer antalet rasterbearbetningstjänstinstanser som ska användas), heltal (0 eller positiva heltal) och procenttal. Procental måste inkludera ett procenttecken (%) efter procentvärdet.

Avgör om ett varningsmeddelande ska visas om det beräknade antalet krediter överstiger det specifika tröskelvärdet (aktiverat) eller om analysen körs utan att visa ett meddelande oavsett den beräknade kreditförbrukningen (inte aktiverat). Analysen följer fortfarande kreditbudgeteringsinställningarna för organisationen.

Anger tröskelvärdet för när ett varningsmeddelande ska visas baserat på den beräknade kreditförbrukningen från att köra analysen. Analysen kan fortfarande köras om beräkningen överstiger gränsen.
