Verktyget Hitta punktkluster identifierar kluster av punktgeoobjekt i bakgrundsbrus baserat på deras spatiala fördelning.

Indatapunktlagret som grupperas till kluster baserat på punktgeoobjektens spatiala fördelning.

Metoden som används för att identifiera kluster.

Minimalt antal punktgeoobjekt som betraktas som ett kluster.

Anger maximalt avstånd runt varje punktgeoobjekt för sökning efter punkter som kan inkluderas i klustret. Om fältet lämnas tomt kommer värdet för Sökavstånd att vara det högsta kärnavståndet i lagret, exklusive den översta procenten av kärnavstånden.

Anger enheterna som används för avstånd.

Ett heltal mellan 0 och 100 som fastställer kompaktheten hos klustren. Om fältet lämnas tomt kommer värdet Klusterkänslighet att beräknas med Kullback-Leibler-divergensen.

Namnet på lagret som skapas och läggs till i kartan.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
