Verktyget Generera resområden skapar områden som kan nås inom en restid eller avstånd.

Punktfunktionerna runt de områden som ska beräknas.

Ange ett eller flera punktgeoobjekt som fungerar som tillfälliga restriktioner (hinder) vid resor på de underliggande gatorna. Det är tillåtet att färdas på gatan men inte genom hindret.

Ange ett eller flera linjegeoobjekt som fungerar som tillfälliga restriktioner (hinder) vid resor på de underliggande gatorna. Linjehindren förbjuder resor överallt där hindren korsar gatorna.

Ett eller flera polygongeoobjekt som fungerar som tillfälliga restriktioner (hinder) vid resor på de underliggande gatorna. Polygonhindren förbjuder resor överallt där polygonerna korsar gatorna.

Färdläget som kommer att användas för att skapa resområden. Färdläget kan baseras på avstånd eller tid.

Resavståndet eller tidsgränsen för varje indatapunkt. Om flera värden läggs till används varje värde för att beräkna resområden för alla indatapunkter.

Enheterna som används för att ange resavstånd eller -tid. Gränsvärdet tolkas i dessa enheter.

Avgör om resområden skapas för resa bort från indatapunkterna eller mot indatapunkterna.

Anger datum och tid då resenären avreser från eller anländer till platserna för indata.

Avgör hur delade områden från olika indata ska hanteras när de når varandra.

Anger om varje utdataområde ska vara en kontinuerlig polygon (omarkerat) eller inkludera hål i gatorna som inte är nåbara men som omges helt av nåbara gator (markerat).

Namnet på lagret som skapas och läggs till i kartan.

Anger om gatorna som används för att skapa resområden inkluderas som ett utdatalager.

Namnet på mappen i Mitt innehåll där resultaten ska sparas.
