Typen av begränsningsgeoobjekt inom vilka indatapunktgeoobjekten aggregeras. Detta kan vara ett polygonlager (polygonlager) eller ett tesselationsrutnät med antingen fyrkanter (fisknätsceller) eller hexagoner (hexagonceller).

Det lager som innehåller polygongeoobjekt i vilket indatapunktgeoobjekten aggregeras.

Det polygonlager som kommer att definiera gränsen för analysen. Punktgeoobjekt inom polygonen inkluderas i hotspot-analysen.

Anger hur man delar upp Analysfält-värden eller aggregerat punktantal. Alternativet Fält delar upp Analysfält-värden efter värdena för ett angivet fält i parametern Indatalager. Alternativet Berikningsdata berikar geoobjekten eller aggregeringsobjekten med folkmängdsdata och delar upp Analysfält-värden eller aggregerat punktantal efter folkmängd. Det här alternativet förbrukar ytterligare krediter.
