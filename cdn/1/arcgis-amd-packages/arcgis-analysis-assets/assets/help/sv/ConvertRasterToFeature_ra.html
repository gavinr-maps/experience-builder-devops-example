Verktyget Konvertera raster till geoobjekt vektoriserar ett raster.

Det indataraster som ska konverteras till geoobjekt.

Fältet som innehåller värdena som ska konverteras.

Typen av geoobjekt (punkt, linje eller polygon) som ska skapas i utdatalagret.

Anger om linjer eller polygoner ska förenklas (jämnas ut).

Anger om utdatapolygoner ska bestå av endelade eller flerdelade geoobjekt.

Flerdelade geoobjekt bearbetas vanligen snabbare. Men om utdata innehåller ett mycket stort antal geoobjekt kanske det inte går snabbare.

Den brytpunktsgräns som används för att dela upp en polygon i mindre polygoner.

Om den lämnas tom delas inte utdatapolygonerna.

Namnet på geoobjektlagret som skapas och läggs till i kartan.
