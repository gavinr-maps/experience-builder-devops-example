Verktyget Berika lager berikar indatageoobjekten med demografi och landskapsdata.

Punkt-, linje- eller polygongeoobjekten som berikas med ytterligare data.

Variablerna som läggs till som attribut i indatageoobjekten.

Anger hur området runt indatapunkt- eller linjegeoobjekten definieras. Hela området runt varje geoobjekt berikas.

Ett numeriskt värde som anger avstånd eller tid runt varje indatageoobjekt som området kommer att utöka.

Enheterna som används för att ange avstånd eller tid som det berikade området utökas runt varje indatgeoobjekt.

Anger om utdatalagret ska innehålla polygongeoobjekt definierade av inställningar för Berikningsområden (markerat) eller indatageoobjekt (omarkerat).
