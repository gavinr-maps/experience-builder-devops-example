Verktyget Planera rutter skapar resrutter med flera stopp för en fordonsflotta.

De punktgeoobjekt som anger var fordonen startar sina rutter.

De punktgeoobjekt som fordonen måste besöka under rutternas gång. De här punkterna blir till tilldelade eller icke tilldelade stopp i utdata.

Avgör om fordonen måste återvända till sin startplats (aktiverat) eller sluta på angivna platser (inaktiverat).

De punktgeoobjekt som anger var fordonen slutar sina rutter. Om lagret bara inkluderar en punkt dirigeras alla fordon till en gemensam slutpunkt. Om lagret inkluderar flera punkter dirigeras varje fordon till en unik slutplats.

Fältet i startlagret som på ett unikt sätt identifierar de enskilda rutterna, fordonen eller förarna. Om fordonen startar på flera platser och slutar på flera platser måste rutt-ID valda för start- och slutlagren matcha.

Fältet i slutlagret som på ett unikt sätt identifierar de enskilda rutterna, fordonen eller förarna. Om fordonen startar på flera platser och slutar på flera platser måste rutt-ID valda för start- och slutlagren matcha.

Ange ett eller flera punktgeoobjekt som fungerar som tillfälliga restriktioner (hinder) vid resor på de underliggande gatorna. Det är tillåtet att färdas på gatan men inte genom hindret.

Ange ett eller flera linjegeoobjekt som fungerar som tillfälliga restriktioner (hinder) vid resor på de underliggande gatorna. Linjehindren förbjuder resor överallt där hindren korsar gatorna.

Ett eller flera polygongeoobjekt som fungerar som tillfälliga restriktioner (hinder) vid resor på de underliggande gatorna. Polygonhindren förbjuder resor överallt där polygonerna korsar gatorna.

Färdmetoden som rutterna skapas med. Färdläget kan baseras på avstånd eller tid.

Datum och avgångstid för fordon.

Maximalt antal fordon som är tillgängliga för att stanna vid stoppen. Om både startlagret och slutlagret innehåller ett geoobjekt kan du ange upp till 100 fordon. Om inget av lagren innehåller fler än ett geoobjekt kan standardvärdet inte redigeras.

Det maximala antalet stopp som varje fordon kan göra på en rutt. Du kan ange ett värde mellan 1 och 200.

Ange den förväntade genomsnittliga servicetiden spenderad vid varje stopp.

Bestämmer om den maximala ruttiden tillämpas på fordon. Ruttiden inkluderar förväntad resetid och servicetider spenderade vid stopp längs rutten.

Ange den maximala tiden en rutt får ta från start till slut.

Namnet på lagret som skapas och läggs till i kartan.

Anger om varje resultatrutt sparas som ett ruttlager.

Namnet på mappen i Mitt innehåll där resultaten ska sparas.
