Verktyget Flerdimensionella huvudkomponenter transformerar de flerdimensionella bildlagren till ett reducerat antal komponenter som utgör variansen för data, så att spatiala och tidsmässia mönster enkelt kan identifieras.

Det flerdimensionella bildlager som ska analyseras.

Den metod som ska användas för att utföra huvudkomponentanalysen. I alternativen ingår Dimensionsreducering och Spatial reducering.

Dimensionen för det flerdimensionella bildlager längs vilken variabeln kommer att analyseras.

Namnet på utdatabildlagret eller -tabellagret som visar huvudkomponenterna som hittas beroende på följande val av läge:

  • Dimensionsreduceringsutdata blir ett multibandsbildlager.
  • Utdata för spatial reducering blir ett tabellager.

Namnet på utdatabildlagret eller -tabellagret som visar supportdata i huvudkomponentsvalet beroende på följande val av läge:

  • Utdata för dimensionsreducering blir ett tabellager.
  • Utdata för spatial reducering blir ett bildlager.

Namnet på tabellen med Eigen-värden, som indikerar variansprocenttalet för varje komponent.

Namnet på variabeln i det flerdimensionella indatabildlagret som används i beräkningen.

Antalet huvudkomponenter som ska beräknas, oftast färre än antalet indataraster.

Namnet på lagret som skapas och läggs till i kartan.

Typ av utdata som skapas. Utdata kan vara bildlager i tileform eller dynamiska bildlager.

Namnet på mappen i Mitt innehåll där resultaten ska sparas.
