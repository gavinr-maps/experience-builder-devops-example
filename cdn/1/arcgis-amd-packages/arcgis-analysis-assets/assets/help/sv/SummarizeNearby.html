Verktyget Summera nära beräknar statistik för geoobjekt inom ett angivet avstånd från andra geoobjekt. Avstånd kan beräknas med ett avstånd med ett rak linje-avstånd eller ett färdläge.

Lagret som ska summeras inom ett avstånd från det närliggande lagret.

Punkt-, linje- eller polygonlagret som används som startpunkter som begränsningsområden ska skapas runt för summering av närliggande geoobjekt.

Metoden som begränsningsområden ska skapas med. Metoden kan vara rak linje-avstånd eller ett färdläge.

Avstånd eller tid som används för att skapa begränsningsområden. Om flera värden läggs till används varje värde för att beräkna begränsningsområden för alla indatageoobjekt.

Enheterna som används för att ange avstånd eller tid.

Ange om resultatlagret ska vara begränsningsområden (markerat) eller samma geometri som indatalager i närheten (omarkerat).

Beräkna den totala storleken på geoobjekten som är summerade i varje begränsningsområde. För linjer beräknas den totala längden. För polygoner beräknas den totala ytan.

Enheterna som används för att beräkna den totala storleken på summerade geoobjekt.

Nummer- eller datumfält från indatalagret som ska summeras, och statistiken som ska beräknas för varje fält.

Fältet från indatalagret som ska användas för att separera data i distinkta grupper för statistiska beräkningar.

Beräkna minoritetsgrupp (minst dominant) och majoritetsgrupp (mest dominant) inom varje sammanfattningsområde.

Beräkna procentandelen summerade geoobjekt som tillhör varje grupp inom varje gräns.

Namnet på lagret som skapas och läggs till i kartan.

Namnet på mappen i Mitt innehåll där resultaten ska sparas.
