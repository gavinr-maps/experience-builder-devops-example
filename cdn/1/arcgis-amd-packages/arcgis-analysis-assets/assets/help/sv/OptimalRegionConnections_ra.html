Verktyget Optimala regionanslutningar beräknar det optimala anslutningsnätverket mellan två eller flera indataregioner.

Det raster eller geoobjektlager som identifierar regionerna som ska anslutas till det optimala nätverket.

Det raster eller geoobjektlager som identifierar hinder som rutter måste skapas runt.

För ett rasterlager behandlas celler som har ett värde (inklusive noll) som ett hinder.

Rasterlagret som definierar impedansen eller kostnaden för att flytta igenom varje cell.

Värdet i varje cellposition motsvarar kostnad-per-enhetsavståndet för att flytta genom cellen. Det får inte vara negativt eller noll.

Anger om avstånd ska beräknas med en planmetod (platt jordyta) eller en geodetisk metod (ellipsoid).

Anger om rutterna ska fortsätta och anslutas inom indataregionerna.

Namnet på lagret för optimalt nätverk av vägar som ansluter varje indataregion som skapas och läggs till i kartan.

Namnet på linjelagret för angränsande anslutningar som skapas och läggs till i kartan. Lagret identifierar alla rutter från varje region till var och en av dess närmaste, eller billigaste, grannar.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
