Verktyget Identifiera ändring med djupinlärning använder en djupinlärningsmodell för att identifiera ändring mellan två rasterlager.

Det indatabildlager före ändring som ska användas som bas för jämförelse.

Det indatabildlager efter ändring som ska användas för att jämföra med originalet.

Det klassificerade utdatabildlager som visar ändringen.

Modellargumenten som finns i djupinlärningsmodellen tillsammans med sina standardvärden.

Det djupinlärningspaket som innehåller modellen som ska användas för att identifiera ändringen mellan bildlager.

Typ av utdata som skapas. Utdata kan vara ett bildlager i tileform eller ett dynamiskt bildlager.

Namnet på mappen i Mitt innehåll där resultatet ska sparas.
