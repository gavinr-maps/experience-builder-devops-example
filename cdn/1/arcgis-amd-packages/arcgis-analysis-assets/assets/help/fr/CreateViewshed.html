L’outil Create Viewshed (Créer un champ de vision) utilise le service d’analyse d’altitude Esri pour identifier les surfaces visibles à partir de localisations d’observation spécifiées.

Entités ponctuelles à utiliser comme localisations d'observation.

Spécifie la hauteur des observateurs au-dessus du sol.

Unités de mesure à utiliser pour le paramètre Observer height (Hauteur de l’observateur).

Spécifie la hauteur des cibles au sol.

Unités de mesure à utiliser pour le paramètre Target height (Hauteur de la cible).

Spécifie la distance d'affichage maximale pour le calcul du champ de vision.

Unités de mesure à utiliser pour le paramètre Maximum viewing distance (Distance de visualisation maximale).

Nom de la couche de polygones de champs de vision à créer et ajouter à la carte. Cette couche contient les zones visibles dans lesquelles les observateurs peuvent observer et le nombre d'observateurs est enregistré dans le champ de fréquence.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
