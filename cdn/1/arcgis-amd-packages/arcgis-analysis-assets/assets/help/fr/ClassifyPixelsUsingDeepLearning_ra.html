L’outil Classify Pixels Using Deep Learning (Classer des pixels à l’aide d’algorithmes de Deep Learning) utilise un modèle de Deep Learning entraîné sur une image en entrée pour générer un raster classé.

Couche d'imagerie à utiliser pour classer les pixels.

Spécifie le mode de traitement des éléments raster dans une couche d’imagerie une fois qu'ils ont été mosaïqués : ensemble ou individuellement.

Paquetage de Deep Learning qui contient le modèle à utiliser pour classer les pixels.

Arguments du modèle présents dans le modèle Deep Learning avec leur valeur par défaut.

Nom de la couche d’imagerie en sortie à classer.

Nom de la couche à créer et ajouter à la carte.

Type de sortie à créer. La sortie peut correspondre à une couche d’imagerie tuilée ou une couche d’imagerie dynamique.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
