L’outil Extraire les données sélectionne les entités dans une étendue et crée un élément dans votre contenu.

Couches à extraire. Les couches prises en charge comprennent les couches dont vous êtes propriétaire et les couches dans lesquelles l’exportation a été activée.

Étendue des entités à extraire.

Spécifie si les entités qui intersectent l’étendue seront découpées. Si cette option est désactivée, les entités entières qui intersectent l’étendue sont extraites. Si elle est activée, les entités qui intersectent l’étendue sont découpées, et seules les portions de ces entités situées dans l’étendue sont extraites.

Format de fichier des données à extraire.

Nom de l’élément en sortie où le fichier extrait sera stocké.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
