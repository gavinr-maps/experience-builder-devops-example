L’outil Find Outliers (Rechercher les points aberrants) détermine s’il existe des localisations possédant des valeurs considérablement différentes de celles de leurs voisines en fonction du modèle spatial de vos données.

Couche ponctuelle ou surfacique dans laquelle les points aberrants sont recherchés.

Indique si les points aberrants sont déterminés en fonction des localisations des entités (Point counts (Nombre de points)) ou des valeurs de champ des entités (Field (Champ)).

Champ à analyser pour déterminer les points aberrants.

Type des entités d’emprise à l’intérieur desquelles les entités en entrée doivent être agrégées. Il peut s’agir d’une couche surfacique (Polygon layer (Couche de polygones)) ou d’une grille de tessellation de carrés (Fishnet cells (Cellules de quadrillage)) ou d’hexagones (Hexagon cells (Cellules d’hexagone)).

Couche contenant les entités surfaciques dans lesquelles les entités ponctuelles en entrée doivent être agrégées.

Couche surfacique qui doit définir l’emprise de l’analyse. Les entités à l’intérieur du polygone sont incluses dans l’analyse des points aberrants.

Indique comment diviser les valeurs Analysis field (Champ d’analyse) ou les totaux de points agrégés. L’option Field (Champ) divise les valeurs Analysis field (Champ d’analyse) par les valeurs d’un champ spécifié dans le paramètre Input layer (Couche en entrée). L’option Enrichment data (Données d’enrichissement) enrichit les entités ou les formes d’agrégation avec des données de population et divise les valeurs Analysis field (Champ d’analyse) ou les totaux de points agrégés par la population. Cette option consomme des crédits supplémentaires.

Indique si l’outil doit être optimisé pour un traitement plus rapide ou une plus grande précision des résultats.

Longueur d’un côté des cellules carrées (quadrillage) ou hexagonales à utiliser pour agréger les entités en entrée.

Spécifie les unités à utiliser pour la taille des cellules.

Canal de distance à utiliser pour définir un voisinage. Les entités voisines à l’intérieur du canal de distance d’une entité cible sont incluses dans le voisinage de cette entité cible lors de l’analyse.
