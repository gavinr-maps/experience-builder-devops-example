L’outil Rechercher selon les attributs et la localisation crée une couche des entités ou des parties d’entités répondant aux critères spatiaux ou attributaires spécifiés.

Couche ponctuelle, linéaire ou de polygones pour laquelle les entités, en fonction d’une combinaison de requêtes attributaires et spatiales, sont recherchées.

Les critères spatiaux ou attributaires à utiliser pour rechercher les entités. Cliquez sur le bouton Build new query (Créer une requête) pour définir un ou plusieurs critères spatiaux ou attributaires. Les entités ou portions d’entités qui respectent tous les critères sont écrites dans la couche résultat.

Les critères spatiaux ou attributaires à utiliser pour rechercher les entités. Cliquez sur le bouton Build new query (Créer une requête) pour définir un ou plusieurs critères spatiaux ou attributaires. Les entités ou portions d’entités qui respectent tous les critères sont écrites dans la couche résultat.

Lorsque seule une portion d’une entité respecte les critères, indique si le résultat inclut l’entité entière ou uniquement la portion de l’entité répondant à tous les critères. Lorsque cette option est sélectionnée, seules les portions de l’entité qui répondent à tous les critères sont renvoyées. Lorsqu’elle n’est pas sélectionnée, l’entité entière est renvoyée à condition qu’au moins une portion de l’entité réponde à tous les critères.

Nom de la couche à créer et ajouter à la carte.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
