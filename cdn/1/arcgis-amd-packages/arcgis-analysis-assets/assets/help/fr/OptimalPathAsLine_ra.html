L'outil Optimal Path as Line (Chemin optimal comme ligne) calcule le chemin entre les destinations et la source optimale sous forme de ligne.

La couche identifie les destinations depuis lesquelles la distance est calculée vers la source de moindre coût.

Si l’entrée est un raster, il doit comporter des cellules qui possèdent des valeurs entières valides (zéro est une valeur valide) pour les destinations et les cellules restantes doivent posséder la valeur NoData.

Champ permettant d’obtenir des valeurs relatives aux emplacements de destination.

Le raster d’accumulation de distance est généralement créé avec l’outils Accumulation de distance ou Allocation de distance.

Le raster de direction arrière est généralement créé à l’aide de l'outil Distance Accumulation (Accumulation de distance) ou Distance Allocation (Allocation de distance).
