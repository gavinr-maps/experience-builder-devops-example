L'outil Find Argument Statistics (Rechercher des statistiques d’argument) extrait la valeur de dimension ou l’index de canal auquel une statistique donnée est atteinte pour chaque pixel dans une couche d'imagerie multidimensionnelle.

Couche d’imagerie multidimensionnelle à évaluer.

Variable de la couche d’imagerie multidimensionnelle à évaluer au cours de l'intervalle défini.

Type de statistique à extraire de la variable ou des variables le long d'une dimension donnée.

Valeur minimale à utiliser pour la durée de l'argument.

Valeur maximale à utiliser pour la durée de l'argument.

Valeur indiquant qu’une statistique d’argument donnée a été atteinte plusieurs fois. Par défaut, il s'agit de la première fois que la statistique d’argument est atteinte.

Indique si la valeur de la dimension est renvoyée la première ou la dernière fois que l’argument est atteint. La valeur par défaut est la première occurrence.

La valeur de dimension à laquelle la valeur de variable spécifiée est atteinte sera extraite.

Spécifie le type de comparaison à utiliser pour extraire la valeur de dimension.

Indique comment la statistique est extraite de la dimension.

Unité de temps pour laquelle chaque statistique doit être extraite.

Indique si les valeurs NoData sont ignorées dans l'extraction.
