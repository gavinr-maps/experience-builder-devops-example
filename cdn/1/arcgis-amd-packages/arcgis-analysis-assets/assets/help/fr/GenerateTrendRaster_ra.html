L'outil Generate Trend Raster (Générer un raster de tendance) estime la tendance de chaque pixel le long d'une dimension pour une ou plusieurs variables d’une couche d'imagerie multidimensionnelle.

Couche d’imagerie multidimensionnelle à utiliser pour générer la tendance.

Dimension le long de laquelle une tendance sera extraite pour la ou les variables sélectionnées dans l’analyse.

La variable ou les variables pour lesquelles les tendances sont calculées.

Unité de temps à utiliser pour la longueur d'une période saisonnière si le type de ligne Kendall saisonnier est utilisé.

Type d’analyse de tendance à effectuer sur les valeurs de pixel le long d’une dimension.

Fréquence ou numéro d’ordre polynomial à utiliser pour l’ajustement de tendance.

Indique si les valeurs NoData ou manquantes sont ignorées lors de l’analyse.

Durée de la variation périodique à modéliser pour le cycle harmonique.

Unité de temps à utiliser pour la durée du cycle harmonique.

Précise si l’erreur quadratique moyenne (EQM) de la ligne d'ajustement de la tendance est calculée.

Précise si la statistique de la qualité d’ajustement R-carré pour la ligne d’ajustement de la tendance est calculée.

Précise si la statistique de valeur P pour le coefficient de pente de la ligne d’ajustement de la tendance est calculée.

Nom de la couche à créer et ajouter à la carte.

Type de sortie à créer. La sortie peut correspondre à une couche d’imagerie tuilée ou une couche d’imagerie dynamique.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
