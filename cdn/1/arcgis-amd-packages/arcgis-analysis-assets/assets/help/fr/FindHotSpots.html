L’outil Find Hot Spots (Rechercher les points chauds) identifie les agrégations statistiquement significatives dans les entités en entrée.

Couche de points ou de polygones sur laquelle une analyse des points chauds doit être effectuée.

Indique si les points chauds ou froids doivent refléter les localisations des entités (Point counts (Nombre de points)) ou les valeurs de champ des entités (Field (Champ)).

Champ à analyser pour les agrégats de valeurs élevées et faibles.

Type des entités d’emprise à l’intérieur desquelles les entités ponctuelles en entrée doivent être agrégées. Il peut s’agir d’une couche surfacique (Polygon layer (Couche de polygones)) ou d’une grille de tessellation de carrés (Fishnet cells (Cellules de quadrillage)) ou d’hexagones (Hexagon cells (Cellules d’hexagone)).

Couche contenant les entités surfaciques dans lesquelles les entités ponctuelles en entrée doivent être agrégées.

Couche surfacique qui doit définir l’emprise de l’analyse. Entités ponctuelles à l’intérieur du polygone à inclure dans l’analyse des points chauds.

Indique comment diviser les valeurs Analysis field (Champ d’analyse) ou les totaux de points agrégés. L’option Field (Champ) divise les valeurs Analysis field (Champ d’analyse) par les valeurs d’un champ spécifié dans le paramètre Input layer (Couche en entrée). L’option Enrichment data (Données d’enrichissement) enrichit les entités ou les formes d’agrégation avec des données de population et divise les valeurs Analysis field (Champ d’analyse) ou les totaux de points agrégés par la population. Cette option consomme des crédits supplémentaires.

Longueur d’un côté des cellules carrées ou hexagonales à utiliser pour agréger les entités en entrée.

Spécifie les unités à utiliser pour la taille des cellules.

Canal de distance à utiliser pour définir un voisinage. Les entités voisines à l’intérieur du canal de distance d’une entité cible sont incluses dans le voisinage de cette entité cible lors de l’analyse des points chauds.

Spécifie les unités à utiliser pour la bande de distance.

Nom de la couche à créer et ajouter à la carte.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
