L’outil Détecter les changements à l’aide du Deep Learning utilise un modèle de Deep Learning pour déterminer les changements entre deux couches raster.

Couche d’imagerie en entrée avant le changement qui sera utilisée comme base de comparaison.

Couche d’imagerie en entrée après le changement qui sera utilisée pour comparaison d’après l’originale.

Couche d’imagerie classée en sortie qui affiche le changement.

Arguments du modèle présents dans le modèle Deep Learning avec leur valeur par défaut.

Paquetage de Deep Learning qui contient le modèle à utiliser pour détecter les changements entre les couches d’imagerie.

Type de sortie à créer. La sortie peut correspondre à une couche d’imagerie tuilée ou une couche d’imagerie dynamique.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
