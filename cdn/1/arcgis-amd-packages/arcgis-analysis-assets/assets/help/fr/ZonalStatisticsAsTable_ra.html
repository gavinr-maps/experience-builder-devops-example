L’outil Statistiques zonales (table) récapitule les valeurs d’un raster dans les zones d’un autre jeu de données et affiche les résultats dans une table.

Entrée qui définit les limites des zones, ou des surfaces, à synthétiser.

Champ contenant les valeurs qui définissent chaque zone.

Couche raster qui contient les valeurs de cellule à utiliser pour calculer les statistiques.

Spécifie le type de statistique à calculer.

Spécifie les valeurs de centile à calculer.

Spécifie la méthode à utiliser si le nombre de valeurs du raster en entrée à calculer est pair.

Spécifie le mode de traitement des valeurs de cellule Input value raster (Raster de valeurs en entrée) pour les données circulaires.

Si ce paramètre n’est pas sélectionné, les statistiques arithmétiques (linéaires) sont calculées. Si ce paramètre est sélectionné, les statistiques circulaires sont calculées.

Spécifie la valeur à utiliser pour arrondir une valeur linéaire à la plage d’une statistique circulaire donnée.

Indique si les cellules NoData figurant dans le paramètre Input value raster (Raster de valeurs en entrée) sont ignorées dans les résultats des zones auxquelles elles appartiennent.

Si ce paramètre est sélectionné, les cellules NoData dans le raster de valeurs sont ignorées dans le calcul statistique. S'il ne l'est pas, les cellules NoData ne sont pas ignorées et toute la zone reçoit la valeur NoData dans le raster en sortie.

Spécifie le mode de traitement des rasters en entrée s’il s’agit de rasters multidimensionnels.

S’il n’est pas sélectionné, les statistiques sont calculées pour toutes les dimensions du jeu de données multidimensionnelles en entrée. S’il est sélectionné, les statistiques sont calculées à partir de la tranche actuelle du jeu de données multidimensionnelles en entrée.

Nom de la couche tabulaire des statistiques zonales à créer et ajouter à la carte.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
