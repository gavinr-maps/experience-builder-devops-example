L'outil Classify Objects Using Deep Learning (Classer des objets à l’aide d’algorithmes de Deep Learning) utilise un modèle de Deep Learning pour attribuer une étiquette de classe ou une catégorie à un objet ou une entité.

Couche d'imagerie ou d'entités à utiliser pour classer les objets.

Couche d’entités qui identifie la localisation de chaque objet à classer ou étiqueter.

Spécifie le mode de traitement des éléments raster dans une couche d’imagerie une fois qu'ils ont été mosaïqués : ensemble ou individuellement.

Paquetage de Deep Learning qui contient le modèle à utiliser pour classer les objets.

Arguments du modèle présents dans le modèle Deep Learning avec leur valeur par défaut.

Le nom du champ qui contiendra l’étiquette de classification dans la couche d’entités en sortie.

Nom de la couche à créer et ajouter à la carte.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
