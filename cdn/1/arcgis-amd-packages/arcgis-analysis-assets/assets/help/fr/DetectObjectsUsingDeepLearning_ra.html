L'outil Detect Objects Using Deep Learning (Détecter des objets à l’aide du Deep Learning) utilise un modèle de Deep Learning sur une couche d'imagerie en entrée pour générer une couche d’entités contenant les objets détectés.

Couche d'imagerie ou d'entités à utiliser pour détecter les objets.

Spécifie le mode de traitement des éléments raster dans une couche d’imagerie une fois qu'ils ont été mosaïqués : ensemble ou individuellement.

Paquetage de Deep Learning qui contient le modèle à utiliser pour détecter les objets.

Arguments du modèle présents dans le modèle Deep Learning avec leur valeur par défaut.

Indique si la suppression non maximale est réalisée pour supprimer les objets en double.

Nom du champ à ajouter à la couche d’entités en sortie pour indiquer la valeur de confiance communiquée.

Nom du champ à ajouter à la couche d’entités en sortie pour indiquer l'étiquette classée de l'objet.

Ratio maximal autorisé pour chaque entité détectée qui chevauche une autre entité. La valeur par défaut est 0.

Nom de la couche à créer et ajouter à la carte.

Nom du dossier de My Content (Mon contenu) dans lequel sera enregistré le résultat.
