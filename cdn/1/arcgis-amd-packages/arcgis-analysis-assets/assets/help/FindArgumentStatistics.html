Find Argument Statistics

The Find Argument Statistics tool extracts the dimension value or band index at which a given statistic is attained for each pixel in a multidimensional imagery layer

The multidimensional imagery layer that will be evaluated.

The dimension of the multidimensional imagery layer from which the variable will be evaluated.

The variable of the multidimensional imagery layer that will be evaluated along the defined interval.

The type of statistic that will be extracted from the variable or variables along a given dimension.

The minimum value that will be used for the duration of the argument.

The maximum value that will be used for the duration of the argument.

The value indicating that a given argument statistic has been reached more than once. The default is the first time the argument statistic is reached.

Specifies how the statistic will be extracted from the dimension.

The unit of time for which each statistic will be extracted.

The name of the layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
