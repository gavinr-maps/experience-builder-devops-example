Το θεματικό επίπεδο σημείου εισόδου που θα ομαδοποιηθεί σε συστάδες με βάση τη χωρική κατανομή των στοιχείων σημείου.

Η μέθοδος που θα χρησιμοποιηθεί για τον εντοπισμό συστάδων.

Ο ελάχιστος αριθμός στοιχείων σημείου που θα μπορεί να θεωρηθεί μία συστάδα.

Προσδιορίζει τη μέγιστη απόσταση γύρω από κάθε στοιχείο σημείου για την αναζήτηση σημείων που μπορούν να συμπεριληφθούν στη συστάδα. Αν μείνει κενό, η τιμή της Απόστασης αναζήτησης θα είναι η μεγαλύτερη βασική απόσταση που βρέθηκε στο θεματικό επίπεδο, εξαιρουμένων των βασικών αποστάσεων στις κορυφαίες 1%.

Καθορίζει τις μονάδες που θα χρησιμοποιηθούν για την απόσταση αναζήτησης.

Ένας ακέραιος μεταξύ 0 και 100 που καθορίζει τη συμπυκνότητα των συστάδων. Αν μείνει κενό, η τιμή της Ευαισθησίας συστάδας θα υπολογιστεί με την Απόκλιση Kullback-Leibler.
