Αυτό το εργαλείο Στατιστική ανάλυση κατά ζώνη συνοψίζει τις τιμές ενός raster εντός των ζωνών ενός άλλου συνόλου δεδομένων.

Η είσοδος που ορίζει τα όρια των ζωνών ή των περιοχών που θα συνοψιστούν.

Το πεδίο που περιέχει τις τιμές που καθορίζουν κάθε ζώνη.

Το θεματικό επίπεδο raster που περιέχει τις τιμές κελιών που θα χρησιμοποιηθούν για τον υπολογισμό στατιστικών στοιχείων.

Προσδιορίζει τον τύπο των στατιστικών στοιχείων που θα υπολογιστούν.

Καθορίζει το εκατοστημόριο των τιμών που θα υπολογιστεί.

Καθορίζει τη μέθοδο που θα χρησιμοποιηθεί όταν ο αριθμός των τιμών από το raster εισόδου προς υπολογισμό είναι άρτιος.

Καθορίζει πώς θα γίνει η επεξεργασία των τιμών κελιών του raster τιμών εισόδου για κυκλικά δεδομένα.

Αν δεν επιλεγεί, θα υπολογιστούν κανονικά αριθμητικά (γραμμικά) στατιστικά στοιχεία. Αν επιλεγεί, θα υπολογιστούν κυκλικά στατιστικά στοιχεία.

Καθορίζει την τιμή που θα χρησιμοποιηθεί για τη στρογγυλοποίηση μιας γραμμικής τιμής στο εύρος ενός δεδομένου κυκλικού στατιστικού στοιχείου.

Καθορίζει αν κελιά NoData στο raster τιμών εισόδου θα παραβλέπονται στα αποτελέσματα των ζωνών στις οποίες εμπίπτουν.

Εάν επιλεγούν, τα κελιά με την τιμή NoData στο raster τιμής δεν θα ληφθούν υπόψη στον στατιστικό υπολογισμό. Αν δεν επιλεγεί, τα κελιά NoData δεν θα παραβλέπονται και ολόκληρη η ζώνη θα λαμβάνει την τιμή NoData στο raster εξόδου.

Καθορίζει τον τρόπο επεξεργασίας των rasters εισόδου εάν είναι πολυδιάστατα.

Εάν δεν επιλεγούν, τα στατιστικά στοιχεία θα υπολογιστούν για όλες της διαστάσεις του πολυδιάστατου συνόλου δεδομένων εισόδου. Εάν επιλεγούν, τα στατιστικά στοιχεία θα υπολογιστούν από το τρέχον τμήμα του πολυδιάστατου συνόλου δεδομένων εισόδου.

Το όνομα του θεματικού επιπέδου του θεματικού επιπέδου στατιστικής ανάλυσης κατά ζώνη που θα δημιουργηθεί και θα προστεθεί στο χάρτη.

Ο τύπος του στοιχείου εξόδου που θα δημιουργηθεί. Το στοιχείο εξόδου μπορεί να είναι tiled imagery layer ή δυναμικό imagery layer.
