Το εργαλείο Εύρεση Ακραίων Τιμών προσδιορίζει αν υπάρχουν τοποθεσίες με σημαντικά διαφορετικές τιμές από τα γειτονικά κελιά που βασίζονται στο χωρικό μοτίβο των δεδομένων.

Το θεματικό επίπεδο σημείου ή πολυγώνου για το οποίο θα βρεθούν οι ακραίες τιμές.

Καθορίζει αν θα καθοριστούν ακραίες τιμές με βάση τοποθεσίες στοιχείων (Αθροίσματα ημείων) ή τιμές πεδίων στοιχείων (Πεδίο).

Το πεδίο που θα αναλυθεί για τον καθορισμό των ακραίων πηγών.

Ο τύπος στοιχείων οριοθέτησης εντός των οποίων θα γίνει η συνάθροιση των στοιχείων εισόδου. Μπορεί να είναι ένα θεματικό επίπεδο πολυγώνου (Θεματικό επίπεδο πολυγώνου,) ή ένα πλέγμα κατάτμησης είτε τετραγώνων (Δικτυωτά κελιά) ή εξαγώνων (Εξαγωνικά κελιά).

Το θεματικό επίπεδο που περιέχει στοιχεία πολυγώνου στα οποία θα γίνει η συνάθροιση των σημειακών στοιχείων εισόδου.

Το θεματικό επίπεδο πολυγώνου που θα καθορίσει το όριο της ανάλυσης. Στοιχεία εντός του πολυγώνου που θα συμπεριληφθούν στην ανάλυση ακραίων τιμών.

Καθορίζει πώς χωρίζονται οι τιμές Πεδίου ανάλυσης ή οι συσσωρευμένοι αριθμοί σημείων. Η επιλογή Πεδίο θα διαιρέσει τις τιμές Πεδίου ανάλυσης με τις τιμές του καθορισμένου πεδίου στην παράμετρο Θεματικό επίπεδο εισόδου. Η επιλογή Εμπλουτισμός δεδομένων θα εμπλουτίσει τα στοιχεία ή τα σχήματα συσσώρευσης με δεδομένα πληθυσμού και θα διαιρέσει τις τιμές Πεδίου ανάλυσης ή τους συσσωρευμένους αριθμούς σημείων με τον πληθυσμό. Αυτή η επιλογή θα καταναλώσει επιπλέον μονάδες κατανάλωσης.

Καθορίζει εάν θα γίνει βελτιστοποίηση του εργαλείου για αύξηση του χρόνου επεξεργασίας ή αυξημένη ακρίβεια των αποτελεσμάτων.

Ακέραια τιμή που θα αρχικοθετήσει γεννήτρια τυχαίων αριθμών.

Το μήκος της πλευράς των τετράγωνων (δικτυωτά) ή εξαγωνικών κελιών που θα χρησιμοποιηθούν για τη συνάθροιση των στοιχείων εισόδου.

Καθορίζει τις μονάδες που θα χρησιμοποιηθούν για το μέγεθος κελιού.

Η φασματική ζώνη απόστασης που θα χρησιμοποιηθεί για τον καθορισμό γειτονιάς. Γειτονικά στοιχεία εντός της φασματικής ζώνης απόστασης ενός στοιχείου προορισμού θα συμπεριληφθούν στη γειτονιά του στοιχείου προορισμού κατά την ανάλυση.

Καθορίζει τις μονάδες που θα χρησιμοποιηθούν για τη φασματική ζώνη απόστασης.

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί και θα προστεθεί στον χάρτη.

Το όνομα του φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
