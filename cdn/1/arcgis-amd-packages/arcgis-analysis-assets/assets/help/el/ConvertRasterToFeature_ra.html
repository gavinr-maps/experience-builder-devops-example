Το εργαλείο Μετατροπή raster σε στοιχείο μετατρέπει raster δεδομένα σε διανυσματικά.

Το raster εισόδου που θα μετατραπεί σε στοιχεία.

Το πεδίο που περιέχει τις τιμές που θα μετατραπούν.

Ο τύπος των στοιχειών (σημείων, γραμμών ή πολυγώνων) τα οποία θα δημιουργηθούν στο θεματικό επίπεδο εισόδου.

Καθορίζει αν οι γραμμές ή τα πολύγωνα θα απλοποιηθούν (θα εξομαλυνθούν).

Καθορίζει αν τα πολύγωνα εξόδου θα αποτελούνται από μονοτμηματικά ή πολυτμηματικά στοιχεία.

Τα χαρακτηριστικά πολυτμηματικών στοιχείων συνήθως υπόκεινται σε επεξεργασία ταχύτερα. Ωστόσο, εάν η έξοδος περιέχει πολύ μεγάλο αριθμό στοιχείων, ενδέχεται να μην είναι ταχύτερη.

Το όριο κορυφών που θα χρησιμεύει για την υποδιαίρεση ενός πολυγώνου σε μικρότερα πολύγωνα.

Αν αφήσετε αυτό το πεδίο κενό, τα πολύγωνα εξόδου δε θα διαιρεθούν.

Το όνομα του feature layer που θα δημιουργηθεί και θα προστεθεί στο χάρτη.
