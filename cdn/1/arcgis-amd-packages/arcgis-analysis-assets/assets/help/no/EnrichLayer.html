Verktøyet Berik lag forbedrer inndatageoobjektene med demografiske data og landskapsdata.

Punkt-, linje- eller polygongeoobjektene som blir beriket med flere data.

Variablene som legges til som attributter for inndatageoobjektene.

Angir hvordan området rundt punkt- eller linjegeoobjektene (inndata) defineres. Hele området rundt hvert geoobjekt blir beriket.

En numerisk verdi som angir avstanden eller tiden rundt hvert inndatageoobjekt som området vil dekke.

Enhetene som brukes til å angi avstanden eller tiden som det berikede området vil dekke rundt hvert inndatageoobjekt.

Angir om utdatalaget vil inneholde polygongeoobjekter definert av innstillingene for Berikelsesområder (valgt), eller inndatageoobjektene (ikke valgt).
