The Locate Regions tool identifies the best regions in the input raster that meet specified size requirements and spatial constraints.

The input utility raster from which the regions will be derived.

The total amount of area for all regions.

Specifies the area units used for the Total area, Region minimum area, and Region maximum area parameters.

Specifies how many regions the Total area will be distributed across.

Specifies the tradeoff between a cell's contribution for maintaining the region shape relative to the utility contribution of the cell's attribute value.

The evaluation criteria to determine which of the candidate regions are most preferred.

Specifies the shape characteristics for the output regions.

Specifies the orientation of the defined shape.

Specifies the minimum area allowed for each region. The units specified by the Area units parameter will be used.

Specifies the maximum area allowed for each region. The units specified by the Area units parameter will be used.

Specifies the minimum distance allowed between regions.

Specifies the maximum distance allowed between regions.

Specifies the distance units that will be used for the Minimum distance between regions and Maximum distance between regions parameters.

The input layer that identifies where regions already exist.

Specifies which neighboring cells to use in the growth of the regions.

Specifies whether or not islands will be allowed within the potential regions.

Specifies the number of seeds from which to grow the potential regions.

Specifies the resolution at which region growth occurs.

Specifies how the regions will be selected.

The name of the Locate Regions raster layer that will be created and added to the map.

The type of output that will be created. The outputs can be Tiled imagery layers or Dynamic imagery layers.
