The Find Hot Spots tool identifies statistically significant clustering in input features.

The input point or polygon layer on which hot spot analysis will be performed.

Specifies whether hot and cold spots will reflect feature locations (Point counts) or feature field values (Field).

The field that will be analyzed for high and low value clusters.

The type of bounding features within which the input point features will be aggregated. This can be a polygon layer (Polygon layer) or a tessellation grid of either squares (Fishnet cells) or hexagons (Hexagon cells).

The layer containing polygon features in which the input point features will be aggregated.

The polygon layer that will define the boundary of the analysis. Point features within the polygon will be included in the hot spot analysis.

Specifies how to divide the Analysis field values or the aggregated point counts. The Field option will divide the Analysis field values by the values of a specified field in the Input layer parameter. The Enrichment data option will enrich the features or the aggregation shapes with population data and divide the Analysis field values or the aggregated point counts by the population. This option will consume additional credits.

The length of a side of the square or hexagon cells that will be used to aggregate the input features.

Specifies the units that will be used for the cell size.

The distance band that will be used to define a neighborhood. Neighboring features within the distance band of a target feature will be included in the target feature's neighborhood during hot spot analysis.

Specifies the units that will be used for the distance band.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
