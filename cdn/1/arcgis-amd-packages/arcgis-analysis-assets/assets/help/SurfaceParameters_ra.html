The Surface Parameters tool calculates parameters of a raster surface such as aspect, slope, and curvature using geodesic methods.

The elevation raster that will be used for calculations.

If a z unit is available in the vertical coordinate system of the input raster, it will be applied automatically.

The surface parameter type that will be measured.

The measurement units of the output slope raster.

Specifies whether geodesic azimuths will be projected.

If checked, geodesic azimuths will be projected to correct the angle distortion caused by the output spatial reference. If unchecked, geodesic azimuths will not be projected.

Specifies whether aspect will be measured from a point on the equator or from the north pole.

If checked, aspect will be measured from a point on the equator. If unchecked, aspect will be measured from the north pole.

The type of surface function that will be fitted around the target cell.

The distance from the target cell center from which the output will be calculated. It determines the neighborhood size used for calculation.

The default value is the cell size of the raster specified in the Input surface raster parameter. If a value is specified, it must be larger than the default.

Specifies whether neighborhood distance will vary with landscape changes.

If checked, neighborhood distance will vary with landscape changes. If unchecked, a fixed neighborhood distance will be used at all locations.

The linear unit of vertical z-values.

The name of the surface raster layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
