The Extract Data tool selects features within an extent and creates an item in your content.

The layers that will be extracted. Supported input layers include layers you own and layers with export enabled.

The extent of features that will be extracted.

Specifies whether features that intersect the extent will be clipped. When unchecked, entire features that intersect the extent will be extracted. When checked, features that intersect the extent will be clipped, and only portions of those features within the extent will be extracted.

The file format of the data that will be extracted.

The name of the output item where the extracted file will be stored.

The name of the folder in My Content where the result will be saved.
