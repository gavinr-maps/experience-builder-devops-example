Lo strumento Estrai dati seleziona le feature all'interno di un'estensione e crea un elemento nel proprio contenuto.

I layer che verranno estratti. I layer di input supportati includono i layer di proprietà dell'utente e i layer con l'esportazione abilitata.

L'estensione delle feature che verranno estratte.

Specifica se le feature che intersecano l'estensione saranno verranno ritagliate. Quando deselezionato, verranno estratte tutte le feature che intersecano l'estensione. Quando selezionato, le feature che intersecano l'estensione verranno ritagliate e verranno estratte solo le porzioni di quelle feature all'interno dell'estensione.

Il formato di file dei dati che verranno estratti.

Il nome dell'elemento di output in cui verrà memorizzato il file estratto.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
