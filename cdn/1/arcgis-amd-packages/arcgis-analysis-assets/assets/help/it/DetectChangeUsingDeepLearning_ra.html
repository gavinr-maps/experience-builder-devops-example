Lo strumento Rileva modifica mediante apprendimento profondo utilizza un modello di apprendimento profondo per rilevare il cambiamento tra due layer raster.

Il layer immagini di input prima della modifica che sarà utilizzato come base per il confronto.

Il layer immagini di input dopo la modifica che sarà utilizzato per il confronto con l'originale.

Il layer immagini classificato di output che mostra il cambiamento.

Gli argomenti del modello presenti nel modello di apprendimento profondo insieme ai rispettivi valori predefiniti.

Il pacchetto di apprendimento profondo contenente il modello che verrà utilizzato per rilevare il cambiamento tra i layer immagini.

Il tipo di output che sarà creato. L'output può essere un layer immagini di tile o un layer immagini dinamico.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
