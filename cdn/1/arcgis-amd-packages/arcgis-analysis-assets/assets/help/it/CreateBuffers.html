Lo strumento Crea buffer crea poligoni che coprono una determinata distanza dalle feature di input.

Le feature punto, linea o poligono che verranno incluse nel buffer.

Specifica se la distanza del buffer verrà definita da una costante (Valore) o da un campo numerico (Campo).

La distanza che verrà utilizzata per creare il buffer. Se vengono aggiunti più valori, ogni valore verrà utilizzato per creare un buffer in ogni feature di input.

Il campo che conterrà la distanza del buffer per ogni feature.

Le unità che verranno utilizzate per specificare la distanza del buffer.

Specifica come verranno gestiti i buffer di diverse feature di input quando si sovrappongono.

Specifica se i buffer a distanza multipla saranno buffer concentrici che si sovrappongono (Dischi sovrapposti) o anelli che non si sovrappongono (Anelli distinti).

Per le feature linea, specifica se il buffer verrà creato su entrambi i lati della linea (Intorno), solo sul lato sinistro (Sinistra), or o solo sul lato destro (Destra).

Per le feature poligono, specifica se le feature poligono di input saranno incluse nel buffer (Includi) o escluse dal buffer (Escludi).

Specifica se la forma del buffer alla fine di una feature linea sarà rotonda (Rotonda) o piatta (Piatta).

Il nome del layer che sarà creato e aggiunto alla mappa.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
