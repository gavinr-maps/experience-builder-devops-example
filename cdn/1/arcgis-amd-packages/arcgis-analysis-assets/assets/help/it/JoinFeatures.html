Lo strumento Unisci feature aggiunge attributi da un layer o da una tabella a un altro layer o un'altra tabella, in base a relazioni spaziali e di attributo specificate. Come opzione, le statistiche possono essere calcolate quando sono presenti più record corrispondenti.

Il layer o la tabella che avrà gli attributi del layer di unione aggiunti alla relativa tabella attributi.

Il layer o la tabella con gli attributi che saranno aggiunti al layer di destinazione.

Specificare se unire i layer in base a una relazione spaziale. Ciò richiede che entrambi i layer abbiano una geometria.

Specificare la relazione spaziale che determina quali caratteristiche sono unite. Le feature del layer di unione che presentano la relazione specificata con il layer di destinazione saranno unite. Le relazioni disponibili dipendono dal tipo di geometria degli input.

La distanza dalle feature nel layer di destinazione che verrà utilizzata per determinare se le feature devono essere unite.

Le unità che verranno utilizzate per specificare la distanza.

Specificare se unire i layer in base ai valori dei campi corrispondenti. Questa è l'unica opzione se uno o più input sono una tabella.

Specificare la relazione di attributi che determina quali feature sono unite. Le feature vengono abbinate se i valori dei campi nel layer di join sono uguali ai valori dei campi nel layer di destinazione.

Specificare come verranno gestite le unioni se più record del layer di unione soddisfano i criteri di unione. Scegliere tra le seguenti opzioni:

  • Unire uno a uno— Unire un singolo record nel layer di unione a ciascuna feature nel layer di destinazione.
  • Unire uno a molti—Unire tutte le feature corrispondenti nel layer di unione al layer di destinazione. Il layer risultati conterrà più record della feature di destinazione.

Quando più feature nel layer di join corrispondono a una feature di destinazione, specificare se unire un singolo record dal layer di unione o se calcolare le statistiche di riepilogo di tutti i record corrispondenti dal layer di unione.

Quando esistono più feature di unione che corrispondono a una feature di destinazione, verranno calcolate le statistiche di riepilogo per tutti i record corrispondenti. Specificare i campi numero o data del layer di unione e le statistiche che verranno calcolate per ogni campo.

Quando esistono più feature di unione che corrispondono a una feature di destinazione, il primo record della tabella attributi del layer di unione sarà incluso nell'output. Come opzione, si può scegliere un campo numero o data e l'ordine di sequenza che determinerà l'ordinamento della tabella attributi.

Determina se le feature di destinazione non corrispondenti saranno incluse nell'output. Un' Unione interna visualizza solo i record di destinazione abbinati, mentre un' Unione a sinistra visualizza tutti i record di destinazione, anche se la feature di destinazione non ha una corrispondenza nel layer di unione.

Il nome del layer o della tabella che verrà creato e aggiunto alla mappa.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
