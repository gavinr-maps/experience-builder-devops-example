Lo strumento Trova per attributi e posizione crea un nuovo layer delle feature o parti di feature che soddisfano i criteri spaziali o di attributo specificati.

Il layer punto, linea o poligono per il quale verranno trovate le feature, basate su una combinazione di attributi e interrogazioni spaziali.

I criteri spaziali o di attributo che verranno utilizzati per trovare le feature. Fare clic sul pulsante Crea nuova interrogazione per definire uno o più criteri spaziali o di attributo. Le feature, o parti di feature, che soddisfano tutti i criteri saranno scritte nel layer risultati.

I criteri spaziali o di attributo che verranno utilizzati per trovare le feature. Fare clic sul pulsante Crea nuova interrogazione per definire uno o più criteri spaziali o di attributo. Le feature, o parti di feature, che soddisfano tutti i criteri saranno scritte nel layer risultati.

Quando solo una parte di una feature soddisfa i criteri, specifica se il risultato include l'intera feature o solo la parte della feature che soddisfa tutti i criteri. In caso di selezione, verranno restituite solo le parti della feature che soddisfano tutti i criteri. In case di deselezione, l'intera feature verrà restituita a condizione che almeno una parte della feature soddisfi tutti i criteri.
