Lo strumento Converti feature in raster converte i dati feature in un dataset raster.

I dati feature che verranno convertiti. Il layer di input può contenere feature punto, linea o poligono.

Il campo che verrà utilizzato per assegnare i valori al raster di output.

La dimensione della cella e l'unità per i raster di output.

Il nome del layer raster che verrà creato e aggiunto alla mappa.

Il tipo di output che sarà creato. L'output può essere un layer immagini di tile o un layer immagini dinamico.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
