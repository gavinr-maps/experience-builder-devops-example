Lo strumento Nibble sostituisce le celle di un raster corrispondenti a una maschera con i valori dei vicini più prossimi.

Il raster di input con le posizioni mascherate che verranno sostituite (ridotte) dal valore del corrispondente vicino più prossimo.

Il raster maschera che identifica le posizioni nel raster di input che verranno sostituite. Le celle con un valore di NoData sono considerate all'interno dell'area mascherata.

Il raster di zona di input. Per ogni zona, le celle di input che si trovano all'interno della maschera saranno sostituite solo dai valori delle celle più vicine all'interno della stessa zona.

Specifica se le celle NoData nel raster di input possono sostituire (nibble) le celle nelle aree mascherate se sono le più vicine.

Se selezionato, le celle NoData nel raster di input possono sostituirsi alle altre nell'area mascherata. Se non selezionato, solo le celle valide nel raster di input possono sostituirsi alle altre nell'area mascherata.

Specifica se le celle NoData nell'area mascherata saranno conservate o sostituite.

Se non selezionato, tutte le celle NoData nell'area mascherata verranno conservate nell'output. Se selezionato, le celle NoData nell'area mascherata verranno sostituite con il valore del vicino più prossimo che si trova all'esterno dell'area mascherata.

Il nome del layer raster sottoposto alla sostituzione che verrà creato e aggiunto alla mappa.

Il tipo di output che sarà creato. Gli output possono essere layer immagini di tile o layer immagini dinamici.

Il nome della cartella in I miei contenuti in cui saranno salvati i risultati.
