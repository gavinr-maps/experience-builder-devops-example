L'opzione Genera anomalia multidimensionale calcola l'anomalia per ciascuna sezione di un layer di immagini multidimensionali esistente per generare un nuovo layer di immagini multidimensionali.

Il layer di immagini multidimensionali che verrà analizzato.

La variabile del layer di immagini multidimensionali che verrà utilizzata per localizzare le anomalie.

Il metodo che verrà utilizzato per calcolare le anomalie nel layer di immagini.

L'intervallo temporale che verrà utilizzato per calcolare la media.

Il layer di immagini di riferimento che contiene le medie calcolate per ogni pixel.

Specifica se i valori NoData o mancanti verranno ignorati nell'analisi.

Il nome del layer che sarà creato e aggiunto alla mappa.

Il tipo di output che sarà creato. L'output può essere un layer immagini di tile o un layer immagini dinamico.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
