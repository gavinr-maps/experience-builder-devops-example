Lo strumento Converti raster in feature vettorizza un raster.

Il raster di input che sarà convertito in feature.

Il campo contenente i valori che saranno convertiti.

Il tipo di feature (punto, linea o poligono) che verrà creato nel layer di output.

Specifica se le linee o i poligoni saranno semplificati (smussati).

Specifica se i poligoni di output saranno costituiti da feature monoparte o multiparte.

Le feature a più parti in genere vengono elaborate più velocemente. Tuttavia, se l'output contiene un numero molto elevato di feature, potrebbe non essere più veloce.

Il limite di vertice che verrà utilizzato per suddividere un poligono in poligoni più piccoli.

Se lasciato vuoto, i poligoni di output non verranno divisi.

Il nome del feature layer che verrà creato e aggiunto alla mappa.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
