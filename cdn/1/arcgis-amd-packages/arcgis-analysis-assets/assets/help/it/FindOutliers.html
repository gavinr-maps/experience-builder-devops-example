Lo strumento Trova outlier determinerà la presenza di posizioni con valori significativamente diversi da quelli dei rispettivi vicini, in base al modello spaziale dei dati.

Il layer punto o poligono dal quale verranno trovati gli outlier.

Specifica se gli outlier verranno determinati in base alle posizioni delle feature (Conteggio punti) o dai valori del campo feature (Campo).

Il campo che verrà analizzato per determinare gli outlier.

Il tipo di feature di delimitazione all'interno del quale verranno aggregate le feature di input. Questo può essere un layer poligono (Layer poligono) o una griglia di tassellazione di quadrati (Celle a rete da pesca) o esagoni (Celle esagonali).

Il layer contenente le feature poligono in cui verranno aggregate le feature punto di input.

Il layer poligono che definirà il confine dell'analisi. Le feature all'interno del poligono verranno incluse nell'analisi degli outlier.

Specifica come dividere i valori del campo Analisi o i conteggi dei punti aggregati. L'opzione Campo dividerà i valori del campo Analisi per i valori di un campo specificato nel parametro Layer di input. L'opzione Aggiunta dati arricchirà le feature o le forme di aggregazione con i dati della popolazione e dividerà i valori del campo Analisi o i conteggi dei punti aggregati per la popolazione. Questa opzione consumerà crediti aggiuntivi.
