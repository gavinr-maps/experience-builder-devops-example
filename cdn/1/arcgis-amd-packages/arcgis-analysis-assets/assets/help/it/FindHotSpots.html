Lo strumento Trova hot spot identifica i clustering statisticamente significativi nelle feature di input.

Il layer punto o poligono di input su cui verrà eseguita l'analisi degli hot spot.

Specifica se gli hot e i cold spot rifletteranno le posizioni delle feature (Conteggio punti) o dai valori del campo feature (Campo).

Il campo che verrà analizzato per i cluster di valore alto e basso.

Il tipo di feature di delimitazione all'interno del quale verranno aggregate le feature punto di input. Questo può essere un layer poligono (Layer poligono) o una griglia di tassellazione di quadrati (Celle a rete da pesca) o esagoni (Celle esagonali).

Il layer contenente le feature poligono in cui verranno aggregate le feature punto di input.

Il layer poligono che definirà il confine dell'analisi. Le feature punto all'interno del poligono saranno incluse nell'analisi degli hot spot.

Specifica come dividere i valori del campo Analisi o i conteggi dei punti aggregati. L'opzione Campo dividerà i valori del campo Analisi per i valori di un campo specificato nel parametro Layer di input. L'opzione Aggiunta dati arricchirà le feature o le forme di aggregazione con i dati della popolazione e dividerà i valori del campo Analisi o i conteggi dei punti aggregati per la popolazione. Questa opzione consumerà crediti aggiuntivi.
