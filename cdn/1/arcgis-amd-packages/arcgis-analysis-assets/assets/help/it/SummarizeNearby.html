Lo strumento Riepiloga nelle vicinanze calcola le statistiche per le feature che si trovano a una distanza specificata da altre feature. La distanza può essere calcolata utilizzando una distanza in linea retta o una modalità di viaggio.

Il layer da riepilogare entro una distanza dal layer vicino.

Il layer punto, linea o poligono utilizzato come posizione di partenza attorno alla quale verranno create le aree di delimitazione per riassumere le feature vicine.

Il metodo con cui verranno create le aree di delimitazione. Il metodo può essere una distanza in linea retta o una modalità di viaggio.

La distanza o il tempo che verrà utilizzato per creare le aree di delimitazione. Se vengono aggiunti più valori, ogni valore verrà utilizzato per calcolare le aree di delimitazione per tutte le feature di input.

Le unità che verranno utilizzate per specificare la distanza o il tempo.

Specifica se il layer dei risultati deve essere costituito dalle aree di delimitazione (spuntato) o dalla stessa geometria del layer vicino di input (non spuntato).

Calcola la dimensione totale delle feature riepilogate in ogni area di delimitazione. Per le linee, viene calcolata la lunghezza totale. Per i poligoni, viene calcolata l'area totale.

Le unità che verranno utilizzate per calcolare la dimensione totale delle feature riepilogate.

I campi numero o data del layer di input da riepilogare e le statistiche che verranno calcolate per ogni campo.

Il campo del layer di input che verrà utilizzato per separare i dati in gruppi distinti per i calcoli statistici.

Calcolare i gruppi minoritari (meno dominanti) e maggioritari (più dominanti) all'interno di ogni area di riepilogo.

Calcolare la percentuale delle feature riepilogate che appartengono a ciascun gruppo all'interno di ciascun confine.

Il nome del layer che sarà creato e aggiunto alla mappa.

Il nome della cartella in I miei contenuti in cui saranno salvati i risultati.
