Lo strumento Trova cluster punti identifica i cluster di feature punto entro il disturbo circostante, in base alla relativa distribuzione spaziale.

Il layer del punto di input che verrà raggruppato in cluster in base alla distribuzione spaziale delle feature punto.

Il metodo che verrà utilizzato per identificare i cluster.

Il numero minimo di feature punto che verranno considerate un cluster.

Specifica la distanza massima intorno a ogni feature punto per cercare i punti che possono essere inclusi nel cluster. Se lasciato vuoto, il valore Cerca distanza sarà la distanza del nucleo massima riscontrata nel layer, escludendo l'1 percento superiore delle distanze del nucleo.

Specifica le unità che verranno utilizzate per la distanza di ricerca.

Un numero intero compreso tra 0 e 100 che determina la compattezza dei cluster. Se lasciato vuoto, il valore Sensibilità del cluster sarà calcolato utilizzando la Divergenza di Kullback-Leibler.

Il nome del layer che sarà creato e aggiunto alla mappa.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
