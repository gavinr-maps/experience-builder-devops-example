Lo strumento Geocodifica posizioni da tabella converte indirizzi o luoghi da una tabella di input in coordinate.

La tabella di input con gli indirizzi o luoghi da geocodificare. L'input deve essere una tabella, un file .csvo un file Excel.

Servizio di localizzazione da utilizzare per la geocodifica. Se il servizio localizzatore da utilizzare non è presente nel menu a discesa, contattare l'amministratore affinché il servizio localizzatore venga configurato per la geocodifica batch.

Specifica se i dati di input sono contenuti in uno o più campi.

I campi saranno utilizzati dal localizzatore per determinare le posizioni dei dati. Alcuni campi devono essere nei dati, ad esempio indirizzo, città, stato e codice postale.

Il paese in cui si trovano i dati. Se tutti gli indirizzi sono relativi a un unico paese, seleziona il paese per garantire la massima precisione dei risultati. In caso contrario, lasciare questo parametro vuoto.

Specifica i campi che saranno inclusi nel dataset di output. Le opzioni sono le seguenti:

  • Tutto: tutti i campi di output del localizzatore disponibili saranno inclusi nei risultati del geocodice.
  • Minimo: un sottoinsieme di campi del localizzatore e dei campi della tabella di input saranno inclusi nei risultati.
  • Solo posizione: il campo Shape del localizzatore e i campi della tabella di input saranno inclusi nei risultati.

Il tipo di output che sarà creato. L'output può essere un feature service, un file .csv o un file .xls.

Il nome dell'elemento di output in cui saranno memorizzati i dati individuati.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
