Lo strumento Statistiche zonali riassume i valori di un raster all'interno delle zone di un altro set di dati.

L'input che definisce i confini delle zone o aree che verranno riepilogate.

Il campo che contiene i valori che definiscono ogni zona.

Il layer raster che contiene i valori delle celle che verranno utilizzati per calcolare le statistiche.

Specifica il tipo di statistica che verrà calcolata.

Specifica il valore percentile che verrà calcolato.

Specifica il metodo che verrà utilizzato quando il numero di valori del raster di input da calcolare è un numero pari.

Specifica come verranno elaborati i Valori delle celle raster di input per i dati circolari.

Se non selezionato, saranno calcolate le statistiche aritmetiche ordinarie (lineari). Se selezionato, saranno calcolate le statistiche circolari.

Specifica il valore che verrà utilizzato per arrotondare un valore lineare all'intervallo di una determinata statistica circolare.

Specifica se le celle NoData nel Raster dei valori di input saranno ignorate nei risultati delle zone in cui rientrano.

Se selezionato, le celle NoData nel raster dei valori verranno ignorate nel calcolo statistico. Se non selezionato, le celle NoData non saranno ignorate e l'intera zona riceverà il valore NoData sul raster di output.

Specifica il modo in cui vengono eleborati i raster di input se sono multidimensionali.

Se non selezionato, le statistiche saranno calcolate per tutte le dimensioni del dataset multidimensionale dell'input. Se selezionato, le statistiche verranno calcolate dalla porzione corrente del dataset multidimensionale dell'input.

Il nome del layer raster delle statistiche zonali che verrà creato e aggiunto alla mappa.

Il tipo di output che sarà creato. L'output può essere un layer immagini di tile o un layer immagini dinamico.

Il nome della cartella I miei contenuti in cui sarà salvato il risultato.
