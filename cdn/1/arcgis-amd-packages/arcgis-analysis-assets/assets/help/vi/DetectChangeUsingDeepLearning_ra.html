Công cụ Phát hiện Thay đổi bằng Deep Learning sử dụng mô hình deep learning để phát hiện thay đổi giữa hai lớp raster.

Lớp hình ảnh đầu vào trước khi thay đổi sẽ được dùng làm cơ sở để so sánh.

Lớp hình ảnh đầu vào sau khi thay đổi sẽ được dùng để so sánh với lớp gốc.

Lớp hình ảnh đã phân loại đầu ra hiển thị thay đổi.

Các đối số mô hình có trong mô hình deep learning cùng với giá trị mặc định của các đối số đó.

Gói deep learning chứa mô hình sẽ được sử dụng để phát hiện thay đổi giữa các lớp hình ảnh.
