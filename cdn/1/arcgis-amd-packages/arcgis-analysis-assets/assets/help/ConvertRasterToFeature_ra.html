The Convert Raster to Feature tool vectorizes a raster.

The input raster that will be converted to features.

The field containing the values that will be converted.

The type of features (point, line, or polygon) that will be created in the output layer.

Specifies whether lines or polygons will be simplified (smoothed).

Specifies whether the output polygons will consist of single-part or multipart features.

Multipart features typically process faster. However, if the output contains a very large number of features, it may not be faster.

The vertex limit that will be used to subdivide a polygon into smaller polygons.

If left empty, the output polygons will not be divided.

The name of the feature layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
