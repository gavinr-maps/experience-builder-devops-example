The Calculate Travel Cost tool measures the travel time or distance between pairs of points.

The point layer representing the origin locations from which travel cost is calculated. Layers with up to 5,000 features are supported.

The point layer representing the origin locations from which travel cost is calculated.

The point layer representing the destination locations toward which travel cost is calculated. Layers with up to 5,000 features are supported.

The point layer representing the destination locations toward which travel cost is calculated.

The field in the From layer that defines the preassigned origin-destination pair. If there are multiple origin and destination locations, this ID must match a value in the To layer route ID field.

The field in the To layer that defines the preassigned origin-destination pair. If there are multiple origin and destination locations, this ID must match a value in the From layer route ID field.

One or more point features that act as temporary restrictions (barriers) when traveling on the underlying streets. Travel is permitted on the street but not through the barrier.

One or more line features that act as temporary restrictions (barriers) when traveling on the underlying streets. The line barriers prohibit travel anywhere the barriers intersect the streets.

One or more polygon features that act as temporary restrictions (barriers) when traveling on the underlying streets. The polygon barriers prohibit travel anywhere the polygons intersect the streets.

The method that will be used to measure distance or time. The method can be line distance or a travel mode.

The date and time of departure from the origin locations in the From layer.

Determines whether the origins and destinations are connected using spider lines or desire lines (Straight line) or the streets on the underlying street network (Follow streets).

The name of the layer that will be created and added to the map.

Specifies whether each result route will be saved as a route layer.

The name of the folder in My Content where the results will be saved.
