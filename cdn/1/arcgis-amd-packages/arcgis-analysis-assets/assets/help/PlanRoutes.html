The Plan Routes tool creates travel routes with multiple stops for a fleet of vehicles.

The point features that specify where the vehicles start their routes.

The point features the vehicles must visit on the course of their routes. These points become assigned or unassigned stops in the output.

Determines whether the vehicles must return to their start location (enabled) or end at specified locations (disabled).

The point features that specify where the vehicles end their routes. If the layer includes only one point, all vehicles are routed to a common end point. If the layer includes multiple points, each vehicle will be routed to a unique end location.

The field in the start layer that uniquely identifies the individual routes, vehicles, or drivers. If the vehicles are starting at multiple locations and ending at multiple locations, the route IDs chosen for the starting and ending layers must match.

The field in the end layer that uniquely identifies the individual routes, vehicles, or drivers. If the vehicles are starting at multiple locations and ending at multiple locations, the route IDs chosen for the starting and ending layers must match.

One or more point features that act as temporary restrictions (barriers) when traveling on the underlying streets. Travel is permitted on the street but not through the barrier.

One or more line features that act as temporary restrictions (barriers) when traveling on the underlying streets. The line barriers prohibit travel anywhere the barriers intersect the streets.

One or more polygon features that act as temporary restrictions (barriers) when traveling on the underlying streets. The polygon barriers prohibit travel anywhere the polygons intersect the streets.

The travel method with which the routes will be created. The travel mode can be based on distance or time.

The date and time of departure for vehicles.

The maximum number of vehicles that are available to visit the stops. If the Start layer and End layer each contain one feature, you can specify up to 100 vehicles. If either layer contains more than one feature, the default value cannot be edited.

The maximum number of stops that each vehicle can make on a route. You can specify a value between 1 and 200.

Specify the expected average service time spent at each stop.

Determines whether the maximum routing time is applied to vehicles. The routing time includes expected travel time and service times spent at stops along the route.

Specify the maximum amount of time a route should last from start to finish.

The name of the layer that will be created and added to the map.

Specifies whether each result route will be saved as a route layer.

The name of the folder in My Content where the results will be saved.
