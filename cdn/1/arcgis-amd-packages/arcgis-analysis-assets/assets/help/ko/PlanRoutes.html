차량의 경로가 시작되는 위치를 지정하는 포인트 피처입니다.

차량이 해당 경로를 따라 방문해야 하는 포인트 피처입니다. 이러한 포인트가 결과에서 할당되거나 할당되지 않은 경유지가 됩니다,

차량이 시작 위치로 돌아가야 하는지(활성화) 또는 지정된 위치에서 종료되어야 하는지(비활성화) 여부를 결정합니다.

차량의 경로가 종료되는 위치를 지정하는 포인트 피처입니다. 레이어에 포인트가 하나만 포함되어 있는 경우 모든 차량이 공통 끝점으로 경로가 지정됩니다. 레이어에 포인트가 여러 개 포함되어 있는 경우 각 차량이 고유한 종료 위치로 경로가 지정됩니다.

개별 경로, 차량 또는 운전자를 고유하게 식별하는 시작 레이어의 필드입니다. 차량의 경로가 여러 위치에서 시작되고 종료되는 경우 시작 레이어와 종료 레이어에 대해 선택한 경로 ID가 일치해야 합니다.

개별 경로, 차량 또는 운전자를 고유하게 식별하는 종료 레이어의 필드입니다. 차량의 경로가 여러 위치에서 시작되고 종료되는 경우 시작 레이어와 종료 레이어에 대해 선택한 경로 ID가 일치해야 합니다.

기본 도로에서 이동할 때 임시 제한(경계)으로 사용할 하나 이상의 포인트 피처입니다. 경계를 통과하지 않는 경우 도로 이동이 허용됩니다.

기본 도로에서 이동할 때 임시 제한(경계)으로 사용할 하나 이상의 라인 피처입니다. 라인 경계는 경계가 도로와 교차하는 모든 위치에서 이동을 금지합니다.

기본 도로에서 이동할 때 임시 제한(경계)으로 사용할 하나 이상의 폴리곤 피처입니다. 폴리곤 경계는 폴리곤이 도로와 교차하는 모든 위치에서 이동을 금지합니다.
