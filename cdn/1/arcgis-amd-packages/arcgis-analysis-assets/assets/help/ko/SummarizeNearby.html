주변 요약 도구는 다른 피처의 지정된 거리 내에 있는 피처에 대한 통계를 계산합니다. 거리는 직선 거리 또는 이동 모드를 사용하여 계산할 수 있습니다.

주변 레이어의 거리 내에서 요약할 레이어입니다.

주변 피처를 요약하기 위해 경계 영역이 생성될 시작 위치로 사용되는 포인트, 라인, 폴리곤 레이어입니다.

경계 영역을 생성할 방법입니다. 방법은 직선 거리 또는 이동 모드일 수 있습니다.

경계 영역을 생성하는 데 사용될 거리 또는 시간입니다. 여러 값이 추가되는 경우 각 값은 모든 입력 피처에 대한 경계 영역을 계산하는 데 사용됩니다.

거리 또는 시간을 지정하는 데 사용될 단위입니다.

결과 레이어가 경계 영역이어야 하는지(선택됨) 아니면 입력 주변 레이어와 동일한 지오메트리여야 하는지(선택 안 됨) 여부를 지정합니다.

각 경계 영역에 요약된 피처의 총 크기를 계산합니다. 라인의 경우 총 길이가 계산됩니다. 폴리곤의 경우 총 면적이 계산됩니다.

요약된 피처의 전체 크기를 계산하는 데 사용될 단위입니다.

요약할 입력 레이어의 숫자 또는 날짜 필드 및 각 필드에 대해 계산될 통계입니다.

통계 계산을 위해 데이터를 개별 그룹으로 구분하는 데 사용되는 입력 레이어의 필드입니다.

각 요약 영역 내에서 소수(열세) 및 다수(우세) 그룹을 계산합니다.

각 경계 내의 각 그룹에 속하는 요약된 피처의 백분율을 계산합니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
