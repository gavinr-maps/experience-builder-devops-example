포인트 보간 도구는 포인트 측정을 기반으로 새로운 위치에서 값을 예측합니다. 이 도구는 피처 결과를 생성합니다.

보간될 위치의 포인트 레이어를 선택합니다.

보간될 측정값을 포함하는 필드를 선택합니다.

계산 속도 또는 정확도 및 정밀도를 우선순위로 지정할지 여부를 선택합니다. 더 정확하고 정밀하게 예측할수록 계산하는 데 오랜 시간이 걸립니다.

보간 예측에 대한 표준 오차 레이어를 결과에 포함할지 여부를 선택합니다.

클래스 구분 값을 계산할 방법을 선택합니다.

결과 레이어에 포함될 클래스(예측 값의 범위) 수를 지정합니다. 각 클래스는 폴리곤 경계를 정의하고 예측 값의 등고선을 나타냅니다. 클래스 구분 값은 분류 유형에 따라 결정됩니다.

사용될 클래스 구분 값을 제공합니다.

연구 영역을 정의하는 데 사용될 폴리곤 레이어를 선택합니다. 연구 영역 내에서만 표시되도록 결과가 잘립니다.

값을 예측하는 데 사용될 위치를 나타내는 포인트 레이어를 선택합니다. 제공된 경우 지정된 위치에서 예측된 값을 포함하는 포인트 레이어가 생성됩니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
