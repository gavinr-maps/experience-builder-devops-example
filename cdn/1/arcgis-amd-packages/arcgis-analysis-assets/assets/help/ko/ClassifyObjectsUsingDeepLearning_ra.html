딥러닝을 사용한 객체 분류 도구는 딥러닝 모델을 사용하여 객체나 피처에 클래스 레이블 또는 범주를 할당합니다.

객체를 분류하는 데 사용될 영상 또는 피처 레이어입니다.

분류되거나 레이블이 지정될 각 객체의 위치를 식별하는 피처 레이어입니다.

영상 레이어의 래스터 항목이 모자이크 처리된 후 함께 또는 개별적으로 처리되는 방식을 지정합니다.

객체를 분류하는 데 사용될 모델이 포함된 딥러닝 패키지입니다.

해당 기본값과 함께 딥러닝 모델에 있는 모델 인수입니다.

결과 피처 레이어에서 분류 레이블이 포함될 필드의 이름입니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
