레이어 보강 도구는 인구 통계 및 경관 데이터로 입력 피처를 개선합니다.

추가 데이터로 보강될 포인트, 라인, 폴리곤 피처입니다.

입력 피처에 속성으로 추가될 변수입니다.

입력 포인트 또는 라인 피처 주변 영역의 정의 방식을 지정합니다. 각 피처 주변의 전체 영역이 보강됩니다.

영역이 확장될 각 입력 피처 주변의 거리 또는 시간을 지정하는 숫자 값입니다.

보강된 영역이 각 입력 피처 주변으로 확장되는 거리 또는 시간을 지정하는 데 사용될 단위입니다.

결과 레이어에 보강 영역 설정(선택됨) 또는 입력 피처(선택 안 됨)로 정의된 폴리곤 피처를 포함할지 여부를 지정합니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
