레이어 병합 도구는 두 레이어의 피처를 새 레이어에 복사합니다. 입력 레이어의 필드를 결합하고 복사할 방법을 제어할 수 있습니다.

병합 레이어와 병합될 포인트, 라인 또는 폴리곤 피처입니다.

입력 레이어와 병합될 레이어입니다.

필드가 병합되는 방법을 지정합니다.

두 레이어에 이름 및 데이터 유형이 동일한 필드가 포함되어 있는 경우 두 필드의 콘텐츠가 해당 이름의 한 필드로 병합됩니다. 이는 일치가 병합 작업 매개변수 값으로 사용된 필드에는 적용되지 않습니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
