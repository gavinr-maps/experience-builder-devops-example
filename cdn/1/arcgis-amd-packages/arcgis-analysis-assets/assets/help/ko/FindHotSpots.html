핫스팟 찾기 도구는 입력 피처에서 통계적으로 유의미한 클러스터링을 식별합니다.

핫스팟 분석이 수행될 입력 포인트 또는 폴리곤 레이어입니다.

핫스팟 및 콜드스팟이 피처 위치(포인트 수) 또는 피처 필드 값(필드)을 반영할지 여부를 지정합니다.

높은 값과 낮은 값 클러스터에 대해 분석될 필드입니다.

입력 포인트 피처가 집계될 경계 피처의 유형입니다. 이는 폴리곤 레이어(폴리곤 레이어) 또는 사각형(그물망 셀)이나 육각형(육각형 셀)의 공간 분할 그리드가 될 수 있습니다.

입력 포인트 피처가 집계될 폴리곤 피처를 포함하는 레이어입니다.

분석의 경계를 정의할 폴리곤 레이어입니다. 폴리곤 내의 포인트 피처는 핫스팟 분석에 포함됩니다.

분석 필드 값 또는 집계된 포인트 개수를 나누는 방법을 지정합니다. 필드 옵션은 분석 필드 값을 입력 레이어 매개변수에 지정된 필드 값으로 나눕니다. 데이터 보강 옵션은 인구 데이터로 피처 또는 집계 쉐이프를 보강하고 분석 필드 값 또는 집계된 포인트 개수를 인구로 나눕니다. 이 옵션을 사용할 경우 크레딧이 추가로 소비됩니다.

입력 피처를 집계하는 데 사용될 사각형 또는 육각형 셀 한 변의 길이입니다.

셀 크기에 사용될 단위를 지정합니다.

인접 영역을 정의하는 데 사용될 거리 밴드입니다. 대상 피처의 거리 밴드 내 인접 피처는 핫스팟 분석 중에 대상 피처의 인접 영역에 포함됩니다.

거리 밴드에 사용될 단위를 지정합니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
