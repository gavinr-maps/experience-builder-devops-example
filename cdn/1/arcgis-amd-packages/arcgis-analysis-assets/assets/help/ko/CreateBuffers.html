버퍼 생성 도구는 입력 피처에서 지정된 거리를 포함하는 폴리곤을 생성합니다.

버퍼될 포인트, 라인 또는 폴리곤 피처입니다.

버퍼 거리가 상수(값) 또는 숫자 필드(필드)로 정의되는지 여부를 지정합니다.

버퍼를 생성하는 데 사용될 거리입니다. 여러 값이 추가되는 경우 각 값은 모든 입력 피처에서 버퍼를 생성하는 데 사용됩니다.

각 피처에 대한 버퍼 거리를 포함할 필드입니다.

버퍼 거리를 지정하는 데 사용될 단위입니다.

서로 다른 입력 피처의 버퍼가 중첩될 때 처리될 방식을 지정합니다.

다중 거리 버퍼가 중첩되는 중심 버퍼(중첩 디스크)인지 아니면 중첩되지 않는 링(고유 링)인지를 지정합니다.

라인 피처의 경우 버퍼를 라인의 양쪽(주변), 왼쪽(왼쪽), 오른쪽(오른쪽) 중 어디에 생성할지를 지정합니다.

폴리곤 피처의 경우 입력 폴리곤 피처를 버퍼에 포함(포함)할지 아니면 버퍼에서 제외(제외)할지를 지정합니다.

라인 피처의 끝에서 버퍼의 모양이 원형(원형)인지 아니면 평면(평면)인지 여부를 지정합니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
