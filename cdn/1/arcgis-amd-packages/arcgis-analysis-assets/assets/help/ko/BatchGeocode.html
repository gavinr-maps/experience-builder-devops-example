테이블 위치 지오코드 도구는 입력 테이블의 주소 또는 장소를 좌표로 변환합니다.

지오코딩할 주소 또는 장소가 있는 입력 테이블입니다. 테이블, .csv 파일 또는 Excel 파일에 입력해야 합니다.

지오코딩에 사용할 로케이터 서비스입니다. 사용할 로케이터 서비스가 드롭다운 메뉴에 없으며, 관리자에게 연락하여 로케이터 서비스를 일괄 지오코딩에 대해 구성하도록 합니다.

입력 데이터를 단일 필드와 다중 필드 중 어디에 포함할지 지정합니다.

로케이터에서 데이터의 위치를 지정하기 위해 사용하는 필드입니다. 주소, 시군구, 광역시도, 우편번호와 같은 특정 필드는 데이터에 있는 것으로 간주됩니다.

데이터가 위치한 국가입니다. 모든 주소가 한 국가 내에 있으면 가장 정확한 결과를 위해 해당 국가를 입력합니다. 그렇지 않으면 이 매개변수를 비워 둡니다.

결과 데이터셋에 포함할 필드를 지정합니다. 옵션은 다음과 같습니다.

  • 모두 — 사용 가능한 모든 로케이터 결과 필드가 지오코드 결과에 포함됩니다.
  • 최소 — 로케이터의 필드 서브셋 및 입력 테이블의 필드가 결과에 포함됩니다.
  • 위치만 — 로케이터의 Shape 필드 및 입력 테이블의 필드가 결과에 포함됩니다.

생성될 결과 유형입니다. 결과는 피처 서비스, .csv 파일 또는 .xls 파일일 수 있습니다.

찾은 데이터를 저장할 결과 항목의 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
