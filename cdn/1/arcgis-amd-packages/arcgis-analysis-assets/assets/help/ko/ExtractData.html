데이터 추출 도구는 범위 내에서 피처를 선택하고 콘텐츠에 항목을 생성합니다.

추출할 레이어입니다. 지원되는 입력 레이어에는 내가 소유한 레이어 및 내보내기가 활성화된 레이어가 포함됩니다.

추출할 피처의 범위입니다.

범위와 교차하는 피처를 클리핑할지 지정합니다. 선택 해제된 경우 범위와 교차하는 피처 전체가 추출됩니다. 선택된 경우 범위와 교차하는 피처가 클리핑되며 범위 안에 있는 해당 피처의 일부만 추출됩니다.

추출할 데이터의 파일 형식입니다.

추출된 파일이 저장될 결과 항목의 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
