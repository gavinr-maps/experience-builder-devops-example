딥러닝을 사용한 변경 감지 도구는 딥러닝 모델을 사용하여 두 래스터 레이어 간의 변경을 감지합니다.

비교 기준으로 사용될 변경 전의 입력 영상 레이어입니다.

원본과 비교하는 데 사용될 변경 후의 입력 영상 레이어입니다.

변경을 보여주는 결과 분류 영상 레이어입니다.

해당 기본값과 함께 딥러닝 모델에 있는 모델 인수입니다.

영상 레이어 간 변화를 감지하는 데 사용될 모델이 포함된 딥러닝 패키지입니다.

생성될 결과 유형입니다. 결과는 타일 영상 레이어 또는 동적 영상 레이어일 수 있습니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
