니블 도구는 마스크에 대응하는 래스터의 셀을 최근린 값으로 바꿉니다.

최근린 값으로 바꿀(니블할) 마스킹된 위치가 있는 입력 래스터입니다.

입력 래스터에서 바꿀 위치를 식별하는 마스크 래스터입니다. 값이 NoData인 셀은 마스킹된 영역 안에 있는 것으로 간주됩니다.

입력 영역 래스터입니다. 각 영역에서 마스크 내에 있는 입력 셀은 동일한 해당 구역 안에 가장 가까이 있는 셀 값으로만 바뀝니다.

입력 래스터의 NoData 셀이 최근린인 경우 마스킹된 영역의 셀을 바꿀(니블할) 수 있는지 여부를 지정합니다.

선택하는 경우 입력 래스터의 NoData 셀을 마스킹된 영역으로 니블할 수 있습니다. 선택하지 않는 경우 입력 래스터의 유효한 셀만 마스킹된 영역에 니블할 수 있습니다.

마스킹된 영역의 NoData 셀을 유지할지 아니면 바꿀지 여부를 지정합니다.

선택하지 않는 경우 마스킹된 영역 내에 있는 모든 NoData 셀을 결과에서 유지합니다. 선택하는 경우 마스킹된 영역 내의 NoData 셀을 마스킹된 영역 밖에 있는 최근린 값으로 바꿉니다(니블합니다).

생성되어 맵에 추가되는 니블된 래스터 레이어의 이름입니다.

생성될 결과 유형입니다. 결과는 타일 영상 레이어 또는 동적 영상 레이어일 수 있습니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
