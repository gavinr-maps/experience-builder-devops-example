구역 통계 도구는 다른 데이터셋의 구역 내에서 래스터 값을 요약합니다.

구역이나 영역의 경계를 정의하는 레이어로, 요약될 입력입니다.

각 구역을 정의하는 값이 포함된 필드입니다.

통계를 계산하는 데 사용될 셀 값이 포함된 래스터 레이어입니다.

계산될 통계 유형을 지정합니다.

계산할 백분위수 값을 지정합니다.

계산될 입력 래스터의 값 개수가 짝수일 경우 사용될 방법을 지정합니다.

원형 데이터에 대해 입력 값 래스터 셀 값을 처리할 방법을 지정합니다.

선택하지 않으면 일반 산술(선형) 통계가 계산됩니다. 선택하면 원형 통계가 계산됩니다.

주어진 원형 통계의 범위로 선형 값을 반올림하는 데 사용될 값을 지정합니다.

입력 값 래스터의 NoData 셀이 속한 구역의 결과에서 해당 셀을 무시할지 여부를 지정합니다.

선택한 경우 값 래스터의 NoData 셀은 통계 계산에서 무시됩니다. 선택하지 않은 경우 NoData 셀은 무시되며 구역 전반의 결과 래스터에 NoData 값이 할당됩니다.

입력 래스터가 다차원일 경우 처리 방법을 지정합니다.

선택하지 않으면 입력 다차원 데이터셋의 모든 차원에서 통계가 계산됩니다. 선택하면 입력 다차원 데이터셋의 현재 슬라이스에서 통계가 계산됩니다.

생성되어 맵에 추가되는 구역 통계 래스터 레이어의 이름입니다.

생성될 결과 유형입니다. 결과는 타일 영상 레이어 또는 동적 영상 레이어일 수 있습니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
