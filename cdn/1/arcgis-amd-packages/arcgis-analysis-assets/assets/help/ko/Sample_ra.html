샘플 도구는 정의된 위치에서 래스터나 래스터 집합의 픽셀 값을 보여주는 테이블 또는 포인트 피처 클래스를 생성합니다.

입력 위치 데이터를 기반으로 샘플링될 값을 포함하는 영상 레이어입니다.

각 샘플의 위치를 포함하는 레이어입니다.

샘플링 프로세스 중에 사용될 리샘플링 알고리즘입니다.

각 위치 또는 입력 피처에 대해 서로 다른 값을 포함하는 레이어의 필드입니다.

입력 영상 레이어에 다차원 데이터가 포함되어 있는지 여부를 나타냅니다.

위치 피처와 관련된 시간, 깊이 또는 기타 취득 데이터를 지정합니다.

다차원 입력 레이어의 차원입니다.

다차원 입력 레이어의 시간 범위를 시작하는 데 사용될 필드입니다.

다차원 입력 레이어의 시간 범위를 종료하는 데 사용될 필드입니다.

차원 매개변수에 추가될 음수가 아닌 숫자 또는 차원 값 범위의 시작을 정의하는 값입니다.

차원 매개변수에 추가될 음수가 아닌 숫자 또는 차원 값 범위의 종료를 정의하는 값입니다.

다차원 영상 레이어를 처리할 때 생성될 통계입니다.

백분위수 통계 유형에 사용될 백분율 값입니다.

입력 위치에 적용될 버퍼 거리입니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과 유형이 피처인지 아니면 테이블인지 여부를 지정합니다.

결과가 행 전체인지 아니면 열 전체인지 여부를 지정합니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
