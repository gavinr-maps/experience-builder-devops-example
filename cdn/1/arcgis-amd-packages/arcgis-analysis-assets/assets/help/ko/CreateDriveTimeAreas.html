이동 영역 생성 도구는 이동 시간 또는 거리 이내에 도달할 수 있는 영역을 생성합니다.

계산할 영역의 주변 포인트 피처입니다.

기본 도로에서 이동할 때 임시 제한(경계)으로 사용할 하나 이상의 포인트 피처입니다. 경계를 통과하지 않는 경우 도로 이동이 허용됩니다.

기본 도로에서 이동할 때 임시 제한(경계)으로 사용할 하나 이상의 라인 피처입니다. 라인 경계는 경계가 도로와 교차하는 모든 위치에서 이동을 금지합니다.

기본 도로에서 이동할 때 임시 제한(경계)으로 사용할 하나 이상의 폴리곤 피처입니다. 폴리곤 경계는 폴리곤이 도로와 교차하는 모든 위치에서 이동을 금지합니다.

이동 영역을 생성하는 데 사용할 이동 모드입니다. 이동 모드는 거리 또는 시간을 기준으로 할 수 있습니다.

각 입력 포인트의 이동 거리 또는 시간 제한입니다. 여러 값이 추가되는 경우 각 값은 모든 입력 포인트에 대한 이동 영역을 계산하는 데 사용됩니다.

이동 거리 또는 시간을 지정하는 데 사용되는 단위입니다. 한계값은 해당 단위로 해석됩니다.

입력 포인트에서 멀리 이동하거나 입력 포인트를 향하여 이동하기 위해 이동 영역을 생성할지 여부를 결정합니다.

트래블러가 입력 위치에서 출발하거나 도착하는 날짜 및 시간을 지정합니다.

여러 입력의 공유 영역이 서로 겹치는 경우 이러한 영역을 처리할 방법을 결정합니다.

각 결과 영역이 하나의 연속 폴리곤이 될지(선택 안 됨) 또는 접근할 수 없지만 접근 가능한 도로로 완전히 둘러싸인 도로에 홀을 포함할지(선택됨) 여부를 지정합니다.

생성되어 맵에 추가되는 레이어 이름입니다.

이동 영역을 생성하는 데 사용되는 도로를 결과 레이어로 포함할지 지정합니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
