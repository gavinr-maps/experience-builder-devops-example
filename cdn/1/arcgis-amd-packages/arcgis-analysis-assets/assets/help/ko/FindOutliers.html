이상치 찾기 도구는 데이터의 공간 패턴을 기반으로 인접 피처와 크게 다른 값이 있는 위치가 있는지 확인합니다.

이상치를 찾을 포인트 또는 폴리곤 레이어입니다.

피처 위치(포인트 수) 또는 피처 필드 값(필드)을 기반으로 이상치를 결정할지 여부를 지정합니다.

이상치를 결정하기 위해 분석될 필드입니다.

입력 피처가 집계될 경계 피처의 유형입니다. 이는 폴리곤 레이어(폴리곤 레이어) 또는 사각형(그물망 셀)이나 육각형(육각형 셀)의 공간 분할 그리드가 될 수 있습니다.

입력 포인트 피처가 집계될 폴리곤 피처를 포함하는 레이어입니다.

분석의 경계를 정의할 폴리곤 레이어입니다. 폴리곤 내의 피처는 이상치 분석에 포함됩니다.

분석 필드 값 또는 집계된 포인트 개수를 나누는 방법을 지정합니다. 필드 옵션은 분석 필드 값을 입력 레이어 매개변수에 지정된 필드 값으로 나눕니다. 데이터 보강 옵션은 인구 데이터로 피처 또는 집계 쉐이프를 보강하고 분석 필드 값 또는 집계된 포인트 개수를 인구로 나눕니다. 이 옵션을 사용할 경우 크레딧이 추가로 소비됩니다.

처리 시간을 줄이거나 결과 정밀도를 개선하기 위해 도구를 최적화할지 지정합니다.

랜덤 번호 생성기를 시작할 정수(Integer) 값입니다.

입력 피처를 집계하는 데 사용될 사각형(그물망) 또는 육각형 셀의 한 변의 길이입니다.

셀 크기에 사용될 단위를 지정합니다.

인접 영역을 정의하는 데 사용될 거리 밴드입니다. 대상 피처의 거리 밴드 내 인접 피처는 분석 중에 대상 피처의 인접 영역에 포함됩니다.

거리 밴드에 사용될 단위를 지정합니다.

생성되어 맵에 추가되는 레이어 이름입니다.

결과를 저장할 내 콘텐츠의 폴더 이름입니다.
