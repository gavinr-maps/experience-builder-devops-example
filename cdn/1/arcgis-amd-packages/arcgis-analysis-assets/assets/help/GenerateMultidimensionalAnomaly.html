Generate Multidimensional Anomaly

The Generate Multidimensional Anomaly computes the anomaly for each slice in an existing multidimensional imagery layer to generate a new multidimensional imagery layer.

The multidimensional imagery layer that will be analyzed.

The variable of the multidimensional imagery layer that will be used to locate anomalies.

The method that will be used to calculate the anomalies in the imagery layer.

The temporal interval that will be used to calculate the mean.

The reference imagery layer that contains the calculated means for each pixel.

Specifies whether NoData or missing values will be ignored in the analysis.

The name of the layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
