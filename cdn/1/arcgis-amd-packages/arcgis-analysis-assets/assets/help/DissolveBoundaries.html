The Dissolve Boundaries tool finds polygons that overlap or share a common boundary and merges them to form a single polygon.

The layer containing polygon features that will be dissolved.

Determines whether the tool will dissolve only the polygons that overlap or share a common border and have the same field value (enabled) or all the polygons that overlap or share a common border (disabled).

Specifies the field used to dissolve polygons based on matching values.

Specifies whether the output can contain multipart polygon features, meaning multiple polygons defined as a single feature (enabled), or can only contain individual features (disabled).

Determines whether summary statistics are calculated for the dissolved polygons.

The number or date fields from the input layer that will be summarized, and the statistics that will be calculated for each field.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
