The Interpolate Points tool predicts values at new locations based on point measurements. The tool creates a raster output.

The point layer that contains the points where the values have been measured.

The numeric field containing the values that will be interpolated.

Specifies whether speed or accuracy will be optimized. More accurate predictions take longer to calculate.

This parameter alters the default values of several other parameters.

The cell size and unit for the output rasters.

Specifies whether the data will be transformed or not. Interpolation is most accurate for data that follows a normal (bell-shaped) distribution.

The size of the local interpolation models that will be combined to create the final prediction map.

The number of neighbors that will be used. A larger number of neighbors generally produces more accurate results, but the results take longer to calculate.

The name of the interpolated raster layer that will be created and added to the map.

Choose whether you also want to create an optional raster of standard errors for the predicted values.

If checked, the output standard error raster will have the same name as the Output raster name, but with the word Errors appended to it. The layer will be created and added to the map.

The type of output that will be created. The outputs can be Tiled imagery layers or Dynamic imagery layers.

The name of the folder in My Content where the results will be saved.
