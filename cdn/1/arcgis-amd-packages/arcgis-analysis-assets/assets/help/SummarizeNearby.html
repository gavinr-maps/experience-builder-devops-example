The Summarize Nearby tool calculates statistics for features within a specified distance of other features. The distance can be calculated using a straight line distance or a travel mode.

The layer to summarize within a distance of the nearby layer.

The point, line, or polygon layer used as the starting locations around which bounding areas will be created to summarize nearby features.

The method with which the bounding areas will be created. The method can be straight line distance or a travel mode.

The distance or time that will be used to create bounding areas. If multiple values are added, each value will be used to calculate bounding areas for all input features.

The units that will be used to specify distance or time.

Specify whether the result layer should be the bounding areas (checked) or the same geometry as the input nearby layer (unchecked).

Calculate the total size of the features summarized in each bounding area. For lines, the total length is calculated. For polygons, the total area is calculated.

The units that will be used to calculate the total size of summarized features.

The number or date fields from the input layer that will be summarized, and the statistics that will be calculated for each field.

The field from the input layer that will be used to separate the data into distinct groups for statistical calculations.

Calculate the minority (least dominant) and majority (most dominant) groups within each summary area.

Calculate the percentage of the summarized features that belong to each group within each boundary.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the results will be saved.
