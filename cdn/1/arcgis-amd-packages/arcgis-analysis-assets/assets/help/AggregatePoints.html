The Aggregate Points tool calculates a count of points and other statistics within specified bins or polygons.

The point layer that will be aggregated into the summary areas.

The polygon layer in which the points will be aggregated.

The type of bounding features within which the input points will be aggregated. This can be a polygon layer or a tessellation grid of either square or hexagon bins that are generated when the tool runs.

The size of the generated bins. For square bins, the number and units specified determine the height and length of the square. For hexagon bins, the number and units specified determine the distance between parallel sides.

The units that will be used to specify bin size.

Determines whether features that have no points appear in the result layer. When unchecked, features without points will not appear in the result. When checked, features without points will appear in the result.

The number or date fields from the input layer that will be summarized, and the statistics that will be calculated for each field.

The field from the input layer that will be used to separate the data into distinct groups for statistical calculations.

Calculate the minority (least dominant) and majority (most dominant) groups within each summary area.

Calculate the percentage of points belonging to each group within each summary area.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
