Especifica com es processaran els valors de cel·la de Ràster de valor d'entrada per a dades circulars.

Si aquesta opció no està seleccionada, es calcularan estadístiques aritmètiques ordinàries (lineals). Si està seleccionada, es calcularan les estadístiques circulars.

Especifica el valor que es farà servir per arrodonir un valor lineal a l'interval d'una estadística circular determinada.

Especifica si les cel·les NoData del Ràster de valor d'entrada s'ignoraran als resultats de les zones a les quals pertanyen.

Si està seleccionada, les cel·les NoData del ràster de valor s'ignoraran al càlcul estadístic. Si no està seleccionada, les cel·les NoData no s'ignoraran, i tota la zona rebrà el valor NoData al ràster de sortida.

Especifica com es processaran els ràsters d'entrada si són multidimensionals.

Si no està seleccionada, les estadístiques es calcularan per a totes les dimensions del dataset multidimensional d'entrada. Si està seleccionada, les estadístiques es calcularan a partir del sector actual del dataset multidimensional d'entrada.
