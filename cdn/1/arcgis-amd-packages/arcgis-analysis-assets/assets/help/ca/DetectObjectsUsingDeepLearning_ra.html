L'eina Detecta objectes amb Aprenentatge profund fa servir un model d'aprenentatge profund en una capa d'imatges d'entrada per produir una capa d'entitats que conté els objectes detectats.

Capa d'imatges o d'entitats que es farà servir per detectar objectes.

Especifica com es processaran els elements ràster d'una capa d'imatges després de crear el mosaic: de manera conjunta o individualment.

Paquet d'aprenentatge profund que conté el model que es farà servir per detectar objectes.

Arguments del model que estan presents al model d'aprenentatge profund, amb els seus valors predeterminats.

Especifica si es durà a terme una supressió no màxima per eliminar objectes duplicats.

Nom del camp que s'afegirà a la capa d'entitats de sortida per indicar el valor de confiança notificat.

Nom del camp que s'afegirà a la capa d'entitats de sortida per indicar l'etiqueta classificada per a l'objecte.

Relació màxima permesa per a cada entitat trobada que se superposi a una altra entitat. El valor per defecte és 0.

Nom de la capa que es crearà i s'afegirà al mapa.

Nom de la carpeta a El meu contingut on es desarà el resultat.
