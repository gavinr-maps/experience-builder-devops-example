L'eina Extreu les dades selecciona les entitats dins d'una extensió i crea un element al vostre contingut.

Capes que s'extrauran. Entre les capes d'entrada compatibles s'inclouen les capes de la vostra propietat i les capes amb l'exportació habilitada.

Extensió de les entitats que s'extrauran.

Especifica si les entitats que intersequen l'extensió es retallaran. Quan aquesta opció estigui desactivada, s'extrauran les entitats senceres que intersequin l'extensió. Quan aquesta opció estigui activada, es retallaran les entitats que intersequin l'extensió i només s'extrauran les parts de les entitats dins de l'extensió.

Format del fitxer de les dades que s'extrauran.

Nom de l'element de sortida on s'emmagatzemarà el fitxer extret.

Nom de la carpeta a El meu contingut on es desarà el resultat.
