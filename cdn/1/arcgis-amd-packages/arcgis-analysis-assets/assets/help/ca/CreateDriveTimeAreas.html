L'eina Genera àrees de viatge crea àrees on es pugui arribar en una distància o temps de recorregut.

Entitats de punts al voltant de les quals es calcularan les àrees.

Una o diverses entitats de punt que actuen com a restriccions temporals (barreres) quan viatgeu per carrers subjacents. Es pot viatjar pel carrer, però no creuar la barrera.

Una o diverses entitats de línia que actuen com a restriccions temporals (barreres) quan viatgeu per carrers subjacents. Les barreres de línia prohibeixen la circulació a qualsevol punt en què les barreres intersequen els carrers.

Una o diverses entitats de polígon que actuen com a restriccions temporals (barreres) quan viatgeu per carrers subjacents. Les barreres de polígon prohibeixen viatjar a qualsevol lloc on els polígons intersequen amb els carrers.

Mode de viatge que s'utilitzarà per crear les àrees de viatge. El mode de viatge es pot basar en la distància o en el temps.

Distància del trajecte o límit de temps per a cada punt d'entrada. Si s'afegeixen diversos valors, cada valor s'utilitzarà per calcular les àrees de viatge per a tots els punts d'entrada.

Unitats que s'utilitzen per especificar la distància o el temps de viatge. El Valor límit s'interpreta en aquestes unitats.

Determina si les àrees de viatge es creen per allunyar-se dels punts d'entrada o per apropar-s'hi.

Especifica la data i l'hora de sortida o arribada del viatger a les ubicacions d'entrada.

Determina com es gestionen les àrees compartides de diferents entrades quan conflueixen entre si.

Especifica si cada àrea de sortida serà un polígon continu (desactivat) o inclourà forats a carrers als quals no es pot arribar, però que estan envoltats per carrers als quals sí que es pot arribar (activat).

Nom de la capa que es crearà i s'afegirà al mapa.

Especifica si els carrers que s'utilitzen per crear les àrees de viatge s'inclouran com una capa de sortida.
