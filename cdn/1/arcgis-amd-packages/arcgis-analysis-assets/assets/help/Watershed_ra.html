The Watershed tool determines the contributing area above a set of cells in a raster.

The direction of flow out of each cell.

The D8 method must be used to determine flow direction.

The locations above which the contributing area will be determined.

The field that will be used to assign values to the pour point locations.

The name of the watershed raster layer that will be created and added to the map. The layer contains different watersheds represented by different cell values from the Pour point field.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
