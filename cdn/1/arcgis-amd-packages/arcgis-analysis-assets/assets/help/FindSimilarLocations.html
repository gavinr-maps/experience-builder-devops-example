The Find Similar Locations tool evaluates the similarity between candidate locations and a reference location based on a set of criteria and ranks the candidate locations from most to least similar.

The point, line, or polygon layer that contains the reference locations that will be matched. If the layer includes multiple locations, all the locations will be averaged to create a single reference location.

The point, line, or polygon layer that contains the search candidate locations. Each search candidate location will be compared to the reference location by the specified Criteria selection value and ranked from most to least similar.

The numeric fields that will be used to evaluate the similarity between the reference location and each search candidate location. Each Criteria selection value will have a Field from reference layer value and a Field from candidate layer value. The Field from reference layer and Field from candidate layer values are matching fields in the Reference locations layer and Search candidates layer values.

The Reference locations layer field that corresponds to the Search candidates layer field specified by the Field from candidate layer value. The values in these matched fields will be compared to evaluate the similarity between the reference location and each search candidate location.

The Search candidates layer field that corresponds to the Reference locations layer field specified by the Field from reference layer value. The values in these matched fields will be compared to evaluate the similarity between the reference location and each search candidate location.

Specifies whether all the locations in the Search candidate layer value will be included in the output layer.

The number of most similar locations in the Search candidate layer value that will be included in the output layer.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
