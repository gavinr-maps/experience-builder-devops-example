The Derive Continuous Flow tool generates a raster of accumulated flow into each cell from an input surface raster with no prior sink or depression filling required.

The elevation raster that will be used for calculation.

The layer that defines real depressions or sinks, where water can flow into but not out from.

The raster layer that defines the fraction of flow that contributes to flow accumulation at each cell.

Specifies the type of flow method that will be used when computing flow direction.

The D8 method assigns flow direction to the steepest downslope neighbor. The MFD method partitions the flow direction across downslope neighbors according to an adaptive partition exponent.

Specifies whether flow at edge cells will follow normal flow rules or always flow outward.

If unchecked, the flow will be toward the inner cell with the steepest drop in z-value. However, if the drop is less than or equal to zero, the cell will flow outward from the surface raster. If checked, the flow direction will always be outward from the surface raster.

The name of the flow accumulation raster layer that will be created and added to the map.

The name of the flow direction raster layer that will be created and added to the map.

The type of output that will be created. The outputs can be Tiled imagery layers or Dynamic imagery layers.

The name of the folder in My Content where the results will be saved.
