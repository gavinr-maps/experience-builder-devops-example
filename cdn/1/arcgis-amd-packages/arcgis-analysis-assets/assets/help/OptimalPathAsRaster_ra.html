The Optimal Path as Raster tool determines the optimal raster path from destinations to sources.

The layer that identifies the destinations from which the distance is calculated to the least costly source.

If the input is a raster, it must consist of cells that have valid integer values (zero is a valid value) for the destinations, and the remaining cells must be assigned NoData.

The field that will be used to obtain values for the destination locations.

The raster that determines the optimal path from the destinations to the sources.

The distance accumulation raster is usually created with the Distance Accumulation or Distance Allocation tool.

The raster that contains calculated directions in degrees.

The back direction raster is usually created with the Distance Accumulation or Distance Allocation tool.

Specifies how the values and zones in the input destination data will be interpreted in the optimal path calculations.

The name of the optimal path as raster layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
