Інструмент аналізу «Підсумувати в межах» обчислює статистику для об'єктів у межах заданих контейнерів або полігонів.

Шар, який буде підсумовано.

Тип обмежувальних об’єктів, у межах яких буде підсумовано вхідні об’єкти. Це може бути шар полігонів або сітка замощення квадратних або шестикутних контейнерів, які генеруються під час роботи інструменту.

Шар полігонів, у межах якого буде підсумовано вхідні об’єкти.

Розмір згенерованих контейнерів. Для квадратних контейнерів указана кількість і одиниці визначають висоту та довжину квадрата. Для шестикутних контейнерів вказана кількість та одиниці визначають відстань між паралельними сторонами.

Одиниці, які будуть використовуватися для визначення розміру контейнера.

Обчислити загальний розмір об’єктів, підсумованих у кожному обмежувальному полігоні. Для ліній обчислюється загальна довжина. Для полігонів обчислюється загальна площа.

Одиниці вимірювання, які будуть використовуватися для обчислення загального розміру підсумованих об'єктів.

Поля чисел або дат із вхідного шару, які будуть підсумовані, і статистика, яка обчислюватиметься для кожного поля.

Поле з вхідного шару, яке буде використане для розділення даних на окремі групи для статистичних обчислень.

Обчислити групу меншості (найменш значущих) і більшості (найбільш значущих) у кожній підсумковій області.

Обчислити відсоток підсумованих об'єктів, які належать до кожної групи в межах кожної підсумованої області.

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
