Інструмент «Виконати інтерполяцію точок» прогнозує значення на нових місцях розташування на основі вимірювань точок. Інструмент створює вихідний растр.

Точковий шар, який включає точки, в яких вимірювалися значення.

Числове поле, що містить значення, які будуть інтерпольовані.

Визначає, чи буде оптимізовано швидкість чи точність. Більш точне прогнозування вимагає більше часу для обчислення.

Цей параметр змінює значення за замовчуванням кількох інших параметрів.

Розмір осередку і одиниці виміру для вихідних растрів.

Визначає, чи будуть дані перетворені чи ні. Інтерполяція найбільш точна для даних, які задовольняють нормальному (дзвоноподібному) розподілу.

Розмір локальних моделей інтерполяції, які будуть об’єднані для створення остаточної карти прогнозування.

Кількість сусідів, які будуть використані. Більше число точок зазвичай забезпечує більш точний результат, але для розрахунку потрібно більше часу.

Ім'я інтерпольованого растрового шару зони, який буде створено та додано до карти.

Вибрати, чи потрібно також створювати додатковий растр стандартних помилок для прогнозованих значень.

Якщо вибрано, вихідний растр стандартних помилок матиме таку саму назву, як і назва Вихідного растра, але до нього буде додано слово «Помилки». Шар буде створено та додано до карти.

Тип вихідних даних, які буде створено. Вихідними даними можуть бути шари зображень з тайлами або динамічні шари зображень.

Назва папки в Моєму вмісті, де буде збережено результат.
