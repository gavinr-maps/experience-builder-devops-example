Інструмент «Створити багатовимірну аномалію» обчислює аномалію для кожного зрізу в існуючому багатовимірному шарі зображень, щоб створити новий багатовимірний шар зображень.

Багатовимірний шар зображень, який буде аналізовано.

Змінна багатовимірного шару зображень, яка буде використовуватися для пошуку аномалій.

Метод, який буде використовуватися для розрахунку аномалій у шарі зображень.

Часовий інтервал, який буде використовуватися для обчислення середнього значення.

Еталонний шар зображень, який містить обчислені середні значення для кожного пікселя.

Визначає, чи ігноруватимуться NoData або відсутні значення під час аналізу.

Назва шару, який буде створено та додано до карти.

Тип вихідних даних, які буде створено. Вихідними даними може бути шар зображень з тайлами або динамічний шар зображень.

Назва папки в Моєму вмісті, де буде збережено результат.
