Інструмент «Геокодування місць розташування з таблиці» перетворює адреси або місця зі вхідної таблиці в координати.

Вхідна таблиця з адресами або місцями для геокодування. Вхідними даними мають бути таблиця, файл .csv або файл Excel.

Сервіс локатора, використовуваний для геокодування. Якщо потрібний сервіс геокодування відсутній у розкривному меню, зверніться до вашого адміністратора, щоб він налаштував сервіс локатора для пакетного геокодування.

Указує, чи вхідні дані містяться в одному або в декількох полях.

Поля, які використовуватимуться локатором для визначення місць розташування даних. Очікується, що в даних є наявними певні поля, як-от адреса, місто, штат та поштовий індекс.

Країна, у якій розташовуються дані. Якщо всі адреси знаходяться в одній країні, укажіть цю країну для отримання максимально точних результатів. Або залиште цей параметр порожнім.

Указує поля, які будуть включені у набір вихідних даних. Доступні такі опції:

  • Усе—У результати геокодування будуть включені всі наявні вихідні поля локатора.
  • Мінімум—У результати будуть включені піднабір полів із локатора і поля зі вхідної таблиці.
  • Тільки місце розташування—У результати будуть включені поле Shape із локатора і поля зі вхідної таблиці.

Тип вихідних даних, які буде створено. Вихідними даними може бути сервіс об'єктів, файл .csv або файл .xls.

Ім'я вихідного елемента, де будуть збережені розміщені дані.

Назва папки в Моєму вмісті, де буде збережено результат.
