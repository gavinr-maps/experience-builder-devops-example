Інструмент «Виконати інтерполяцію точок» прогнозує значення на нових місцях розташування на основі вимірювань точок. Інструмент створює вихідні об'єкти.

Вибрати точковий шар місць розташування, які будуть інтерпольовані.

Вибрати поле, що містить виміряні значення, які будуть інтерпольовані.

Вибрати, чи буде пріоритетом швидкість обчислення чи точність. Для обчислення більш точних прогнозів потрібно більше часу.

Вибрати, чи буде включений у вихідні дані шар стандартних помилок для прогнозів інтерполяції.

Вибрати, як будуть обчислюватися значення роздільників класу.

Указати кількість класів (діапазонів прогнозованих значень), які будуть включені в шар результатів. Кожен клас визначає кордони полігону та представляє контури прогнозованих значень. Значення роздільників класу визначаються типом класифікації.

Указати значення роздільників класу, які будуть використані.

Вибрати шар полігонів, який буде використовуватися для визначення області, що вивчається. Результати будуть обрізані для відображення лише в межах області, що вивчається.

Вибрати шар точок, що представляє місця розташування, які будуть використовуватися для прогнозування значень. Якщо надано, шар точок, що містить прогнозовані значення у вказаних місцях розташування, буде створено.

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
