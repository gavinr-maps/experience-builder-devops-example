Інструмент «Підсумувати поруч» обчислює статистику для об’єктів на певній відстані від інших об’єктів. Відстань можна обчислити за допомогою відстані по прямій лінії або режиму переміщення.

Шар для підсумовування на відстані від сусіднього шару.

Шар точок, ліній або полігонів, який використовується як початкове місце розташування, навколо якого буде створено обмежувальні області для підсумування прилеглих об'єктів.

Метод, за допомогою якого будуть створені обмежувальні області. Методом може бути відстань по прямій лінії або режим переміщення.

Відстань або час, який буде використано для створення обмежувальних областей. Якщо додаються кілька значень, кожне значення буде використано для обчислення обмежувальних площ для всіх вхідних об’єктів.

Одиниці вимірювання, які будуть використовуватися для визначення відстані або часу.

Укажіть, чи повинен шар результатів бути обмежувальними областями (вибрано) чи такою ж геометрією, що й вхідний прилеглий шар (не вибрано).

Обчисліть загальний розмір об’єктів, підсумованих у кожній обмежувальній області. Для ліній обчислюється загальна довжина. Для полігонів обчислюється загальна площа.

Одиниці вимірювання, які будуть використовуватися для обчислення загального розміру підсумованих об'єктів.

Поля чисел або дат із вхідного шару, які будуть підсумовані, і статистика, яка обчислюватиметься для кожного поля.

Поле з вхідного шару, яке буде використане для розділення даних на окремі групи для статистичних обчислень.

Обчислити групу меншості (найменш значущих) і більшості (найбільш значущих) у кожній підсумковій області.

Обчисліть відсоток підсумованих об'єктів, які належать до кожної групи в межах кожного кордону.

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
