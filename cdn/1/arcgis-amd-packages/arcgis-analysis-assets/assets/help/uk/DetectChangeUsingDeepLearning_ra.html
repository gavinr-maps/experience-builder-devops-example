Інструмент "Виявити зміни за допомогою глибокого навчання" використовує модель глибокого навчання для виявлення змін між двома шарами растрів.

Вхідний шар зображень до зміни, який буде використовуватися як основа для порівняння.

Вхідний шар зображень після зміни, який буде використовуватися для порівняння з оригіналом.

Вихідний класифікований шар зображень, який показує зміни.

Аргументи моделі, які присутні в моделі глибокого навчання разом зі значеннями за замовчуванням.

Пакет глибокого навчання, який містить модель, що буде використана для виявлення змін між шарами зображень.

Тип вихідних даних, які буде створено. Вихідними даними може бути шар зображень з тайлами або динамічний шар зображень.

Назва папки в Моєму вмісті, де буде збережено результат.
