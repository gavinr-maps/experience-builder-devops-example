Інструмент «Агрегувати точки» обчислює кількість точок та іншу статистику в межах указаних контейнерів або полігонів.

Шар точок, які буде агреговано в області підсумовування.

Шар полігонів, у якому будуть агреговані точки.

Тип обмежувальних об’єктів, у межах яких буде агреговано вхідні точки. Це може бути шар полігонів або сітка замощення квадратних або шестикутних контейнерів, які генеруються під час роботи інструменту.

Розмір згенерованих контейнерів. Для квадратних контейнерів указана кількість і одиниці визначають висоту та довжину квадрата. Для шестикутних контейнерів вказана кількість та одиниці визначають відстань між паралельними сторонами.

Одиниці, які будуть використовуватися для визначення розміру контейнера.

Визначає, чи відображаються об’єкти, які не мають точок, у шарі результатів. Якщо не вибрано, об'єкти без точок не будуть відображатися в результатах. Якщо вибрано, об'єкти без точок будуть відображатися в результатах.

Поля чисел або дат із вхідного шару, які будуть підсумовані, і статистика, яка обчислюватиметься для кожного поля.

Поле з вхідного шару, яке буде використане для розділення даних на окремі групи для статистичних обчислень.

Обчислити групу меншості (найменш значущих) і більшості (найбільш значущих) у кожній підсумковій області.

Обчислити відсоток точок, що належать кожній групі в межах кожної області підсумовування.

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
