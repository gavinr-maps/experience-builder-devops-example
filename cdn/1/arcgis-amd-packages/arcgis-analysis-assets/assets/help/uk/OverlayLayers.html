Інструмент «Накласти шари» об'єднує два шари в один шар на основі просторового відношення.

Точковий, лінійний або полігональний шар, який буде накладатись.

Шар для накладення на вхідний шар.

Метод накладення, який буде використовуватися для визначення, того як вхідний шар та шар накладення об’єднуються.

Тип об’єктів (точкові, лінійні або полігональні), які будуть створені у вихідному шарі.

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
