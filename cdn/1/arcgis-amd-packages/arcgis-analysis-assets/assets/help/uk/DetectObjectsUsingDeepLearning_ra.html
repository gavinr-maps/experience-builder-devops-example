Інструмент «Виявити об'єкти, використовуючи глибоке навчання» використовує модель глибокого навчання на вхідному шарі зображень для створення векторного шару, що містить виявлені об’єкти.

Зображення або векторний шар, який буде використовуватися для виявлення об’єктів.

Визначає спосіб обробки растрових елементів у шарі зображень після мозаїки: разом чи окремо.

Пакет глибокого навчання, який містить модель, яка буде використовуватися для виявлення об’єктів.

Аргументи моделі, які присутні в моделі глибокого навчання разом зі значеннями за замовчуванням.

Вказує, чи виконуватиметься ігнорування немаксимумів для видалення дублікатів об’єктів.

Ім’я поля, яке буде додано до вихідного векторного шару, щоб вказати повідомлене значення достовірності.

Ім’я поля, яке буде додано до вихідного векторного шару, щоб вказати класифікований напис для об’єкта.

Максимально допустимий коефіцієнт для кожного знайденого об'єкта, який перекриває інший об'єкт. Значення за замовчуванням 0.

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
