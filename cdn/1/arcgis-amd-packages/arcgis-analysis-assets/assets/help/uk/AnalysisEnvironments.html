Параметри середовища для аналізу в Map Viewer.

Визначає систему координат для аналізу та шар результатів.

Визначає область, яку необхідно використовувати для аналізу.

Регулює екстент вихідних даних так, щоб вони відповідали вирівнюванню осередку вказаного шару растру прив'язки.

Визначає розмір осередку або роздільну здатність, які будуть використовуватись для аналізу та створення вихідного растрового шару.

Визначає шар маски, в якому для аналізу будуть використовуватися лише осередки, що потрапляють у область маски.

Визначає, як будуть інтерполюватися значення пікселів, коли вхідний і вихідний растри не збігаються.

Визначає тип використовуваного процесора. Доступні варіанти ЦП, ГП і Автоматично. Якщо задано Автоматично, тип процесора визначатиметься інструментом.

Указує кількість екземплярів сервісу обробки растру, використовуваних під час роботи інструменту. Підтримуються такі значення як порожні рядки (інструмент визначатиме кількість екземплярів сервісу обробки растру), цілі числа (0 і додатні цілі числа) і відсоткові значення. Відсоткові значення повинні включати знак відсотку (%) і число.

Визначає, чи з'являтиметься попереджувальне повідомлення, коли орієнтовна кількість кредитів перевищує задане порогове значення (активовано), або чи виконуватиметься аналіз без відображення повідомлення незалежно від орієнтованого споживання кредитів (не активовано). Втім, в аналізі й надалі враховуватимуться налаштування формування бюджету кредитів для організації.

Указує порогове значення, при досягненні якого з'являється попереджувальне повідомлення на основі орієнтовного споживання кредитів при виконанні аналізу. Якщо орієнтовна кількість перевищить порогове значення, виконання аналізу продовжиться.
