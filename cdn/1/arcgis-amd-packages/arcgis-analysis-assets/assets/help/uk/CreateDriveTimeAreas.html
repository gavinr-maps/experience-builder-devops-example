Інструмент «Створити області переміщення» створює області, яких можна дістатися в межах часу подорожі або відстані.

Точкові об’єкти, навколо яких будуть обчислені області.

Один або більше точкових об'єктів, які діють у якості тимчасових обмежень (бар'єрів) під час подорожування нижчележачими вулицями. Рух по вулиці дозволено, але тільки не через бар'єр.

Один або більше лінійних об'єктів, які діють у якості тимчасових обмежень (бар'єрів) під час подорожування нижчележачими вулицями. Лінійні бар'єри забороняють рух у будь-якому місці, де бар'єри перетинаються з вулицями.

Один або більше полігональних об'єктів, які діють у якості тимчасових обмежень (бар'єрів) під час подорожування нижчележачими вулицями. Полігональні бар'єри забороняють рух у будь-якому місці, де полігони перетинають вулиці.

Режим подорожі, який буде використаний для створення областей переміщення. Режим подорожі може залежати від відстані або часу.

Відстань або часовий ліміт переміщення для кожної вхідної точки. Якщо додаються кілька значень, кожне значення буде використано для обчислення областей переміщення для всіх вхідних точок.

Одиниці вимірювання, які використовуються для визначення відстані або часу переміщення. Граничне значення інтерпретується в цих одиницях.

Визначає, чи створюються області переміщення для переміщень від вхідних точок чи до вихідних точок.

Визначає дату й час, коли мандрівник відправляється з або прибуває до вхідних місць розташування.

Визначає, як обробляються спільні області з різних вхідних точок, коли вони досягають одна одної.

Визначає, чи кожна вихідна область буде одним безперервним полігоном (не вибрано), чи включатиме отвори на вулицях, які недоступні, але повністю оточені доступними вулицями (вибрано).

Назва шару, який буде створено та додано до карти.

Указує, чи включати вулиці, які використовуються для створення областей переміщення, як вихідний шар.

Назва папки в Моєму вмісті, де буде збережено результат.
