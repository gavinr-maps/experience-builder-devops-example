Інструмент «Згенерувати замощення» створює сітку полігонів на основі заданих екстенту, форми та розміру.

Форма замощень, які будуть створені.

Визначає, чи буде розмір замощення визначатися його площею (Площа) чи значенням відстані (Відстань).

Розмір (площа або відстань) згенерованих замощень. Для квадратів, шестикутників, трикутників і ромбів відстань визначає висоту замощення. Для поперечних шестикутників відстань визначає ширину замощення.

Роздільна здатність H3 замощень. Роздільна здатність знаходиться в діапазоні від 0 до 15. З кожним збільшенням роздільної здатності площа полігонів зменшуватиметься на одну сьому розміру за найнижчої роздільної здатності.

Указує одиниці вимірювання, які будуть використовуватися для розміру замощення.

Екстент, який покриватиме замощення.

Для векторних шарів указує, чи охоплюватимуть замощення весь екстент чи лише ті області, де замощення перетинаються з об’єктами.

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
