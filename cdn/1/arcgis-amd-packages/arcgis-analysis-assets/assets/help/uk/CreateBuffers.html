Інструмент «Створити буфери» створює полігони, які покривають дану відстань від вхідних об'єктів.

Точкові, лінійні або полігональні об'єкти, які будуть буферизуватись.

Визначає, чи буферна відстань буде визначена константою (Значення) чи числовим полем (Field).

Відстань, яка буде використана для створення буфера. Якщо додаються кілька значень, кожне значення буде використано для створення буфера для кожного вхідного об'єкта.

Поле, яке буде містити буферну відстань для кожної функції.

Одиниці вимірювання, які будуть використовуватись для визначення буферної відстані.

Визначає, як будуть оброблятися буфери з різних вхідних об'єктів, коли вони накладаються.

Визначає, чи будуть буфери з кількома відстанями концентричними буферами, що перекриваються (Диски, що перекриваються), чи кільцями, які не перекриваються (Окремі кільця).

Для лінійних об’єктів указує, чи буфер буде створено з обох сторін лінії (Навколо), лише з лівого боку (Ліворуч) чи лише з правого боку (Праворуч).

Для полігональних об’єктів указує, чи будуть вхідні полігональні об’єкти включені до буфера (Включити) чи виключені з буфера (Виключити).

Визначає, чи буде форма буфера в кінці лінійного об’єкта круглою (Кругла) чи плоскою (Плоска).

Назва шару, який буде створено та додано до карти.

Назва папки в Моєму вмісті, де буде збережено результат.
