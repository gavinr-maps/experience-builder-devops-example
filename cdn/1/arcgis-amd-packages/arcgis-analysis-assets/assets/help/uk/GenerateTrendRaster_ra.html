Інструмент «Згенерувати растр тренду» оцінює тренд для кожного пікселя за виміром для однієї або декількох змінних у багатовимірному растрі зображень.

Багатовимірний шар зображень, який буде використовуватися для створення трендів.

Вимір, за яким витягатиметься тренд для змінної або змінних, вибраних в аналізі.

Змінна або змінні, для яких розраховуватимуться тренди.

Одиниця часу, яка буде використовуватися для тривалості сезонного періоду під час використання лінії типу «Сезонний-Кендалл».

Тип аналізу трендів, який буде використовуватися для значень пікселів за виміром.

Частота або порядковий номер полінома, який буде використовуватися для припасування тренду.

Визначає, чи ігноруватимуться NoData або відсутні значення під час аналізу.

Довжина періодичної зміни, яка буде змодельована для гармонійного циклу.

Одиниця часу, яка буде використовуватися для вимірювання тривалості гармонійного циклу.

Визначає, чи буде обчислюватися середньоквадратична помилка (RMSE) лінії припасування тренду.

Визначає, чи буде обчислюватися R-квадрат статистики відповідності для лінії відповідності тренду.

Визначає, чи буде обчислено статистичне значення p для коефіцієнта нахилу лінії відповідності тренду.

Назва шару, який буде створено та додано до карти.

Тип вихідних даних, які буде створено. Вихідними даними може бути шар зображень з тайлами або динамічний шар зображень.

Назва папки в Моєму вмісті, де буде збережено результат.
