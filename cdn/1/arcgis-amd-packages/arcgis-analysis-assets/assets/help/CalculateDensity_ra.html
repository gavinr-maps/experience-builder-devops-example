The Calculate Density tool calculates a density estimate from point and line features. The tool creates a raster output.

The point or line layer from which density will be calculated.

The field that specifies the number of incidents at each location.

The layer that defines the barriers. The barriers can be a feature layer of polyline or polygon features.

The cell size and unit that will be used for the output raster.

Specifies the distance to search points or lines when calculating density, using a value and its units.

The units that will be used for the output area.

Density is a count divided by area, and this parameter specifies the unit of the area in the density calculation.

The name of the density raster layer that will be created and added to the map.

The type of output that will be created. The outputs can be Tiled imagery layers or Dynamic imagery layers.

The name of the folder in My Content where the results will be saved.
