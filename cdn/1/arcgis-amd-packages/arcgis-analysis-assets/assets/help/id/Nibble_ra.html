Alat Nibble menggantikan sel-sel raster yang sesuai dengan topeng dengan nilai-nilai tetangga terdekat.

Raster input dengan lokasi bertopeng yang akan diganti (dikikis) dengan nilai di sekitar terdekatnya.

Raster topeng yang mengidentifikasi lokasi di raster input yang akan diganti. Sel dengan nilai Tidak ada data dianggap berada dalam area bertopeng.

Raster zona input. Untuk setiap zona, sel input yang berada di dalam masker hanya akan digantikan oleh nilai sel terdekat dalam zona yang sama.

Menentukan apakah sel Tidak ada data di raster input dapat menggantikan (mengikis) sel di area yang disamarkan jika sel tersebut merupakan sel disekitar terdekat.

Jika dicentang, sel Tidak ada data di raster input dapat masuk ke area yang ditutupi. Jika tidak dicentang, hanya sel valid dalam raster input yang dapat masuk ke area yang ditutupi.

Menentukan apakah sel Tidak ada data di area bertopeng akan dipertahankan atau diganti.

Jika tidak dicentang, sel Tidak ada data apa pun dalam area bertopeng akan dipertahankan dalam output. Jika dicentang, sel Tidak ada data di dalam area bertopeng akan diganti (dikikis) dengan nilai di sekitar terdekat yang berada di luar area bertopeng.

Nama layer raster yang akan dibuat dan ditambahkan ke peta.

Jenis output yang akan dibuat. Output dapat berupa layer citra Ubin atau layer citra Dinamis.
