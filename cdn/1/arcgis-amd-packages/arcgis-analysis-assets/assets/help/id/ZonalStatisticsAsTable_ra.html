Menentukan nilai persentil yang akan dihitung.

Menentukan metode yang akan digunakan ketika jumlah nilai dari raster input yang akan dihitung berjumlah genap.

Menentukan cara pemrosesan nilai sel Raster nilai input untuk data sirkular.

Jika tidak dicentang, statistik aritmetika asli (linear) akan dihitung. Jika dicentang, statistik sirkular akan dihitung.

Menentukan nilai yang akan digunakan untuk untuk membulatkan nilai linier ke rentang statistik sirkular yang ditentukan.

Menentukan apakah sel NoData di Raster nilai input akan diabaikan pada hasil zona yang termasuk di dalamnya.

Jika dicentang, sel NoData di raster nilai akan diabaikan di penghitungan statistik. Jika tidak dicentang, sel NoData akan diabaikan dan seluruh zona akan menerima nilai NoData di raster output.

Menentukan cara pemrosesan raster input jika raster input tersebut bersifat multidimensional.

Jika tidak dicentang, statistik akan dihitung untuk seluruh dimensi set data multidimensi. Jika dicentang, statistik akan dihitung dari irisan saat ini dari input set data multidimensi.
