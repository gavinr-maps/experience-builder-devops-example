Layer titik input akan dikelompokkan ke beberapa klaster berdasarkan distribusi spasial fitur titik.

Menentukan unit yang akan digunakan untuk mencari jarak.

Bilangan bulat antara 0 dan 100 yang menentukan kepadatan kluster. Jika dikosongkan, nilai Sensitivitas kluster akan dihitung menggunakan Divergensi Kullback-Leibler.
