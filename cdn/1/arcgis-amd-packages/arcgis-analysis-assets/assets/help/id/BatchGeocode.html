Alat Lokasi Geocode dari Tabel mengubah alamat atau tempat dari tabel input menjadi koordinat.

Tabel input dengan alamat atau tempat yang akan di-geocode. Inputnya harus berupa tabel, file .csv, atau file Excel.

Layanan penentu lokasi yang digunakan untuk geocoding. Jika service penentu lokasi yang ingin Anda gunakan tidak berada di menu tarik-turun, hubungi administrator agar service penentu lokasi dikonfigurasi untuk geocoding kelompok.

Menentukan apakah input data terdapat dalam satu bidang atau beberapa bidang.

Bidang yang akan digunakan oleh penentu lokasi untuk menentukan lokasi data. Bidang tertentu diasumsikan ada dalam data, seperti alamat, kota, negara bagian, dan kode pos.

Negara tempat data berada. Jika semua alamat berasal dari satu negara, berikan negara tersebut untuk memastikan hasil yang paling akurat. Jika tidak, kosongkan parameter ini.

Menentukan bidang yang akan disertakan dalam kumpulan data output. Opsinya sebagai berikut:

  • Semua—Semua kolom output penentu lokasi yang tersedia akan disertakan dalam hasil geocode.
  • Minimal—Subkumpulan bidang dari pencari lokasi serta bidang dari tabel input akan disertakan dalam hasil.
  • Hanya lokasi—Bidang Shape dari penentu lokasi serta bidang dari tabel input akan disertakan dalam hasil.

Jenis output yang akan dibuat. Outputnya dapat berupa fitur layanan, file .csv, atau file .xls.
