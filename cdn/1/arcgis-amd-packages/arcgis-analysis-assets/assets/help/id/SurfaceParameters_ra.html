Alat Parameter Permukaan menghitung parameter permukaan raster seperti aspek, kemiringan, dan lengkungan menggunakan metode geodesik.

Raster evelasi yang akan digunakan untuk penghitungan.

Jika unit z tersedia dalam sistem koordinat vertikal raster input, unit tersebut akan diterapkan secara otomatis.

Jenis parameter permukaan yang akan diukur.

Unit pengukuran raster kemiringan output.

Menentukan apakah azimut geodesik akan diproyeksikan.

Jika dicentang, azimut geodesik akan diproyeksikan untuk membenarkan distorsi angle yang disebabkan oleh referensi spasial output. Jika tidak dicentang, azimut geodesik tidak akan diproyeksikan.

Menentukan apakah aspek akan diukur dari titik di khatulistiwa atau dari kutub utara.

Jika dicentang, aspek akan diukur dari titik di khatulistiwa. Jika tidak dicentang, aspek akan diukur dari kutub utara.

Jenis fungsi permukaan yang akan disesuaikan di sekitar sel sasaran.

Jarak dari pusat sel sasaran dari mana output akan dihitung. Ini menentukan ukuran lingkungan yang digunakan untuk perhitungan.

Nilai defaultnya adalah ukuran sel raster yang ditentukan di parameter Raster permukaan input. Jika nilai ditentukan, maka nilai harus lebih besar dari default.

Menentukan apakah jarak lingkungan akan berbeda-beda dengan perubahan lanskap.

Jika dicentang, jarak lingkungan akan berbeda-beda dengan perubahan lanskap. Jika tidak dicentang, jarak lingkungan tetap akan digunakan di semua lokasi.
