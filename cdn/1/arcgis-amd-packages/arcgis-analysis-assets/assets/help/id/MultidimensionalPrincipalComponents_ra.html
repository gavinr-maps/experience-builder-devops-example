Alat Komponen Utama Multidimensi mengubah layer citra multidimensi menjadi sejumlah komponen yang dikurangi yang memperhitungkan perbedaan data, sehingga pola spasial dan temporal dapat dengan mudah diidentifikasi.

Layer citra multidimensi akan dianalisis.

Metode yang akan digunakan untuk melakukan analisis komponen utama. Pilihannya mencakup pengurangan Dimensi dan pengurangan Spasial.

Dimensi layer citra multidimensi yang variabelnya akan dianalisis.

Nama layer citra output atau layer tabel yang menunjukkan komponen utama yang ditemukan bergantung pada pilihan mode berikut:

  • Output pengurangan dimensi akan menjadi layer citra multiband.
  • Output pengurangan spasial akan menjadi layer tabel.

Nama layer citra output atau layer tabel yang memperlihatkan data pendukung dalam pilihan komponen utama tergantung pada pilihan mode berikut:

  • Output pengurangan dimensi akan menjadi layer tabel.
  • Output pengurangan spasial akan berupa layer citra.
