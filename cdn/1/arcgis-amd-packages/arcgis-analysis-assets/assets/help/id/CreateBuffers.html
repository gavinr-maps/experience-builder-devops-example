Alat Buat Buffer membuat poligon yang mencakup jarak yang diberikan dari fitur input.

Fitur titik, garis, atau poligon akan dijadikan buffer.

Menentukan apakah jarak buffer akan ditentukan oleh nilai tetap (Nilai) atau bidang numerik (Bidang).

Jarak yang akan digunakan untuk membuat buffer. Jika ada banyak nilai ditambahkan, tiap nilai akan digunakan untuk membuat buffer pada setiap fitur input.

Bidang yang akan berisi jarak buffer untuk tiap fitur.

Unit yang akan digunakan untuk menentukan jarak buffer.

Menentukan penanganan saat buffer dari fitur input yang berbeda saling tumpang tindih.

Menentukan apakah bufer dengan banyak jarak akan berupa buffer konsentris yang tumpang tindih (Cakram tumpang tindih) atau cincin yang tidak tumpang tindih (Cincin berbeda).

Untuk fitur garis, menentukan apakah buffer akan dibuat pada kedua sisi garis (Sekitar), hanya pada sisi kiri (Kiri), atau hanya pada sisi kanan (Kanan).

Untuk fitur poligon, menentukan apakah fitur poligon input akan disertakan dalam buffer (Sertakan) atau dikeluarkan dari buffer (Keluarkan).

Menentukan apakah bentuk buffer di akhir fitur garis akan berupa bulat (Round) atau datar (Datar).
