การตั้งค่าสภาพแวดล้อมสำหรับการวิเคราะห์ใน Map Viewer

ระบุระบบพิกัดสำหรับการวิเคราะห์ และชั้นข้อมูลผลลัพธ์

ระบุพื้นที่ซึ่งจะใช้สำหรับการวิเคราะห์

ปรับขอบเขตของเอาท์พุท เพื่อให้ตรงกับการจัดแนวเซลล์ของชั้นข้อมูลสแน็ปราสเตอร์ที่กำหนด

ระบุขนาดของเซลล์หรือความละเอียดที่จะนำมาใช้เพื่อสร้างชั้นข้อมูลแรสเตอร์เอาท์พุท

ระบุชั้นข้อมูลมมาสก์ ซึ่งมีเฉพาะเซลล์ที่อยู่ภายในพื้นที่มาสก์เท่านั้น ที่จะใช้สำหรับการวิเคราะห์

ระบุว่าจะประมาณค่าพิกเซลในช่วงอย่างไร เมื่อแรสเตอร์อินพุทและเอาท์พุทไม่อยู่ในแนวเดียวกัน

ระบุประเภทของโปรเซสเซอร์ที่จะใช้ ตัวเลือกได้แก่ CPU, GPU และ อัตโนมัติ หากระบุ อัตโนมัติ ไว้ เครื่องมือจะเป็นตัวกำหนดประเภทโปรเซสเซอร์

ระบุจำนวนของอินสแตนซ์บริการประมวลผลแรสเตอร์ที่ใช้ในระหว่างการรันเครื่องมือ ค่าที่รองรับได้แก่ สตริงเปล่า (เครื่องมือจะกำหนดจำนวนของอินสแตนซ์บริการประมวลผลแรสเตอร์ที่จะใช้) จำนวนเต็ม (0 หรือจำนวนเต็มบวก) และเปอร์เซ็นต์ เปอร์เซ็นต์จะต้องมีเครื่องหมายเปอร์เซ็นต์ (%) ตามหลังค่าเปอร์เซ็นต์

กำหนดว่าข้อความเตือนจะปรากฎขึ้นหรือไม่ หากจำนวนเครดิตโดยประมาณเกินค่าเกณฑ์ที่กำหนดไว้ (เมื่อเปิดใช้งาน) หรือจะรันการวิเคราะห์โดยไม่แสดงข้อความ และไม่สนใจการใช้เครดิตโดยประมาณ (เมื่อไม่เปิดใช้งาน) การวิเคราะห์จะยังคงทำงานตามการตั้งค่างบประมาณเครดิตสำหรับองค์กร

ระบุค่าเกณฑ์ที่จะแสดงข้อความคำเตือน อิงตามการใช้เครดิตโดยประมาณจากการรันการวิเคราะห์ จะยังคงสามารถรันการวิเคราะห์ได้ แม้การประมาณการจะเกินค่าเกณฑ์
