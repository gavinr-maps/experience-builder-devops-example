The Find by Attributes and Location tool creates a new layer of the features or parts of features that meet specified spatial or attribute criteria.

The point, line, or polygon layer for which features, based on a combination of attribute and spatial queries, will be found.

The spatial or attribute criteria that will be used to find features. Click the Build new query button to define one or more spatial or attribute criteria. Those features, or portions of features, that satisfy all criteria will be written to the result layer.

When only a portion of a feature meets the criteria, specifies whether the result includes the entire feature or only the portion of the feature that meets all criteria. When checked, only the portions of the feature that meet all criteria will be returned. When unchecked, the entire feature will be returned provided at least a portion of the feature meets all criteria.

The name of the layer that will be created and added to the map.
