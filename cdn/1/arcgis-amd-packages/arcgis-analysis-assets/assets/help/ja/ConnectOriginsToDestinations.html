移動コストを計算する終点位置を表すポイント レイヤー。

事前に割り当てられた起点と終点のペアを定義する [始点レイヤー] のフィールド。 起点位置と終点位置が複数ある場合、この ID は [終点レイヤー ルート ID フィールド] の値と一致する必要があります。

あらかじめ割り当てられた起点と終点のペアを定義する [終点レイヤー] のフィールド。 起点位置と終点位置が複数ある場合、この ID は [始点レイヤー ルート ID フィールド] の値と一致する必要があります。
