The Merge Layers tool copies features from two layers into a new layer. You can control how the fields from the input layers are joined and copied.

The point, line, or polygon features that will be merged with the merge layer.

The layer that will be merged with the input layer.

Specifies how fields will be merged.

If both layers contain a field with an identical name and data type, the content of both fields will be merged into one field of that name. This does not apply to fields when Match is used as the Merge operation parameter value.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
