The Create Buffers tool creates polygons that cover a given distance from input features.

The point, line, or polygon features that will be buffered.

Specifies whether the buffer distance will be defined by a constant (Value) or a numeric field (Field).

The distance that will be used to create the buffer. If multiple values are added, each value will be used to create a buffer at every input feature.

The field that will contain the buffer distance for each feature.

The units that will be used to specify the buffer distance.

Specifies how buffers from different input features will be handled when they overlap.

Specifies whether multiple-distance buffers will be concentric buffers that overlap (Overlaping disks) or rings that do not overlap (Distinct rings).

For line features, specifies whether the buffer will be created on both sides of the line (Around), on the left side only (Left), or on the right side only (Right).

For polygon features, specifies whether the input polygon features will be included in the buffer (Include) or excluded from the buffer (Exclude).

Specifies whether the shape of the buffer at the end of a line feature will be round (Round) or flat (Flat).

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
