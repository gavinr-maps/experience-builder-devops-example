Инструмент Найти горячие точки определяет наличие статистически значимой кластеризации входных объектов.

Входной точечный или полигональный слой, для которого будет выполняться анализ горячих точек.

Определяет, будут ли горячие и холодные точки отражать местоположения объектов (Число точек) или значения в поле объектов (Поле).

Поле, которое будет анализироваться на предмет кластеров высоких и низких значений.

Тип граничных объектов, в пределах которых будут агрегироваться входные точечные объекты. Это может быть полигональный слой (Полигональный слой) или мозаичная сетка, состоящая из квадратов (Прямоугольные ячейки) либо из шестиугольников (Шестиугольные ячейки).

Слой полигональных объектов, в пределах которых будут агрегированы входные точечные объекты.

Полигональный слой, задающий границу анализа. В результаты анализа горячих точек будут включены точечные объекты внутри полигона.

Определяет, как будут делиться значения в Поле анализа или количество агрегированных точек. Опция Поле позволяет делить значения в Поле анализа по значениям в указанном в параметре Входной слой поле. Опция Обогащение данных обогатит объекты или геометрию агрегирования данными о населении и разделит значения Поля анализа или агрегированное количество точек на численность населения. С этой опцией могут потребоваться дополнительные кредиты.

Длина стороны квадратных или шестиугольных ячеек, которые будут использоваться для агрегирования входных объектов.

Указывает единицы, которые будут использоваться для размера ячейки.

Диапазон расстояний, который будет использоваться для задания окрестности. Соседние объекты, находящиеся в пределах диапазона расстояний целевого объекта, будут включены в окрестность целевого объекта во время анализа горячих точек.

Указывает единицы измерения, которые будут использоваться для диапазона расстояний.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
