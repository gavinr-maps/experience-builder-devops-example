Инструмент Интерполировать точки вычисляет наиболее вероятные значения в новых местоположениях на базе измерений точек. Инструмент создает растровые данные.

Точечный слой, содержащий точки, в которых будут измеряться значения.

Числовое поле, содержащее значения для интерполяции.

Определяет, что будет оптимизировано: скорость или точность. Для более точных прогнозов требуется больше времени.

Этот параметр изменяет значения по умолчанию для нескольких других параметров.

Размер ячеек и единицы измерения для выходных растров.

Определяет, будут ли преобразованы данные. Интерполяция будет наиболее точной для данных с нормальным (в форме колокола) графиком распределения значений.

Размер локальных моделей интерполяции, которые будут скомбинированы для создания итоговой прогнозной карты.

Используемое число соседей. Чем больше соседей - тем более точные результаты могут быть получены и тем дольше будет выполняться расчет.

Имя интерполируемого растрового слоя, который будет создан и добавлен на карту.

Определите, хотите ли вы для прогнозных значений создать дополнительный растр стандартных ошибок.

Если отмечено, то у выходного растра стандартной ошибки будет то же имя, что и у Имени выходного растра, но к нему также будет добавлено слово "Ошибки". Слой будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
