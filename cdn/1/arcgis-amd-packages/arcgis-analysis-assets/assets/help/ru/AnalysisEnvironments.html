Настройки среды анализа в Map Viewer.

Задает систему координат для анализа и итогового слоя.

Указывает область, которая будет использоваться для анализа.

Настраивает экстент выходных данных таким образом, чтобы он соответствовал выравниванию ячеек указанного растрового слоя привязки.

Задает размер ячейки или разрешение, использующееся для анализа и при создании выходного растрового слоя.

Указывает слой маски, в котором для анализа будут использоваться только ячейки, попадающие в область маски.

Определяет, как будут интерполироваться значения пикселов в случае, когда входной и выходной растр не выровнены.

Определяет тип процессора, который будет использоваться. Опции: ЦПУ, ГПУ и Автоматически. Если указано Автоматически, инструмент сам определит тип процессора.

Указывает количество экземпляров сервиса обработки растра, используемых во время запуска инструмента. Поддерживаемые значения включают пустые строки (инструмент определит количество используемых экземпляров сервиса обработки растра), целые числа (0 или положительные целые числа) и проценты. Проценты должны включать знак процента (%) после процентного значения.

Определяет, показывать ли предупреждающее сообщение, если расчетное количество кредитов превышает заданный порог (включено), или анализ будет выполняться без отображения сообщения независимо от расчетного числа кредитов (не включено). При анализе все равно будут учтены настройки расхода кредитов, установленные для организации.

Задает пороговое значение, при достижении которого будет отображаться предупреждающее сообщение, основанное на предполагаемом расходе кредитов для выполнения анализа. Даже если число кредитов превышает порог, анализ все равно может быть выполнен.
