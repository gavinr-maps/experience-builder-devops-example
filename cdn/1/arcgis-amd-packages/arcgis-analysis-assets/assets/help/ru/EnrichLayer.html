Инструмент Обогатить слой улучшает входные объекты, добавляя им демографические и ландшафтные данные.

Точечные, линейные или полигональные объекты, которые будут обогащаться дополнительными данными.

Переменные, которые будут добавлены как атрибуты ко входным объектам.

Определяет, как будет задаваться область вокруг входных точечных или линейных объектов. Будет обогащена вся область вокруг каждого объекта.

Числовое значение, задающее расстояние или время, на которое будет распространяться область вокруг каждого входного объекта.

Единицы, которые будут использоваться для задания расстояния или времени, на которое обогащенная область будет простираться вокруг каждого входного объекта.

Указывает, будет ли выходной слой содержать полигональные объекты, заданные настройками Области обогащения (отметка стоит), или входными объектами (отметка не стоит).

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
