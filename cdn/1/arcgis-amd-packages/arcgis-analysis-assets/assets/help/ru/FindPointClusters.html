Инструмент Найти кластеры точек определяет среди точечных объектов кластеры в окружающем шуме на основе их пространственного распределения.

Входной точечный слой, объекты которого будут сгруппированы в кластеры на основе их пространственного распределения.

Метод, который будет использоваться для идентификации кластеров.

Минимальное число точечных объектов, которые будут считаться кластером.

Задает максимальное расстояние вокруг каждого точечного объекта для поиска точек, которые могут быть включены в кластер. Если оставить поле пустым, значением Расстояния поиска будет максимальное расстояние между ядрами, найденное в слое, за исключением расстояний, попавших в первый процент максимальных расстояний между ядрами.

Указывает единицы измерения, которые будут использоваться в качестве расстояния поиска.

Целочисленное значение от 0 до 100, которое определяет компактность кластеров. Если поле оставлено пустым, значение Чувствительности кластера будет вычислено посредством Дивергенции Кулбака-Лейблера.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
