Многомерные главные компоненты — это инструмент анализа растров, который преобразует многомерные слои изображений в меньшее число компонентов, учитывающих дисперсию данных, чтобы было легче идентифицировать пространственные и временные закономерности.

Слой многомерного изображения, который будет проанализирован.

Это метод, который будет использоваться для выполнения анализа главных компонент. Опции включают Уменьшение измерений и Пространственное уменьшение.

Измерение слоя многомерных изображений, по которому будет анализирована переменная.

Имя выходного слоя изображений или слоя таблицы, показывающего выявленные главные компоненты, зависит от следующих настроек режима:

  • Для режима Уменьшение измерений выходным слоем будет многоканальный слой изображений.
  • Для режима Пространственное уменьшение выходным слоем слой таблицы.

Имя выходного слоя изображений или слоя таблицы, поддерживаемые данные в главных компонентах, зависит от следующих настроек режима:

  • Для режима Уменьшение измерений выходным слоем слой таблицы.
  • Для режима Пространственное уменьшение выходным слоем слой изображений.

Имя таблицы собственных значений, в которой указывается процент отклонения каждого компонента.

Имя переменной входного многомерного слоя изображений, используемой для вычисления.

Количество главных компонент для вычисления, обычно меньшее, чем количество входных растров.

Имя слоя, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
