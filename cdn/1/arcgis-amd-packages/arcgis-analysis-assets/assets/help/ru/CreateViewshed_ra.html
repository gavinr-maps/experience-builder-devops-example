Инструмент Геодезическая видимость находит местоположения на поверхности растра, которые видимы для входных местоположений наблюдателей.

Растр высот, на основе которого будет вычислена обоюдная видимость.

Местоположения наблюдателя, для которых будет вычислена обоюдная видимость.

Определяет оптимизацию вычисления обоюдной видимости на основе скорости, либо точности.

Задает максимальное видимое расстояние для вычисления области видимости.

Это расстояние может быть задано значением и единицами измерения, либо из поля в параметре Входные точечные или линейные объекты наблюдателей.

Задает минимальное видимое расстояние для вычисления области видимости.

Это расстояние может быть задано значением и единицами измерения, либо из поля в параметре Входные точечные или линейные объекты наблюдателей.

Определяет тип максимального и минимального расстояния видимости.

Если не отмечено – расстояния видимости будут рассматриваться как двумерные евклидовы расстояния. Если отмечено – расстояния видимости будут рассматриваться как трехмерные расстояния.

Определяет высоты поверхности для местоположения точек наблюдателя.

Эта высота может быть задана значением и единицами измерения, либо из поля в параметре Входные точечные или линейные объекты наблюдателей.

Определяет высоту наблюдателей над землей.

Эта высота может быть задана значением и единицами измерения, либо из поля в параметре Входные точечные или линейные объекты наблюдателей.

Определяет высоту целей на земле.

Эта высота может быть задана значением и единицами измерения, либо из поля в параметре Входные точечные или линейные объекты наблюдателей.

Имя растрового слоя видимости, который будет создан и добавлен на карту. Этот слой содержит видимые области, которые смогут увидеть наблюдатели, а значения ячеек представляют число наблюдателей, которые видят эту ячейку.

Имя растрового слоя уровней над поверхностью земли, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
