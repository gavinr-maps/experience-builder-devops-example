Инструмент Суммировать в пределах вычисляет статистику для объектов в пределах указанных ячеек или полигонов.

Слой суммирования.

Тип граничных объектов, в пределах которых будут суммироваться входные объекты. Это может быть полигональный слой или мозаичная сетка квадратных или гексагональных бинов, создаваемых при работе инструмента.

Полигональный слой, в пределах которых будут суммироваться входные объекты.

Размер создаваемых бинов. Для квадратных бинов указанные число и единицы измерения определяют размеры квадрата. Для шестиугольных бинов число и единицы измерения определяют расстояние между параллельными сторонами.

Единицы измерения, которые будут использоваться для задания размера бина.

Вычислить итоговый размер объектов, суммированных в каждом ограничивающем полигоне. Для линий вычисляется итоговая длина. Для полигонов вычисляется итоговая площадь.

Единицы измерения, которые будут использоваться для вычисления итогового размера суммируемых объектов.

Числовые поля или поля дат из входного слоя для суммирования, а также статистика, которая должна быть вычислена для каждого поля.

Поле из входного слоя, которое будет использоваться для разделения данных на отдельные группы для вычисления статистики.

Рассчитать группы меньшинства (наименее доминирующего) и большинства (наиболее доминирующего) в каждой области суммирования.

Вычислить процент просуммированных объектов, которые принадлежат каждой группе в пределах каждой области суммирования.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
