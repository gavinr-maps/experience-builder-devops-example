Инструмент Суммированный центр и дисперсия находит центральный объект, центральную тенденцию и эллипс стандартного отклонения набора объектов.

Точечные, линейные или полигональные объекты, для которых будут вычислены центральный объект, центральная тенденция и эллипс стандартного отклонения.

Задает методы, которые будут суммировать центр и дисперсию объектов.

Задает размер полигонального объекта-эллипса в стандартных отклонениях. Если расстояние между усредненным центром и местоположениями объектов подчиняется распределению Рэлея, параметры 1 стандартное отклонение, 2 стандартных отклонения и 3 стандартных отклонения выведут полигональный объект-эллипс, который покроет 63%, 89% и 99,9% объектов, соответственно.

Поле, которое будет использовано для определения веса объекта.

Поле, которое будет использовано для группировки объектов. Выбранные Типы суммирования будут вычислены для каждой группы.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
