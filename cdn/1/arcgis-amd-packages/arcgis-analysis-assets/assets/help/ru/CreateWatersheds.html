Инструмент Создать водоразделы определяет область стока, находящуюся вверх по течению, для указанных точек.

Местоположения, в которых будет создан водосбор вверх по течению.

Максимально допустимое расстояние для перемещения местоположения входной точки к ближайшей линии стока или самому высокому суммарному стоку.

Единицы, которые будут использоваться для параметра Расстояние поиска.

Имя векторного слоя, содержащего полигоны водосборной области, и скорректированные точки устьев, которые будут созданы и добавлены на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
