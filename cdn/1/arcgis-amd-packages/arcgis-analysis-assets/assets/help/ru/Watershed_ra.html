Инструмент Водосборная область определяет водосборную область, расположенную выше набора ячеек растра.

Направление стока для каждой ячейки.

Для определения направления стока необходимо использовать метод D8.

Местоположения, для которых будет определяться расположенная выше них водосборная область.

Поле, используемое для присвоения значений местоположениям точек устьев.

Имя растрового слоя водосборной области, который будет создан и добавлен на карту. Слой содержит разные водосборные области, представленные разными значениями ячеек из поля Pour point.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
