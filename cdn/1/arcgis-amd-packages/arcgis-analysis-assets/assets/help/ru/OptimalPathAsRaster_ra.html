Инструмент оптимальный путь как растр определяет оптимальный путь от назначений к источникам в формате растра.

Слой, задающий назначение, от которого рассчитывается расстояние до источника с наименьшей затратами стоимостью.

Если входными данными является растр, он должен состоять из ячеек, имеющих допустимые целые значения для назначений (ноль - допустимое значение), остальным ячейкам должно быть присвоено NoData.

Поле, которое будет использоваться для получения значений для местоположений назначения.

Растр, определяющий оптимальный путь от назначений к источникам.

Растр накопления расстояния обычно создается с помощью инструментов Накопление расстояния или Распределение по расстоянию.

Растр, который содержит рассчитанные направления в градусах.

Растр обратного направления обычно создается с помощью инструментов Накопление расстояния или Распределение по расстоянию.

Определяет способ интерпретации значений и зон во входных данных объектов назначения при расчетах оптимального пути.

Имя слоя оптимального пути как растра, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
