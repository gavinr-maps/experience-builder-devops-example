Инструмент Суммировать ближайшие вычисляет статистику для объектов, находящихся на заданном расстоянии от других объектов. Расстояние можно вычислить, используя расстояние по прямой или с учетом дорожной сети.

Слой, в котором нужно вычислить суммирование в пределах расстояния от слоя ближайших объектов.

Точечный, линейный или полигональный слой, который используется в качестве стартовых местоположений, вокруг которых будут созданы ограничивающие области для суммирования ближайших объектов.

Метод, на основании которого будут построены ограничивающие области. Могут быть выбраны следующие методы: расстояние по прямой или режим передвижения.

Расстояние или время, использующиеся для создания граничных областей. Если будет добавлено несколько значений, то каждое из них будет использоваться, чтобы вычислить ограничивающие области для всех входных объектов.

Единицы измерения, которые будут использоваться для задания расстояний или времени.

Укажите, должен ли результирующий слой быть ограничивающими областями (отмечено) или такой же геометрией, что и входной слой ближайших объектов (не отмечено).

Вычислить итоговый размер объектов, суммированных в каждой ограничивающей области. Для линий вычисляется итоговая длина. Для полигонов вычисляется итоговая площадь.

Единицы измерения, которые будут использоваться для вычисления итогового размера суммируемых объектов.

Числовые поля или поля дат из входного слоя для суммирования, а также статистика, которая должна быть вычислена для каждого поля.

Поле из входного слоя, которое будет использоваться для разделения данных на отдельные группы для вычисления статистики.

Рассчитать группы меньшинства (наименее доминирующего) и большинства (наиболее доминирующего) в каждой области суммирования.

Вычислить процент просуммированных объектов, которые принадлежат каждой группе в пределах каждой границы.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
