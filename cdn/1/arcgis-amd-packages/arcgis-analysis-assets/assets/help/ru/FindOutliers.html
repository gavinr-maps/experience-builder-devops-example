Точечный или полигональный слой, для которого будут вычислены выбросы.

Задает, будут ли выбросы определяться местоположениями объектов (Число точек) или значениями в поле объектов (Поле).

Поле, которое будет анализироваться с целью нахождения выбросов.

Тип граничных объектов, в пределах которых будут агрегироваться входные объекты. Это может быть полигональный слой (Полигональный слой) или мозаичная сетка, состоящая из квадратов (Прямоугольные ячейки) либо из шестиугольников (Шестиугольные ячейки).

Слой полигональных объектов, в пределах которых будут агрегированы входные точечные объекты.

Полигональный слой, задающий границу анализа. Объекты внутри полигона будут включены в анализ выбросов.

Определяет, как будут делиться значения в Поле анализа или количество агрегированных точек. Опция Поле позволяет делить значения в Поле анализа по значениям в указанном в параметре Входной слой поле. Опция Обогащение данных обогатит объекты или геометрию агрегирования данными о населении и разделит значения Поля анализа или агрегированное количество точек на численность населения. С этой опцией могут потребоваться дополнительные кредиты.

Определяет, следует ли оптимизировать работу инструмента с целью сокращения времени обработки или повышения точности результатов.

Целочисленное значение, которое инициирует генератор случайных чисел.

Длина стороны квадратных (сетка) или шестиугольных ячеек, которые будут использоваться для агрегирования входных объектов.

Указывает единицы, которые будут использоваться для размера ячейки.

Диапазон расстояний, который будет использоваться для задания окрестности. Соседние объекты, находящиеся в пределах диапазона расстояний целевого объекта, будут включены в окрестность целевого объекта во время анализа.

Указывает единицы измерения, которые будут использоваться для диапазона расстояний.

Имя слоя, который будет создан и добавлен на карту.
