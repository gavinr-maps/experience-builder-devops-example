Инструмент Прогнозировать, используя растр тренда, вычисляет прогнозируемый многомерный слой изображения с помощью выходного слоя тренда изображений, который был создан инструментом Создать растр тренда.

Входной многомерный слой изображения тренда, который будет использоваться для прогнозирования.

Переменная многомерного слоя изображения тренда, который будет использоваться для прогнозирования.

Метод, который будет использоваться для указания значений прогнозируемых измерений.

Установите дату начала и окончания, высоту или глубину интервала измерения, который будет использоваться в прогнозе.

Дата начала, высота или глубина интервала измерения, который будет использоваться в прогнозе.

Дата окончания, высота или глубина интервала измерения, который будет использоваться в прогнозе.

Количество шагов между двумя значениями измерений, которые должны быть включены в прогноз.

Единицы измерения, которые будут применены к значению интервала.

Имя слоя, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
