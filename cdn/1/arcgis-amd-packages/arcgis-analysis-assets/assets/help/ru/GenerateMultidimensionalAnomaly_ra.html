Создать многомерную аномалию вычисляет аномалию для каждого среза в существующем слое многомерного изображения для создания нового слоя многомерного изображения.

Слой многомерного изображения, который будет проанализирован.

Переменная слоя многомерного изображения, которая будет использоваться для выявления аномалий.

Метод, который будет использоваться для вычисления аномалий в слое изображений.

Временной интервал, который будет использоваться для вычисления среднего значения.

Базовый слой изображений, который содержит вычисленные значения для каждого пиксела.

Определяет, будут ли игнорироваться значения NoData или пропущенные значения в анализе.

Имя слоя, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
