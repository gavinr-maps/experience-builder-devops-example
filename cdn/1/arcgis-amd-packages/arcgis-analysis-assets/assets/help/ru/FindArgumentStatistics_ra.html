Инструмент Найти статистику аргументов извлекает значение измерения или индекс канала, при котором достигается заданная статистика для каждого пиксела в слое многомерного изображения.

Слой многомерного изображения, который будет оцениваться.

Измерение слоя многомерного изображения, по которому будет оцениваться переменная.

Переменная слоя многомерного изображения, которая будет оцениваться по заданному интервалу.

Тип статистики для извлечения из переменной или переменных по заданному измерению.

Минимальное значение, которое должно использоваться для получения продолжительности аргумента.

Максимальное значение, которое должно использоваться для получения продолжительности аргумента.

Значение, указывающее, что данное значение было достигнуто статистикой аргумента несколько раз. По умолчанию это значение, которое было достигнуто статистикой аргумента в первый раз.

Указывает, будет ли значение измерения возвращено при достижении статистики аргумента в первый или в последний раз. По умолчанию - в первый раз.

Будет получено значение величины, при котором достигается указанное значение переменной.

Указывает тип сравнения, который будет использован для извлечения значения измерения.

Определяет, как будет извлекаться статистика из заданного измерения.

Единица времени, для которой будет извлекаться статистика.

Определяет, будут ли значения NoData игнорироваться при извлечении.

Имя слоя, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
