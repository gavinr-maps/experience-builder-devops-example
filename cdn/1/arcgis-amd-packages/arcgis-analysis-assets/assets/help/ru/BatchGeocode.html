Инструмент Геокодировать местоположения из таблицы конвертирует адреса или местоположения из входной таблицы в координаты.

Входная таблица с адресами или местоположениями для геокодирования. Входной файл должен быть таблицей, файлом .csv или файлом Excel.

Сервис локатора, используемый для геокодирования. Если сервис локатора, который вы хотите использовать, не находится в ниспадающем меню, свяжитесь с администратором, чтобы сервис локатора настроили для пакетного геокодирования.

Указывает, содержатся ли входные данные в одном поле или в нескольких полях.

Поля, которые будут использоваться локатором для определения местоположения данных. Предполагается, что в данных присутствуют определенные поля, такие как адрес, город, штат и почтовый индекс.

Страна, в которой находятся данные. Если все адреса находятся в одной стране, выберите эту страну, чтобы обеспечить более точные результаты. В противном случае, оставьте этот параметр пустым.

Определяет поля, которые будут включены в выходной набор данных. Можно выбрать следующие опции:

  • Все – все доступные выходные поля локатора будут включены в результаты геокодирования.
  • Минимальное — в результаты будет включен поднабор полей из локатора, а также поля из входной таблицы.
  • Только местоположение — в результаты будет включено поле Shape из локатора, а также поля из входной таблицы.

Тип создаваемых выходных данных. Выходными данными может быть сервис объектов, файл .csv или файл .xls.

Имя выходного элемента, в котором будут сохранены данные местоположений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
