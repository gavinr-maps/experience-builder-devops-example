Инструмент Выявить объекты при помощи глубокого обучения запускает модель глубокого обучения для входного слоя изображений с целью получения векторного слоя с найденными объектами.

Слой изображений или векторный слой, который будет использоваться для обнаружения объектов.

Определяет, как растровые элементы слоя изображений будут обрабатываться после создания мозаики: вместе или по отдельности.

Пакет глубокого обучения, содержащий модель, которая будет использоваться для обнаружения объектов.

Аргументы модели, содержащиеся в модели глубокого обучения вместе со значениями по умолчанию.

Указывает, будет ли выполняться немаксимальное подавление для удаления повторяющихся объектов.

Имя поля, которое будет добавлено к выходному векторному слою для указания переданного значения достоверности.

Имя поля, которое будет добавлено к выходному векторному слою для указания классифицированной метки для объекта.

Максимальное соотношение, допустимое для каждого найденного объекта, перекрывающего другой объект. По умолчанию значение равно 0.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
