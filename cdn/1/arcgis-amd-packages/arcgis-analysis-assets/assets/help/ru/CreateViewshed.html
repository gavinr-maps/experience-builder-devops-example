Инструмент Создать область видимости использует сервис Esri Elevation Analysis (Анализ высот) для определения областей, которые будут видны из указанных местоположений наблюдателей.

Точечные объекты, соответствующие местоположениям наблюдателей.

Определяет высоту наблюдателей над землей.

Единицы измерения, которые будут использоваться для параметра Высота наблюдателя.

Определяет высоту целей на земле.

Единицы измерения, которые будут использоваться для параметра Высота цели.

Задает максимальное видимое расстояние для вычисления области видимости.

Единицы измерения для параметра Максимальное расстояние видимости.

Имя полигонального слоя видимости, который будет создан и добавлен на карту. Этот слой содержит видимые области, которые смогут увидеть наблюдатели, а количество наблюдателей записывается в поле частоты.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
