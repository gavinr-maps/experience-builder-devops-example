Инструмент Наложение слоев комбинирует два слоя в один общий слой на основе их пространственного отношения.

Точечный, линейный или полигональный слой, для которого будет выполнено наложение.

Слой, который будет накладываться на входной слой.

Метод наложения, который будет использоваться для определения способа объединения входного слоя и слоя наложения.

Тип объектов (точки, линии, полигоны), создаваемых в выходном векторном слое.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
