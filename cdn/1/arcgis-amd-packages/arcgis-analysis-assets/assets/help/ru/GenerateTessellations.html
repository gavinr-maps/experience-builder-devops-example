Инструмент Создать замощение создает сетку полигонов на основе заданного экстента, формы и размера.

Форма создаваемого замощения.

Определяет, будет ли размер замощения определяться его площадью (Площадь) или по значению расстояния (Расстояние).

Размер (площадь или расстояние) создаваемых полигонов замощения. Для квадратов, шестиугольников, треугольников и ромбов расстояние определяет высоту элементов сетки. Для поперечных шестиугольников расстояние определяет ширину элементов сетки.

Разрешение H3 замощений. Разрешение находится в диапазоне от 0 до 15. С каждым увеличением размера разрешения площадь полигонов будет уменьшаться на одну седьмую размера при более низком разрешении.

Указывает единицы, которые будут использоваться для размера замощения.

Экстент, который покрывается замощением.

Для векторных слоев определяет, будет ли замощение покрывать экстент целиком, либо только те области, где замощение пересекается с объектами.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
