Инструмент Создать буферы создает полигоны, которые покрывают заданное расстояние от входных объектов.

Точечные, линейные или полигональные объекты, для которых будут строиться буферные полигоны.

Определяет, будет ли буферное расстояние задаваться константой (Значение) или числовым полем (Поле).

Расстояние, которое будет использоваться для создания буфера. Если будет добавлено несколько значений, каждое значение будет использоваться для создания буфера каждого входного объекта.

Поле, которое будет содержать буферное расстояние для каждого объекта.

Единицы измерения, которые будут использоваться для задания буферного расстояния.

Определяет, как будут обрабатываться буферы перекрывающихся входных объектов.

Определяет, будут ли буферы с несколькими расстояниями представлять собой концентрические перекрывающиеся буферы (Перекрывающиеся диски), или не перекрывающиеся кольца (Отдельные кольца).

Для линейных объектов определяет, будет ли буфер создаваться с обеих сторон линии (Вокруг), только с левой (Слева) или только с правой стороны от линейного объекта (Справа).

Для полигональных объектов определяет, будет ли входной полигональный объект включен в буфер (Включить) или исключен из него (Исключить).

Определяет, будет ли на конце линейного объекта буфер круглым (Круглый) или плоским (Плоский).

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
