Инструмент Создать область в пути создает области, до которых можно добраться за определенное время в пути или расстояние.

Точечные объекты, вокруг которых будут вычислены области.

Укажите один или более точечный объект, который действует как временное ограничение (иначе говоря, барьер) при перемещении по улицам. Передвижение разрешено по улице, но не через барьер.

Укажите один или более линейный объект, который действует как временное ограничение (иначе говоря, барьер) при перемещении по улицам. Линейные барьеры запрещают проезд в любом месте, где барьеры пересекаются с улицами.

Укажите один или более полигональный объект, который действует как временное ограничение (иначе говоря, барьер) при перемещении по улицам. Полигональные барьеры запрещают проезд в любом месте, где полигоны пересекаются с улицами.

Режим перемещения, который будет использоваться для создания областей пути. Режим передвижения может быть основан на расстоянии или времени.

Ограничения по расстоянию перемещения или ограничение по времени для каждой входной точки. Если будет добавлено несколько значений, то каждое из них будет использоваться, чтобы вычислить области в пути для всех входных точек.

Единицы измерения, которые будут использоваться для определения расстояния или времени в пути. Значение Предельных значений интерпретируется в этих единицах измерения.

Определяет, как создаются ли области в пути: при перемещении от входных точек или к входным точкам.

Указывает дату и время отправления или прибытия путешественника из/в входные местоположения.

Определяет, как обрабатываются общие области из разных входных объектов, когда они достигают друг друга.

Указывает, будет ли каждая выходная область представлять собой один непрерывный полигон (отключено) или содержать пробелы на улицах, которые недоступны, но полностью окружены доступными улицами (отмечено).

Имя слоя, который будет создан и добавлен на карту.

Определяет, будут ли улицы, которые используются для создания областей передвижения, добавлены в выходной слой.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
