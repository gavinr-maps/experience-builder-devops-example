Инструмент Выявить изменения при помощи глубокого обучения использует обученную модель глубокого обучения для выявления изменений между двумя растрами.

Входной слой изображений до изменений, который будет использован как основа для сравнения.

Входной слой изображений после изменения, который будет использован для сравнения с исходным.

Выходной классифицированный слой изображений, который отразит изменения.

Аргументы модели, содержащиеся в модели глубокого обучения вместе со значениями по умолчанию.

Пакет глубокого обучения, содержащий модель, которая будет использована для выявления изменений между слоями изображений.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
