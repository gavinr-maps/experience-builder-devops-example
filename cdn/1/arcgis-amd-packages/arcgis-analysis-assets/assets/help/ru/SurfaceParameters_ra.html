Инструмент Параметры поверхности вычисляет параметры растровой поверхности, такие как экспозиция, уклон и кривизна, с использованием геодезических методов.

Растр высот, который будет использоваться для расчета.

Если единицы измерения z-значения имеются вертикальной системе координат входного растра, то они будут применяться автоматически.

Тип параметра поверхности, который будет измерен.

Единицы измерения выходного растра уклонов.

Определяет, будут ли спроецированы геодезические азимуты.

Если отмечено, геодезические азимуты будут спроецированы с учетом корректного искажения углов, вызванного выходной пространственной привязкой. Если не отмечено, то геодезические азимуты не будут спроецированы.

Указывает, будет ли экспозиция измеряться от точки на экваторе или от северного полюса.

Если отмечено, то экспозиция будет измеряться из точки на экваторе. Если не отмечено, то экспозиция будет измеряться от северного полюса.

Тип функции поверхности, которая будет установлена вокруг целевой ячейки.

Расстояние от центра целевой ячейки, из которого будут вычисляться выходные данные. Оно определяет размер окрестности, которая используется для вычислений.

Значением по умолчанию будет размер ячейки растра, заданный в параметре Входной растр поверхности. Если задано значение, то оно должно быть больше значения по умолчанию.

Указывает, будет ли расстояние до окрестности меняться в зависимости от изменений ландшафта.

Если отмечено, то расстояние до окрестности будет меняться в зависимости от изменений ландшафта. Если не отмечено, то во всех местоположениях будет использоваться фиксированное расстояние окрестности.

Линейные единицы измерения вертикальных z-значений.

Имя растрового слоя поверхности, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
