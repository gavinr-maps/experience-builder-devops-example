Инструмент Конвертировать растр в объекты векторизует растры.

Входной растр, который будет конвертирован в пространственные объекты.

Поле со значениями, которые будут использованы для конвертации.

Тип объектов (точки, линии, полигоны), создаваемых в выходном слое.

Задает упрощение (сглаживание) линий или полигонов.

Определяет, будут ли выходные полигоны состоять из одной или из нескольких частей.

Составные объекты обычно обрабатываются быстрее. Но если на выходе будет создано очень много объектов, этот процесс может замедлиться.

Предел числа вершин, который будет использоваться для разделения большого полигона на меньшие.

При оставлении пустым, выходные полигоны делиться не будут.

Имя векторного слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
