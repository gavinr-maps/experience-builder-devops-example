Инструмент Найти схожие местоположения оценивает сходство между местоположениями-кандидатами и базовым местоположением на основе набора критериев и ранжирует предполагаемые местоположения от наиболее похожих к наименее похожим.

Точечный, линейный или полигональный слой, содержащий базовые местоположения для сопоставления. Если слой содержит несколько местоположений, все эти местоположения будут усреднены для получения одного базового местоположения.

Точечный, линейный или полигональный слой, содержащий местоположения-кандидаты для поиска. Каждое найденное местоположение-кандидат будет сравниваться с базовым местоположением по указанному значению Критерия выборки и ранжироваться от наиболее похожего до наименее похожего.

Числовые поля, которые будут использоваться для оценки сходства между базовым местоположением и каждым из местоположений-кандидатов. Каждое значение Критерия выборки будет содержать значение Поля базового слоя и значение Поля слоя кандидатов. Значения Поле базового слоя и Поле слоя кандидатов являются соответствующими полями в значениях Слой базовых местоположений и Слой поиска кандидатов.

Поле Слой базовых местоположений, соответствующее полю Слой поиска кандидатов, заданному значением Поле слоя кандидатов. Значения в соответствующих полях будут использоваться для оценки сходства базовых местоположений и каждого из местоположений-кандидатов.

Поле Слоя поиска кандидатов, соответствующее полю Слоя базовых местоположений, заданному значением Поле базового слоя. Значения в сопоставленных полях будут использоваться для оценки сходства между базовым местоположением и каждым из местоположений-кандидатов.

Определяет, будут ли все местоположения Слоя поиска кандидатов включены в выходной слой.

Число наиболее похожих местоположений Слоя поиска кандидатов, которые будут включены в выходной слой.

Имя слоя, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
