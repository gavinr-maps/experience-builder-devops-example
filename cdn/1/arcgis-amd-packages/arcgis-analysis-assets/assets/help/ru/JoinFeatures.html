Инструмент Присоединить объекты добавляет атрибуты из одного слоя или таблицы в другой на основании заданных пространственных и атрибутивных отношений. Дополнительно, когда есть несколько совпадающих записей, можно вычислить статистику.

Слой или таблица, который получит атрибуты из слоя, присоединяемого к его таблице атрибутов.

Слой или таблица с атрибутами, которые будут присоединяться к целевому слою.

Укажите, нужно ли присоединять слои на основе пространственных отношений. Необходимо, чтобы оба слоя содержали геометрию.

Укажите пространственное отношение, определяющее, какие объекты присоединяются. Объекты в слое соединения, у которых есть заданное отношение к целевому слою, будут присоединены. Доступные отношения зависят от типа геометрии входных данных.

Расстояние от объектов в целевом слое, которое будет ипользования для определения того, будут ли присоединены объекты.

Единицы измерения, которые будут использоваться для задания расстояний.

Укажите, нужно ли присоединять слои на основе сопоставления значений полей. Это единственная опция, если на вход подается одна или несколько таблиц.

Укажите атрибутивное отношение, определяющее, какие объекты присоединяются. Атрибуты объектов соединяются, если значения поля в присоединяемом слое идентичны значениям поля в целевом слое.

Укажите, как должны выполняться соединения, если несколько записей из присоединяемого слоя соответствуют критерию соединения. Выберите одну из следующих опций:

  • Присоединить один-к-одному - присоединить одну запись в слое соединения к каждому объекту в целевом слое.
  • Присоединить один-ко-многим – присоединить все совпадающие объекты в присоединяемом слое с объектами целевого слоя. Полученный в результате слой будет содержать несколько записей для целевого объекта.

Когда несколько записей в присоединяемой таблице соответствуют целевому объекту, укажите, следует ли присоединить одну запись из слоя соединения, либо вычислить суммарную статистику для всех совпадающих записей из слоя соединения.

Когда есть несколько соединяемых объектов, которые соответствуют целевому объекту, то для всех них будет вычислена суммарная статистика. Укажите число полей дат из слоя соединения, а также статистику, которая должна быть вычислена для каждого поля.

Когда есть несколько объектов соединения, которые соответствуют целевому объекту, то в выходной результат будет включена только первая запись из таблицы атрибутов присоединяемого слоя. Дополнительно выберите число или поле даты, а также порядок сортировки, который определит порядок записей в таблице атрибутов.

Определяет, будут ли несовпадающие целевые объекты добавлены в результат. Внутреннее соединение возвращает только целевые записи, где Соединение слева возвращает все целевые записи, даже если у целевого объекта нет никаких сопоставлений в слое соединения.

Имя слоя или таблицы, который будет создан и добавлен на карту.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
