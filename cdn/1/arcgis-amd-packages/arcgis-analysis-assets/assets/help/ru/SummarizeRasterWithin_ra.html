Инструмент Зональная статистика суммирует значения растра в пределах зон другого набора данных.

Входные данные, задающие границы зон или областей, которые будут суммироваться.

Поле, которое содержит значения, определяющие каждую зону.

Растровый слой, содержащий значения ячеек, которые будут использоваться для расчета статистики.

Определяет тип вычисляемой статистики.

Определяет вычисляемое значение процентиля.

Определяет метод интерполяции, используемый, когда количество значений входного растра, которое нужно вычислить, является четным.

Определяет, как значения ячеек Растра входных значений будут обрабатываться для круговых данных.

Если опция не отмечена, будет вычислена обычная арифметическая (линейная) статистика. Если опция не отмечена, будет вычислена круговая статистика.

Определяет значение, которое будет использоваться для округления линейного значения до диапазона данной круговой статистики.

Определяет, будут ли значения NoData Растра входных значений игнорироваться в результатах для зон, в которые они попадают.

Если отмечено, ячейки NoData в растре значений будут игнорироваться в вычислении статистики. Если не отмечено, ячейки NoData не будут игнорироваться, и вся зона получит значение NoData на выходном растре.

Определяет, как будут обрабатываться входные растры, если они являются многомерными.

Если параметр не отмечен, статистика будет вычисляться для всех измерений входного многомерного набора данных. Если параметр отмечен, статистика будет вычисляться на основе текущего среза входного многомерного набора данных.

Имя растрового слоя зональной статистики, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
