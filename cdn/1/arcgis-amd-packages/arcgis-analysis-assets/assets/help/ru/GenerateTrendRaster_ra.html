Инструмент Создать растр тренда оценивает тренд для каждого пиксела вдоль измерения для одной или нескольких переменных в слое многомерного изображения.

Слой многомерного изображения, который будет использоваться для создания тренда.

Измерение, по которому будет извлечен тренд для переменной или переменных, выбранных в анализе.

Переменная или переменные, для которых будут рассчитываться тренды.

Единицы измерения времени, которые будут использоваться для длины сезонного периода при использовании линейного типа Сезонного Кендалла.

Тип анализа тренда, который будет выполнен для значений пикселов в данном измерении.

Частота или номер порядка полинома для использования в подгонке тренда.

Определяет, будут ли игнорироваться значения NoData или пропущенные значения в анализе.

Длина периодической вариации, которая будет моделироваться для гармонического цикла.

Устанавливает единицу времени, которая будет использоваться для длины гармонического цикла.

Указывает, следует ли вычислить среднеквадратичную ошибку (RMSE) линии тренда.

Указывает, следует ли вычислять статистику пригодности R-квадрата для линии тренда.

Указывает, следует ли вычислять статистику p-значения для коэффициента уклона линии тренда.

Имя слоя, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
