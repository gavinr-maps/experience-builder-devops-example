Векторные данные, которые будут конвертированы. Входной слой может содержать точки, линии или полигоны.

Поле, которое будет использоваться для присвоения значений выходному растру.

Размер ячеек и единицы измерения для выходных растров.

Имя растрового слоя, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.
