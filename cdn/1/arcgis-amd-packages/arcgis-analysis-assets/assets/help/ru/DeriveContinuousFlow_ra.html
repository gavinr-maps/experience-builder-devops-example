Инструмент Получить непрерывный поток создает растр суммарного стока в каждой ячейке входного растра поверхности без предварительного заполнения впадин или локальных понижений.

Растр высот, который будет использоваться для расчета.

Определяет реальные локальные понижения или впадины, в которые вода может поступать, но не вытекать из них.

Растровый слой, который определяет долю стока, которая накапливается в каждой ячейке.

Определяет, какой метод будет использоваться при вычислении направления стока.

Метод D8 назначает направление стока до соседнего пиксела, расположенного вниз по склону с максимальным уклоном. Метод MFD распределяет направление стока по всем соседям ниже по склону в соответствии с показателем адаптивного разделения.

Определяет, будет ли поток в краевых ячейках подчиняться обычным (нормальным) правилам определения стока или же всегда вытекать наружу.

Если отметка не стоит, направление стока будет направлено к внутренней ячейке с самым крутым падением z-значения. Однако если снижение меньше или равно нулю, сток из ячейки будет «вытекать» с растра поверхности. Если опция отмечена, направление потока всегда будет направлено наружу от растра поверхности.

Имя растрового слоя суммарного стока, который будет создан и добавлен на карту.

Имя растрового слоя направления стока, который будет создан и добавлен на карту.

Тип создаваемых выходных данных. Результатом может быть Слой листов изображений или Динамический слой изображений.

Имя папки в разделе Мои ресурсы, в которую будет сохранен результат.
