「計劃路線」工具可針對車隊建立包含多個停靠點的行進路線。

指定車輛路線開頭的點圖徵。

車輛必須在其路線上造訪的點圖徵。 這些點在輸出中將變成已分配或未分配的停靠點。

決定車輛是否必須返回其起始位置 (啟用)，或在指定位置結束 (停用)。

指定車輛路線結尾的點圖徵。 如果圖層僅包含一點，則所有車輛都將路由到一個公共端點。 如果圖層包含多點，則每輛車都將路由到唯一的終點位置。

起始圖層中唯一識別各個路線、車輛或司機的欄位。 如果車輛起始於多個位置且在多個位置結束，則為起始和結束圖層所選的路線 ID 必須相符。

結束圖層中唯一識別各個路線、車輛或司機的欄位。 如果車輛起始於多個位置且在多個位置結束，則為起始和結束圖層所選的路線 ID 必須相符。

在基礎街道行進時要作為臨時限制 (屏障) 的一或多個點圖徵。 允許在街道行進，但無法通過屏障。

在基礎街道行進時要作為臨時限制 (屏障) 的一或多個線圖徵。 線屏障禁止在屏障與街道相交的任何地方行進。

在基礎街道行進時要作為臨時限制 (屏障) 的一或多個多邊形圖徵。 多邊形屏障禁止在多邊形與街道的任何相交處行進。

將用於建立路線的行進方法。 行進模式可根據距離或時間。

車輛出發的日期和時間。

可造訪停靠點的車輛數上限。 如果 起始圖層和 結束圖層各包含一個圖徵，則您最多可以指定 100 部車輛。 如果任一圖層包含多個圖徵，則無法編輯預設值。

每輛車在一條路線上可以停靠的停靠點數上限。 您可以指定一個介於 1 和 200 之間的值。

指定每個停靠點所需的預期平均服務時間。

決定是否將最大路線規劃時間套用於車輛。 路線規劃時間包括預期行進時間，及花費在該路線沿線停靠點的服務時間。

指定某路路徑開始到結束應持續的最長時間。

將建立並新增到地圖之圖層的名稱。

指定是否將每條結果路線儲存為路線圖層。
