The Summarize Within tool calculates statistics for features within specified bins or polygons.

The layer that will be summarized.

The type of bounding features within which the input features will be summarized. This can be a polygon layer or a tessellation grid of either square or hexagon bins that are generated when the tool runs.

The polygon layer within which the input features will be summarized.

The size of the generated bins. For square bins, the number and units specified determine the height and length of the square. For hexagon bins, the number and units specified determine the distance between parallel sides.

The units that will be used to specify bin size.

Calculate the total size of the features summarized in each bounding polygon. For lines, the total length is calculated. For polygons, the total area is calculated.

The units that will be used to calculate the total size of summarized features.

The number or date fields from the input layer that will be summarized, and the statistics that will be calculated for each field.

The field from the input layer that will be used to separate the data into distinct groups for statistical calculations.

Calculate the minority (least dominant) and majority (most dominant) groups within each summary area.

Calculate the percentage of the summarized features that belong to each group within each summary area.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the results will be saved.
