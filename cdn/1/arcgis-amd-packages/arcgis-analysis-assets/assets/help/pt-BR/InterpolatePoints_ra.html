A ferramenta Interpolar Pontos prevê valores em novos locais com base nas medições de ponto. A ferramenta cria uma saída raster.

A camada de ponto que contém os pontos onde os valores foram medidos.

O campo numérico que contém os valores que serão interpolados.

Especifica se a velocidade ou a precisão serão otimizadas. Predições mais precisas tomam mais tempo para calcular.

Este parâmetro altera os valores padrão de vários outros parâmetros.

O tamanho da célula e a unidade para os rasters de saída.

Especifica se os dados serão transformados ou não. A interpolação é mais precisa para dados que seguem uma distribuição normal (em forma de sino).

O tamanho dos modelos de interpolação local que serão combinados para criar o mapa de previsão final.

O número de vizinhos que serão usados. Um número maior de vizinhos geralmente produz resultados mais precisos, mas os resultados levam mais tempo para serem calculados.

O nome da camada raster interpolada que será criada e adicionada ao mapa.

Escolha se também deseja criar um raster opcional de erros padrão para os valores previstos.

Se marcado, o raster de erro padrão de saída terá o mesmo nome que o Nome do raster de saída, mas com a palavra Erros anexada a ele. A camada será criada e adicionada ao mapa.

O tipo de saída que será criado. As saídas podem ser camadas de imagens mosaicadas ou camadas de imagens dinâmicas.

O nome da pasta em Meu conteúdo onde o resultado será salvo.
