A ferramenta Resumir Dentro calcula estatísticas para feições dentro de caixas ou polígonos especificados.

A camada que será resumida.

O tipo de feições delimitadoras dentro das quais as feições de entrada serão resumidas. Isso pode ser uma camada de polígono ou uma grade de mosaico de caixas quadradas ou, hexagonais que são geradas quando a ferramenta é executada.

A camada de polígono dentro da qual as feições de entrada serão resumidas.

O tamanho das caixas geradas. Para caixas quadradas, o número e as unidades especificadas determinam a altura e o comprimento do quadrado. Para caixas de hexágono, o número e unidades especificadas determinam a distância entre os lados paralelos.

As unidades que serão usadas para especificar o tamanho da caixa.

Calcule o tamanho total das feições resumidas em cada polígono delimitador. Para linhas, o comprimento total será calculado. Para polígonos, a área total será calculada.

As unidades que serão usadas para calcular o tamanho total das feições resumidas.

Os campos de número ou data da camada de entrada a serem resumidos e as estatísticas que serão calculadas para cada campo.

O campo da camada de entrada que será usado para separar os dados em grupos distintos para cálculos estatísticos.

Calcule os grupos minoritários (menos dominantes) e majoritários (mais dominantes) dentro de cada área de resumo.

Calcule a porcentagem das feições resumidas que pertencem a cada grupo dentro de cada área de resumo.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em Meu conteúdo onde o resultado será salvo.
