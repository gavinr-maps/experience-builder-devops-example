A ferramenta Detectar Objetos Utilizando Aprendizagem Detalhada usa um modelo de aprendizagem detalhada em uma camada de imagem de entrada para produzir uma camada de feição contendo os objetos detectados.

A imagem ou camada de feição que será usada para detectar objetos.

Especifica como os itens raster em uma camada de imagem serão processados ​​após serem mosaicados: juntos ou individualmente.

O pacote de aprendizagem detalhada que contém o modelo que será usado para detectar os objetos.

Os argumentos do modelo que estão presentes no modelo de aprendizagem detalhada junto com seus valores padrão.

Especifica se a supressão não máxima será executada para remover objetos duplicados.

O nome do campo que será adicionado à camada de feição de saída para indicar o valor de confiança relatado.

O nome do campo que será adicionado à camada de feição de saída para indicar o rótulo classificado para o objeto.

A proporção máxima permitida para cada feição encontrada que se sobrepõe a outra feição. O valor padrão é 0.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em Meu conteúdo onde o resultado será salvo.
