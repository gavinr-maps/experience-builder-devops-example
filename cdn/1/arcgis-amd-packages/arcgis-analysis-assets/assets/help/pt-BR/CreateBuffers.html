A ferramenta Criar Buffers cria polígonos que cobrem uma determinada distância das feições de entrada.

As feições de ponto, linha ou polígono que serão armazenados no buffer.

Especifica se a distância do buffer será definida por uma constante (Valor) ou um campo numérico(Campo).

A distância que será usada para criar o buffer. Se vários valores forem adicionados, cada valor será usado para criar um buffer em cada feição de entrada.

O campo que conterá a distância do buffer para cada feição.

As unidades que serão usadas para especificar a distância do buffer.

Especifica como os buffers de diferentes feições de entrada serão tratados quando eles se sobrepõem.

Especifica se os buffers de múltiplas distâncias serão buffers concêntricos que se sobrepõem (Discos sobrepostos) ou anéis que não se sobrepõem (Anéis distintos).

Para feições de linha, especifica se o buffer será criado em ambos os lados da linha (Arredondar), do lado esquerdo apenas (Esquerdo),ou apenas do lado direito (Direito).

Para feições de polígono, especifica se as feições de polígono de entrada serão incluídas no buffer (Incluir) ou excluído do buffer (Excluir).

Especifica se a forma do buffer no final de uma feição de linha será arredondada (Arredondar) ou achatada (Achatar).

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em Meu conteúdo onde o resultado será salvo.
