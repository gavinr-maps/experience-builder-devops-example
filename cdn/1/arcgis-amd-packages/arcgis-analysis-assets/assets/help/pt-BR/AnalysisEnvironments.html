Configurações de ambiente para análise no Map Viewer.

Especifica o sistema de coordenadas para análise e a camada resultante.

Especifica a área a ser usada para análise.

Ajusta a extensão da saída para que corresponda ao alinhamento da célula da camada raster de ajuste especificada.

Especifica o tamanho ou resolução da célula que será usada para análise e para criar a camada raster de saída.

Especifica uma camada de máscara, na qual apenas as células que se enquadram na área de máscara serão usadas para análise.

Especifica como os valores de pixel serão interpolados quando os rasters de entrada e saída não estiverem alinhados.

Especifica o tipo de processador a ser usado. As opções são CPU, GPU e Automático. Se Automático for especificado, a ferramenta definirá o tipo de processador.

Especifica o número de instâncias de serviço de processamento raster usadas durante a execução da ferramenta. Os valores suportados incluem strings vazias (a ferramenta determinará o número de instâncias de serviço de processamento raster a serem usadas), números inteiros (0 ou números inteiros positivos) e porcentagens. As porcentagens devem incluir um sinal de porcentagem (%) após o valor percentual.

Determina se uma mensagem de aviso aparecerá se o número estimado de créditos ultrapassar o limite especificado (habilitado) ou se a análise será executada sem exibir mensagem independentemente do consumo estimado de créditos (não habilitado). A análise ainda respeitará as configurações de orçamento de crédito da organização.

Especifica o limite no qual uma mensagem de aviso aparecerá com base no consumo de crédito estimado a partir da execução da análise. A análise ainda poderá ser executada se a estimativa ultrapassar o limite.
