A ferramenta Detectar Alteração Utilizando Aprendizagem Detalhada usa um modelo de aprendizagem detalhada para detectar alterações entre duas camadas raster.

A camada de imagens de entrada antes da alteração que será usada como base para comparação.

A camada de imagens de entrada após a alteração que será usada para comparação com o original.

A camada de imagens classificadas de saída que mostra a alteração.

Os argumentos do modelo que estão presentes no modelo de aprendizagem detalhada junto com seus valores padrão.

O pacote de aprendizagem detalhada que contém o modelo que será usado para detectar a mudança entre as camadas de imagens.

O tipo de saída que será criado. A saída pode ser uma camada de imagem mosaicada ou uma camada de imagem dinâmica.

O nome da pasta em Meu conteúdo onde o resultado será salvo.
