A ferramenta Estatística Zonal como Tabela resume os valores de um raster dentro das zonas de outro conjunto de dados e registra os resultados em uma tabela.

A entrada que define os limites das zonas, ou áreas, que serão resumidas.

O campo que contém os valores que definem cada zona.

A camada raster que contém os valores das células que serão usadas para calcular as estatísticas.

Especifica o tipo de estatística que será calculada.

Especifica os valores de percentil que serão calculados.

Especifica o tipo de estatística que especifica o método que será usado quando o número de valores do raster de entrada a serem calculados for um número par será calculado.

Especifica como os valores da célula do raster de valor de entrada serão processados ​​para dados circulares.

Se desmarcado, estatísticas aritméticas comuns (lineares) serão calculadas. Se marcado, as estatísticas circulares serão calculadas.

Especifica o valor que será usado para arredondar um valor linear para o intervalo de uma determinada estatística circular.

Especifica se as células NoData no raster de valor de entrada serão ignoradas nos resultados das zonas em que se enquadram.

Se marcado, as células NoData no raster de valor serão ignoradas no cálculo estatístico. Se desmarcada, as células NoData não serão ignoradas e toda a zona receberá o valor NoData no raster de saída.

Especifica como os rasters de entrada serão processados se forem multidimensionais.

Se desmarcado, as estatísticas serão calculadas para todas as dimensões do conjunto de dados multidimensional de entrada. Se marcado, as estatísticas serão calculadas a partir da parte atual do conjunto de dados multidimensional de entrada.

O nome de camada da tabela de estatística zonal que será criada e adicionada ao mapa.

O nome da pasta em Meu conteúdo onde o resultado será salvo.
