A ferramenta Resumir Mais Próximo calcula estatísticas para feições dentro de uma distância especificada de outras feições. A distância pode ser calculada usando uma distância em linha reta ou um modo de viagem.

A camada a ser resumida dentro de uma distância da camada próxima.

A camada de ponto, linha ou polígono usada como os locais iniciais em torno dos quais as áreas delimitadoras serão criadas para resumir as feições próximas.

O método com o qual as áreas delimitadoras serão criadas. O método pode ser uma distância em linha reta ou um modo de viagem.

A distância ou tempo que será usado para criar áreas delimitadoras. Se vários valores forem adicionados, cada valor será usado para calcular áreas delimitadoras para todos as feições de entrada.

As unidades que serão usadas para especificar distância ou tempo.

Especifique se a camada resultante deve ser as áreas delimitadoras (marcada) ou a mesma geometria da camada próxima de entrada (desmarcada).

Calcule o tamanho total das feições resumidas em cada área delimitadora. Para linhas, o comprimento total será calculado. Para polígonos, a área total será calculada.

As unidades que serão usadas para calcular o tamanho total das feições resumidas.

Os campos de número ou data da camada de entrada a serem resumidos e as estatísticas que serão calculadas para cada campo.

O campo da camada de entrada que será usado para separar os dados em grupos distintos para cálculos estatísticos.

Calcule os grupos minoritários (menos dominantes) e majoritários (mais dominantes) dentro de cada área de resumo.

Calcule a porcentagem das feições resumidas que pertencem a cada grupo dentro de cada limite.

O nome da camada que será criada e adicionada ao mapa.

O nome da pasta em Meu conteúdo onde o resultado será salvo.
