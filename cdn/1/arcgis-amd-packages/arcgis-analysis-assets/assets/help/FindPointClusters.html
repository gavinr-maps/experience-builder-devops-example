The Find Point Clusters tool identifies clusters of point features within surrounding noise based on their spatial distribution.

The input point layer that will be grouped into clusters based on the spatial distribution of the point features.

The method that will be used to identify clusters.

The minimum number of point features that will be considered a cluster.

Specifies the maximum distance around each point feature to search for points that can be included in the cluster. If left blank, the Search distance value will be the highest core distance found in the layer, excluding the top 1 percent of core distances.

Specifies the units that will be used for the search distance.

An integer between 0 and 100 that determines the compactness of the clusters. If left blank, the Cluster sensitivity value will be calculated using the Kullback-Leibler Divergence.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
