“计划路径”工具可为车队创建具有多个停靠点的出行路径。

指定车辆开始路径的点要素。

车辆必须在其路径上访问的点要素。 这些点会在输出中变为已分配或未分配的停靠点。

确定车辆是否必须返回其起始位置（启用）或在指定位置结束（禁用）。

指定车辆结束路径的点要素。 如果图层仅包含一个点，则所有车辆路径都将经过一个公共端点。 如果图层包含多个点，则每辆车都车辆路径都将经过唯一的终点位置。

起始图层中用于唯一标识各个路径、车辆或司机的字段。 如果车辆在多个位置开始并在多个位置结束，则为起始图层和结束他层选择的路径 ID 必须匹配。

结束图层中用于唯一标识各个路径、车辆或司机的字段。 如果车辆在多个位置开始并在多个位置结束，则为起始图层和结束他层选择的路径 ID 必须匹配。

一个或多个点要素，作为在基础街道上行驶时的临时限制（障碍）。 允许在街道上行驶，但不可以通过障碍。

一个或多个线要素，作为在基础街道上行驶时的临时限制（障碍）。 线障碍禁止在障碍与街道相交的任何位置行驶。

一个或多个面要素，作为在基础街道上行驶时的临时限制（障碍）。 面障碍禁止在面与街道相交的位置通行。

创建路径时使用的出行方式。 出行模式可以基于距离或时间。

车辆出发的日期和时间。

可访问停靠点的车辆的最大数量。 如果 起始图层和 结束图层各包含一个要素，则最多可以指定 100 台辆车。 如果任一图层包含多个要素，则无法编辑默认值。

每辆车在一条路径上可以停靠的最大数量。 您可以指定介于 1 和 200 之间的值。

指定在每个停靠点所花费的预期平均服务时间。

确定是否将最长路径规划时间应用于车辆。 路径时间包括预期行驶时间及花费在该路径沿线停靠点的服务时间。

指定某路径从开始到结束应持续的最长时间。

要创建并添加至地图的图层的名称。

指定是否将每条结果路径另存为路径图层。

我的内容中要保存结果的文件夹名称。
