Detect Objects Using Deep Learning

The Detect Objects Using Deep Learning tool uses a deep learning model on an input imagery layer to produce a feature layer containing the detected objects.

The imagery layer that will be used to detect objects.

Specifies how raster items in an imagery layer will be processed after being mosaicked: together or individually.

The deep learning package that contains the model that will be used to detect the objects.

The model arguments that are present in the deep learning model together with their default values.

Specifies whether nonmaximum suppression will be performed to remove duplicate objects.

The name of the field that will be added to the output feature layer to indicate the confidence value reported.

The name of the field that will be added to the output feature layer to indicate the classified label for the object.

The maximum ratio allowed for each found feature that overlaps another feature. The default value is 0.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the result will be saved.
