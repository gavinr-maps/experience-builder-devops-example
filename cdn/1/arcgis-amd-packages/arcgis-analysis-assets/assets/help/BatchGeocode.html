The Geocode Locations from Table tool converts addresses or places from an input table to coordinates.

The input table with the addresses or places to be geocoded. The input must be a table, a .csv file, or an Excel file.

The locator service to use for geocoding. If the locator service you want to use is not in the drop-down menu, contact your administrator to have the locator service configured for batch geocoding.

Specifies whether the input data is contained in a single field or multiple fields.

The fields that will be used by the locator to determine the locations of the data. Certain fields are assumed to be in the data, such as address, city, state, and postal code.

The country in which the data is located. If all of the addresses are from one country, provide that country to ensure the most accurate results. Otherwise, leave this parameter empty.

Specifies the fields that will be included in the output dataset. The options are as follows:

  • All—All available locator output fields will be included in the geocode results.
  • Minimal—A subset of fields from the locator as well as the fields from the input table will be included in the results.
  • Location only—The Shape field from the locator as well as the fields from the input table will be included in the results.

The type of output that will be created. The output can be a feature service, .csv file, or .xls file.

The name of the output item where the located data will be stored.

The name of the folder in My Content where the result will be saved.
