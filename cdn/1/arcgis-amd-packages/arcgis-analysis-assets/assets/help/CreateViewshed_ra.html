The Geodesic Viewshed tool identifies the locations on a raster surface that are visible to the input observer locations.

The elevation raster on which the viewshed will be calculated.

The observer locations for which the viewshed will be calculated.

Optimizes viewshed calculation based on either speed or accuracy.

Specifies the maximum viewing distance for the viewshed calculation.

The distance can be defined by a value and its units, or from a field in the Input observer point or line features.

Specifies the minimum viewing distance for the viewshed calculation.

The distance can be defined by a value and its units, or from a field in the Input observer point or line features.

Specifies the maximum and minimum viewing distances type.

If unchecked, the viewing distances will be treated as two-dimensional Euclidean distances. If checked, the viewing distances will be treated as three-dimensional distances.

Specifies the surface elevation of the observer locations.

The elevation can be defined by a value and its units, or from a field in the Input observer point or line features.

Specifies the height of the observers above ground.

The height can be defined by a value and its units, or from a field in the Input observer point or line features.

Specifies the height of targets on the ground.

The height can be defined by a value and its units, or from a field in the Input observer point or line features.

The name of the viewshed raster layer that will be created and added to the map. This layer contains the visible areas where the observers can see, and the cell values represent the number of observers that can view the cells.

The name of the above ground level (AGL) raster layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
