The Zonal Statistics tool summarizes the values of a raster within the zones of another dataset.

The input that defines the boundaries of the zones, or areas, that will be summarized.

The field that contains the values that define each zone.

The raster layer that contains the cell values that will be used to calculate statistics.

Specifies the statistic type that will be calculated.

Specifies the percentile value that will be calculated.

Specifies the method that will be used when the number of values from the input raster to be calculated is an even number.

Specifies how the Input value raster cell values will be processed for circular data.

If unchecked, ordinary arithmetic (linear) statistics will be calculated. If checked, circular statistics will be calculated.

Specifies the value that will be used to round a linear value to the range of a given circular statistic.

Specifies whether NoData cells in the Input value raster will be ignored in the results of the zones they fall within.

If checked, NoData cells in the value raster will be ignored in the statistic calculation. If unchecked, NoData cells will not be ignored, and the entire zone will receive the NoData value on the output raster.

Specifies how the input rasters will be processed if they are multidimensional.

If unchecked, the statistics will be calculated for all dimensions of the input multidimensional dataset. If checked, statistics will be calculated from the current slice of the input multidimensional dataset.

The name of the zonal statistics raster layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
