The Multidimensional Principal Components tool transforms the multidimensional imagery layers into a reduced number of components that account for the variance of the data, so that spatial and temporal patterns can be readily identified.

The multidimensional imagery layer that will be analyzed.

The method that will be used to perform the principal component analysis. The options include Dimension reduction and Spatial reduction.

The dimension of the multidimensional imagery layer along which the variable will be analyzed.

The name of the output imagery layer or table layer showing the principal components found depending on the following choices of mode:

  • Dimension reduction output will be a multiband imagery layer.
  • Spatial reduction output will be a table layer.

The name of the output imagery layer or table layer showing the supporting data in the principal components choice depending on the following choices of mode:

  • Dimension reduction output will be a table layer.
  • Spatial reduction output will be an imagery layer.

The name of the Eigenvalues table, which indicates the variance percentage of each component.

The name of the variable of the input multidimensional imagery layer used in the computation.

The number of principal components to compute, usually fewer than the number of input rasters.

The name of the layer that will be created and added to the map.

The type of output that will be created. The outputs can be Tiled imagery layers or Dynamic imagery layers.

The name of the folder in My Content where the results will be saved.
