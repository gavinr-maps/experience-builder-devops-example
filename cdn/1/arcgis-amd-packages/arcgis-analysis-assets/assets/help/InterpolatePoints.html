The Interpolate Points tool predicts values at new locations based on point measurements. The tool creates feature outputs.

Choose the point layer of the locations that will be interpolated.

Choose the field containing the measured values that will be interpolated.

Choose whether calculation speed or accuracy and precision will be prioritized. More accurate and precise predictions take longer to calculate.

Choose whether a layer of standard errors for the interpolation predictions will be included in the output.

Choose how the class break values will be calculated.

Specify the number of classes (ranges of predicted values) that will be included in the result layer. Each class defines the boundary of a polygon and represents contours of the predicted values. The class break values are determined by the classification type.

Provide the class break values that will be used.

Choose a polygon layer that will be used to define a study area. The results will be clipped to only display within the study area.

Choose a point layer representing the locations that will be used to predict values. If provided, a point layer containing predicted values at the specified locations will be created.

The name of the layer that will be created and added to the map.

The name of the folder in My Content where the results will be saved.
