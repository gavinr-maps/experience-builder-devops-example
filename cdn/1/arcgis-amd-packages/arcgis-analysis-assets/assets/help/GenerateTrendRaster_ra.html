The Generate Trend Raster tool estimates the trend for each pixel along a dimension for one or more variables in a multidimensional imagery layer.

The multidimensional imagery layer that will be used to generate the trend.

The dimension along which a trend will be extracted for the variable or variables selected in the analysis.

The variable or variables for which trends will be calculated.

The time unit that will be used for the length of a seasonal period when using the Seasonal-Kendall line type.

The type of trend analysis that will be performed on pixel values along a dimension.

The frequency or the polynomial order number that will be used for the trend fitting.

Specifies whether NoData or missing values will be ignored in the analysis.

The length of periodic variation that will be modeled for the harmonic cycle.

The time unit that will be used for the length of the harmonic cycle.

Specifies whether the root mean square error (RMSE) of the trend fit line will be calculated.

Specifies whether the R-squared goodness-of-fit statistic for the trend fit line will be calculated.

Specifies whether the p-value statistic for the slope coefficient of the trend fit line will be calculated.

The name of the layer that will be created and added to the map.

The type of output that will be created. The output can be a Tiled imagery layer or a Dynamic imagery layer.

The name of the folder in My Content where the result will be saved.
