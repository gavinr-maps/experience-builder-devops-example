Das Werkzeug "Punkt-Cluster suchen" identifiziert Cluster aus Punkt-Features im Datenrauschen des Umfelds basierend auf deren räumlicher Verteilung.

Der Eingabe-Punkt-Layer, der basierend auf der räumlichen Verteilung der Punkt-Features in Cluster gruppiert wird.

Die Methode zum Identifizieren von Clustern.

Die minimale Anzahl von Punkt-Features, die als Cluster eingestuft werden sollen.

Gibt die maximale Entfernung um jedes Punkt-Feature für die Suche nach Punkten an, die in den Cluster aufgenommen werden können. Wenn die Option leer gelassen wird, ist der Wert Suchentfernung die größte Kernentfernung, die im Layer gefunden wurde, wobei die Kernentfernungen des obersten 1 Prozents ausgeschlossen werden.

Gibt die Einheiten an, die für die Suchentfernung verwendet werden.

Eine ganze Zahl zwischen 0 und 100, die die Kompaktheit der Cluster bestimmt. Wenn die Option leer gelassen wird, wird der Wert Cluster-Empfindlichkeit mithilfe Kullback-Leibler Divergenz berechnet.

Der Name des Layers, der erstellt und der Karte hinzugefügt wird.

Der Name des Ordners in Eigene Inhalte, in dem das Ergebnis gespeichert werden soll.
