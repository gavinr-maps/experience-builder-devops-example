Das Werkzeug "Objekte mit Deep Learning erkennen" verwendet ein Deep-Learning-Modell auf einem Eingabe-Bilddaten-Layer, um einen Feature-Layer mit den erkannten Objekten zu erstellen.

Der Bilddaten- oder Feature-Layer, der für die Objekterkennung verwendet wird.

Gibt an, wie Raster-Elemente in einem Bilddaten-Layer verarbeitet werden, nachdem sie mosaikiert wurden: zusammen oder einzeln.

Das Deep-Learning-Paket, das das für die Objekterkennung zu verwendende Modell enthält.

Die Modellargumente, die zusammen mit ihren Standardwerten im Deep-Learning-Modell vorhanden sind.

Gibt an, ob eine Non Maximum Suppression durchgeführt wird, um doppelte Objekte zu entfernen.

Der Name des Feldes, das dem Ausgabe-Feature-Layer hinzugefügt wird, um den gemeldeten Konfidenzwert anzugeben.

Der Name des Feldes, das dem Ausgabe-Feature-Layer hinzugefügt wird, um die klassifizierte Beschriftung für das Objekt anzugeben.

Das maximal zulässige Verhältnis für jedes gefundene Feature, das ein anderes Feature überlappt. Der Standardwert ist 0.

Der Name des Layers, der erstellt und der Karte hinzugefügt wird.

Der Name des Ordners in Eigene Inhalte, in dem das Ergebnis gespeichert werden soll.
