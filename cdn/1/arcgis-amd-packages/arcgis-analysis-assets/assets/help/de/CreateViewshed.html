Das Werkzeug "Sichtfelder erstellen" verwendet den Höhenanalyse-Service von Esri, um Flächen zu identifizieren, die von angegebenen Beobachterpositionen aus sichtbar sind.

Die Punkt-Features, die als Beobachterpositionen verwendet werden sollen.

Gibt die Höhe der Beobachter über dem Boden an.

Die Maßeinheiten, die für den Parameter Beobachterhöhe verwendet werden.

Gibt die Höhe der Ziele am Boden an.

Die Maßeinheiten, die für den Parameter Zielhöhe verwendet werden.

Gibt die maximale Betrachtungsentfernung für die Sichtfeldberechnung an.

Die Maßeinheiten, die für den Parameter Maximale Betrachtungsentfernung verwendet werden.

Der Name des Sichtfeld-Polygon-Layers, der erstellt und der Karte hinzugefügt wird. Dieser Layer enthält die sichtbaren Bereiche, in denen die Beobachter die beobachteten Objekte sehen können. Das Feld "Häufigkeit" gibt die Anzahl der Beobachter an.

Der Name des Ordners in Eigene Inhalte, in dem das Ergebnis gespeichert werden soll.
