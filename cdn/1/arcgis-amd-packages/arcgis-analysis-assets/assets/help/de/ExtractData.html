Das Werkzeug "Daten extrahieren" wählt Features innerhalb einer Ausdehnung aus und erstellt ein Element in Ihrem Inhalt.

Die Layer, die extrahiert werden. Unterstützte Eingabe-Layer umfassen Layer, die Sie besitzen, und Layer mit aktivierter Exportfunktion.

Die Ausdehnung der Features, die extrahiert werden.

Gibt an, ob Features ausgeschnitten werden, die die Ausdehnung überschneiden. Wenn deaktiviert, werden vollständige Features, die die Ausdehnung überschneiden, extrahiert. Wenn aktiviert, werden Features, die die Ausdehnung überschneiden, ausgeschnitten, und nur die Teile dieser Features, die innerhalb der Ausdehnung liegen, werden extrahiert.

Das Dateiformat der Daten, die extrahiert werden.

Der Name des Ausgabeelements, in dem die extrahierte Datei gespeichert wird.

Der Name des Ordners in Eigene Inhalte, in dem das Ergebnis gespeichert werden soll.
