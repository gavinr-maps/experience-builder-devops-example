Das Werkzeug "Positionen aus Tabelle geokodieren" wandelt Adressen oder Positionen aus einer Eingabetabelle in Koordinaten um.

Die Eingabetabelle mit den Adressen oder Orten, die geokodiert werden sollen. Die Eingabe muss eine Tabelle, eine .csv-Datei oder eine Excel-Datei sein.

Der Locator-Service, der für die Geokodierung verwendet werden soll. Wenn der Locator-Service, den Sie verwenden möchten, nicht im Dropdown-Menü aufgeführt ist, wenden Sie sich an Ihren Administrator, um den Locator-Service für Batch-Geokodierung konfigurieren zu lassen.

Gibt an, ob die Eingabedaten in einem einzelnen Feld oder in mehreren Feldern enthalten sind.

Die Felder, die vom Locator verwendet werden, um die Datenspeicherorte zu bestimmen. Es wird davon ausgegangen, dass bestimmte Felder in den Daten enthalten sind, z. B. Adresse, Stadt, Bundesland und Postleitzahl.

Das Land, auf das sich die Daten beziehen. Wenn alle Adressen aus einem Land stammen, geben Sie dieses Land an, um genaueste Ergebnisse sicherzustellen. Andernfalls lassen Sie diesen Parameter leer.

Gibt die Felder an, die im Ausgabe-Dataset berücksichtigt werden sollen. Die folgenden Optionen sind verfügbar:

  • Alle: Alle verfügbaren Locator-Ausgabefelder werden in den Geokodierungsergebnissen berücksichtigt.
  • Minimal: In die Ergebnisse werden eine Teilmenge der Felder aus dem Locator sowie die Felder aus der Eingabetabelle einbezogen.
  • Nur Position: Das Feld Shape aus dem Locator sowie die Felder aus der Eingabetabelle werden in die Ergebnisse einbezogen.

Die Art der Ausgabe, die erstellt wird. Die Ausgabe kann ein Feature-Service, eine .csv-Datei oder eine .xls-Datei sein.

Der Name des Ausgabeelements, in dem die verorteten Daten gespeichert werden.

Der Name des Ordners in Eigene Inhalte, in dem das Ergebnis gespeichert werden soll.
