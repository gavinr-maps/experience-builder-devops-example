Das Werkzeug "Nach Attributen und Position suchen" erstellt einen neuen Layer aus den Features oder Teilen von Features, die bestimmte räumliche oder Attributkriterien erfüllen.

Der Punkt-, Linien- oder Polygon-Layer, für den Features basierend auf einer Kombination aus Attribut- und räumlichen Abfragen gesucht werden.

Die räumlichen oder Attributkriterien, die für die Feature-Suche verwendet werden. Klicken Sie auf die Schaltfläche Neue Abfrage erstellen, um mindestens ein räumliches oder Attributkriterium zu erstellen. Diese Features oder Teile der Features, die allen Kriterien entsprechen, werden in den Ergebnis-Layer geschrieben.

Die räumlichen oder Attributkriterien, die für die Feature-Suche verwendet werden. Klicken Sie auf die Schaltfläche Neue Abfrage erstellen, um mindestens ein räumliches oder Attributkriterium zu erstellen. Diese Features oder Teile der Features, die allen Kriterien entsprechen, werden in den Ergebnis-Layer geschrieben.

Wenn nur ein Teil eines Features die Kriterien erfüllt, wird angegeben, ob das Ergebnis das vollständige Feature oder nur den Teil des Features enthält, der alle Kriterien erfüllt. Wenn aktiviert, werden nur die Teile der Features zurückgegeben, die alle Kriterien erfüllen. Wenn nicht aktiviert, wird das vollständige Feature zurückgegeben, sofern zumindest ein Teil des Features alle Kriterien erfüllt.

Der Name des Layers, der erstellt und der Karte hinzugefügt wird.

Der Name des Ordners in Eigene Inhalte, in dem das Ergebnis gespeichert werden soll.
