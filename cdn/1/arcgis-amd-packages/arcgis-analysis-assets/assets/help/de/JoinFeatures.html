Mit dem Werkzeug "Features verbinden" können Attribute von einem Layer oder einer Tabelle basierend auf angegebenen räumlichen Beziehungen und Attributbeziehungen einem anderen Layer bzw. einer anderen Tabelle hinzugefügt werden. Optional können Statistiken berechnet werden, wenn mehrere übereinstimmende Datensätze vorhanden sind.

Der Layer oder die Tabelle, an dessen bzw. deren Attributtabelle Attribute des Join-Layers angehängt werden.

Der Layer oder die Tabelle mit den Attributen, die an den Ziel-Layer angehängt werden.

Geben Sie an, ob Layer basierend auf ihrer räumlichen Beziehung verbunden werden sollen. Dies erfordert, dass beide Layer eine Geometrie aufweisen.

Geben Sie die räumliche Beziehung an, anhand derer bestimmt wird, welche Features verbunden werden. Features im Join-Layer, die die angegebene Beziehung zum Ziel-Layer aufweisen, werden verbunden. Die verfügbaren Beziehungen hängen vom Geometrietyp der Eingaben ab.

Die Entfernung von den Features im Ziel-Layer, anhand derer bestimmt wird, ob Features verbunden werden sollen.

Die Einheiten, die zur Angabe der Entfernung verwendet werden.

Geben Sie an, ob Layer basierend auf übereinstimmenden Feldwerten verbunden werden sollen. Wenn eine der Eingaben eine Tabelle ist, ist dies die einzige Option.

Geben Sie die Attributbeziehung an, anhand derer bestimmt wird, welche Features verbunden werden. Die Features werden zugeordnet, wenn die Feldwerte im Join-Layer den Feldwerten im Ziel-Layer entsprechen.

Geben Sie an, wie Verbindungen verarbeitet werden, wenn mehrere Datensätze aus dem Join-Layer die Kriterien für eine Verbindung erfüllen. Ihnen stehen folgende Optionen zur Auswahl:

  • Eins zu eins verbinden: Verbindet einen einzelnen Datensatz im Join-Layer mit jedem Feature im Ziel-Layer.
  • Eins zu vielen verbinden: Verbindet alle übereinstimmenden Features im Join-Layer mit dem Ziel-Layer. Der Ergebnis-Layer enthält mehrere Datensätze des Ziel-Features.

Wenn mehrere Features im Join-Layer mit einem Ziel-Feature übereinstimmen, können Sie hier angeben, ob ein einzelner Datensatz aus dem Join-Layer verbunden werden soll oder ob Summenstatistiken aller übereinstimmenden Datensätze aus dem Join-Layer berechnet werden sollen.

Wenn es mehrere Join-Features gibt, die mit einem Ziel-Feature übereinstimmen, werden für alle übereinstimmenden Datensätze Summenstatistiken berechnet. Geben Sie die Zahlen- oder Datumsfelder aus dem Join-Layer und die Statistiken an, die für jedes Feld berechnet werden.

Wenn es mehrere Join-Features gibt, die mit einem Ziel-Feature übereinstimmen, wird der erste Datensatz aus der Attributtabelle des Join-Layers bei der Ausgabe berücksichtigt. Wählen Sie optional ein Zahlen- oder Datumsfeld und die Sortierreihenfolge für die Sortierung der Attributtabelle aus.

Bestimmt, ob Ziel-Features ohne Übereinstimmung bei der Ausgabe berücksichtigt werden sollen. Ein Inner Join gibt nur übereinstimmende Zieldatensätze zurück, während ein Left Join alle Zieldatensätze zurückgibt, selbst wenn das Ziel-Feature keine Übereinstimmung im Join-Layer aufweist.

Der Name des Layers oder der Tabelle, der bzw. die erstellt und der Karte hinzugefügt wird.

Der Name des Ordners in Eigene Inhalte, in dem das Ergebnis gespeichert werden soll.
